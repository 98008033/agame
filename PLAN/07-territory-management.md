# 领地经营系统

## 七、领地经营系统

### 7.1 领地发展路线

```
阶段1: 荒村（人口<50）
  └─ 基础生存：农田、猎户、伐木
  └─ 核心挑战：资源平衡、匪患

阶段2: 村庄（人口50-200）
  └─ 基础建筑：仓库、集市、哨塔
  └─ 核心挑战：外交关系、贸易

阶段3: 镇子（人口200-1000）
  └─ 高级建筑：书院、使馆、神庙
  └─ 核心挑战：阵营选择、资源竞争

阶段4: 城市（人口1000-5000）
  └─ 特色建筑：按阵营选择不同建筑风格
  └─ 核心挑战：战争、科技竞赛

阶段5: 王城（人口5000+）
  └─ 终极建筑：奇观、军事要塞、魔法塔
  └─ 核心挑战：统一大陆或维持和平
```

### 7.2 建筑系统

```
生产建筑:
  ├─ 农田 → 粮食产出（可升级灌溉）
  ├─ 猎户小屋 → 粮食+皮毛
  ├─ 伐木场 → 木材
  ├─ 矿洞 → 石材/铁矿
  ├─ 工坊 → 加工品（需要对应科技）
  └─ 商栈 → 贸易收入

功能建筑:
  ├─ 民居 → 增加人口上限
  ├─ 仓库 → 增加存储上限
  ├─ 集市 → 开启贸易功能
  ├─ 哨塔 → 预警+防御
  ├─ 书院 → 研究科技
  ├─ 酒馆 → 吸引NPC+情报
  └─ 兵营 → 训练士兵

阵营特色建筑:
  ├─ [苍龙] 炼丹房、机关坊、太学院
  ├─ [霜狼] 符文殿堂、兽栏、锻炉
  ├─ [金雀花] 银行、交易所、炼金实验室
  └─ [边境] 自由市场、佣兵工会、黑市

特殊建筑:
  ├─ 使馆 → 与特定国家建立关系
  ├─ 神庙 → 选择魔法体系
  ├─ 奇观 → 提供全局buff
  └─ 地下室 → 隐藏设施（间谍/走私）
```

### 7.3 资源系统

| 资源 | 来源 | 用途 | 获取方式 |
|------|------|------|----------|
| 粮食 | 农业 | 养活人口 | 农田、狩猎、贸易 |
| 木材 | 伐木 | 建设、燃料 | 伐木场、贸易 |
| 石材 | 采石 | 高级建筑 | 矿洞、贸易 |
| 铁矿 | 采矿 | 装备、武器 | 矿洞、贸易 |
| 金币 | 贸易/税收 | 通用货币 | 商栈、任务 |
| 人口 | 自然增长 | 劳动力、军队 | 时间、难民、招募 |
| 情报 | 间谍/探索 | 外交、决策 | 酒馆、任务 |
| 魔力水晶 | 魔法研究 | 施法、附魔 | 探索、特殊事件 |
