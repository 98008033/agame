# Agame 职业团体系统设计

> **核心理念**：NPC社会不只是个体的集合，而是由各种团体编织而成的网络。工会、商会、学院、军团——这些团体是社会运转的骨架，也是NPC和玩家获取资源、提升技能、建立人脉的关键途径。

---

## 一、职业团体的定义与功能

### 1.1 什么是职业团体？

职业团体是由具有相同或相关技能、职业背景的NPC组成的组织。它们：
- 不是"阵营"（阵营是政治实体）
- 不是"家族"（家族是血缘实体）
- 是"职业共同体"——基于技能和利益形成的横向连接

```
团体类型矩阵：

按技能类型划分：
├── 工艺类：铁匠工会、织造工会、木匠工会
├── 商业类：商会、船业联盟、银行公会
├── 学术类：学院、书院、符文殿堂
├── 军事类：军团、佣兵工会、骑士团
├── 魔法类：五行学院、契约法师塔、符文师议会
└── 服务类：治愈师协会、情报网络、运输行会

按组织形态划分：
├── 松散型：定期聚会，无强制约束（如铁匠工会）
├── 紧密型：需要效忠，有义务和权利（如军团）
├── 精英型：门槛严格，地位高（如五行学院）
└── 隐秘型：不公开，需要特殊条件加入（如情报网络）
```

### 1.2 职业团体的六大功能

| 功能 | 说明 | 玩家如何受益 |
|------|------|--------------|
| **技能传承** | 团体内可以学习技能、互相传授 | 向团体内NPC学习技能更快 |
| **资源共享** | 团体成员可以共享材料、工具、情报 | 获得团体专属资源渠道 |
| **保护互助** | 团体成员遇到困难时互相帮助 | 团体NPC可能主动帮助你 |
| **利益协调** | 协调同业者的利益分配、标准制定 | 避免恶性竞争，获得合理定价 |
| **社会地位** | 加入知名团体带来身份认可 | 获得"××工会成员"头衔 |
| **政治影响** | 团体可以集体发声，影响政策 | 团体可能支持你的政治行动 |

```

  2. 获得"太学学生"身份
```

### 2.2 霜狼联邦——部落联盟型

**特点**：团体与部族绑定，技能传承通过师徒和家族，决斗和战功是晋升关键。

#### 2.2.1 符文魔法类团体

| 团体 | 名称 | 总部 | 门槛 | 功能 | 关键NPC |
|------|------|------|------|------|---------|
| **符文殿堂** | 极光台议会 | 极光台 | 符文技能L3+ + 部族认可 | 符文师培养、符文标准制定 | 雪鹰·望月 |
| **符文工匠会** | 铁炉堡分会 | 铁炉堡 | 符文技能L2+ + 锻造技能L2+ | 符文武器制造 | 火符文师·炎灵 |
| **萨满后裔会** | 各部族长老 | 各部落 | 符文技能L1+ + 血缘/师徒 | 传承祖灵沟通术 | 各部族长老 |

**霜狼特色传承机制**：
```
符文传承的"三部曲"：

第一步：血缘传承
  - 符文师倾向于把技艺传给子女
  - 血狼部族：战斗符文世家
  - 雪鹰部族：魔法符文世家
  - 灰熊部族：锻造符文世家

第二步：师徒传承
  - 无血缘者可拜师学艺
  - 需要完成"学徒考验"（通常是体力或意志测试）
  - 师徒关系比血缘更紧密（誓言绑定）

第三步：战功认证
  - 符文师需要用符文参加战斗证明能力
  - 成功使用符文击敌 → 获得"战用符文师"称号
  - 战功越多 → 在议会中的地位越高
```

```
  - 角斗需要角斗场裁判见证
  - 死伤自负，荣誉至上
```

**特点**：团体以利益和契约为纽带，金钱是门槛，契约是约束，商业精神主导。

| **商会联盟** | 各城商会 | 各城镇 | 商业技能L2+ + 入会费100 | 贸易协作、信息共享 | 铜臭、各地商人 |
| **船业联盟** | 千帆船业联盟 | 黄金港 | 商业技能L3+ + 拥有船只 | 航运协调、港口话语权 | 船王·千帆 |
| **保险公会** | 稳舵保险联盟 | 金雀花城 | 商业技能L3+ + 信誉认证 | 保险服务、风险评估 | 稳舵 |

**金雀花商业特色**：
```
契约入会制度：

几乎所有金雀花团体都采用契约入会：
  1. 支付入会费（金钱门槛）
  2. 签订入会契约（法律约束）
  3. 契约条款包括：
     - 会员义务（缴费、守规）
     - 会员权利（资源共享、投票权）
     - 退出条款（违约惩罚）
     - 期限（通常1年，可续约）

契约等级：
  普通会员：基础权利，入会费100
  高级会员：优先权利，入会费500
  核心会员：决策参与，入会费2000 + 推荐

退出机制：
  正常退出：契约期满，无惩罚
  违约退出：支付违约金（通常是入会费×3）
  强制退出：被公会驱逐，信用受损
```

**跨阵营团体的特殊性**：
```
加入跨阵营团体的好处：

  - 更广阔的资源网络

  - 可能被本阵营质疑忠诚
  - 团体可能被某阵营渗透

    - 金雀花可能希望你偏向他们
    - 你需要巧妙平衡，否则可能被驱逐
```

## 四、团体与玩家的交互

### 4.1 加入团体的流程

```
通用加入流程：

步骤1：了解团体
  - 通过晨报、NPC对话了解某团体
  - 或主动询问相关NPC："有没有铁匠工会？"

步骤2：确认门槛
  - 技能要求（技能等级）
  - 资源要求（金钱、物品）
  - 关系要求（推荐、效忠）
  - 附加条件（契约、考验）

步骤3：申请加入
  - 向团体管理者提交申请
  - 可能需要完成入会任务/考验

步骤4：正式加入
  - 获得"××团体成员"身份
  - 获得"××工会学徒"头衔（初级）
  - 记录在团体档案中

步骤5：参与活动
  - 定期团体活动（聚会、任务、考核）
  - 团体内社交（认识其他成员）
  - 团体资源使用（学习、共享）
```

### 4.2 团体内晋升

```
团体内晋升的四种方式：

1. 技能晋升型（苍龙学术团体）
   - 技能等级提升 + 考核通过
   - 例：五行学院学徒 → 弟子 → 高级弟子

2. 战功晋升型（霜狼军事团体）
   - 战功累积 + 同僚认可
   - 例：战团新兵 → 战士 → 副首领

3. 金钱晋升型（金雀花商业团体）
   - 财富累积 + 契约续签
   - 例：普通会员 → 高级会员 → 核心会员

4. 贡献晋升型（边境团体）
   - 完成团体任务 + 奮献认可
   - 例：新手佣兵 → 老手佣兵 → 精英佣兵
```

### 4.3 团体提供的资源

| 资源类型 | 说明 | 示例 |
|----------|------|------|
| **技能学习** | 团体内NPC可以传授技能 | 铁匠工会内找工匠长学锻造 |
| **材料共享** | 团体成员可以借用材料 | 锻造工会共享稀有矿石 |
| **工具借用** | 使用团体共有的高级工具 | 学院借用魔法研究设备 |
| **信息优先** | 团体成员优先获得情报 | 商会优先获得商路情报 |
| **任务渠道** | 团体提供专属任务来源 | 佣兵工会任务板 |
| **社交网络** | 认识同业NPC | 团体聚会结识同行 |
| **身份头衔** | 获得"××成员"头衔 | "五行学院弟子"头衔 |
| **保护互助** | 团体成员互相帮助 | 工匠受伤时工会资助 |

### 4.4 团体的义务与约束

```
加入团体不只是"获得好处"，也有义务：

技能传承义务（学术/工艺团体）：
  - 高等级成员需要教导低等级成员
  - 或定期贡献研究成果

缴费义务（商业团体）：
  - 定期缴纳会费
  - 金雀花契约团体违约需付惩罚金

服役义务（军事团体）：
  - 需要响应团体的军事行动
  - 拒绝可能被降级或驱逐

保密义务（魔法/情报团体）：
  - 不能泄露团体秘密
  - 泄露可能被追责甚至诅咒

行为约束（所有团体）：
  - 不能损害团体利益
  - 不能背叛团体成员
  - 违反规则可能被驱逐或惩罚
```

## 五、团体与阵营的关系

### 5.1 团体的政治立场

```
团体的政治立场分三类：

阵营绑定型：
  - 团体明确效忠某阵营
  - 例：苍龙正规军团、霜狼战团
  - 加入 = 效忠阵营
  - 玩家需考虑阵营关系

阵营倾向型：
  - 团体倾向某阵营，但不强制效忠
  - 例：五行学院（倾向苍龙）、银行公会（倾向金雀花）
  - 加入不影响阵营声望，但活动可能影响

阵营中立型：
  - 团体声称中立，服务所有阵营
  - 例：治愈师联盟、佣兵国际
  - 加入不影响阵营关系
  - 但团体可能被阵营渗透
```

### 5.2 团体影响政策

```
大型团体可以影响阵营政策：

商业团体影响经济政策：
  - 银行公会 → 影响利率、货币发行
  - 粮商联盟 → 影响粮价、储备政策

学术团体影响文化政策：
  - 太学 → 影响科举、教育
  - 五行学院 → 影响魔法法规

军事团体影响军事政策：
  - 军团 → 影响军费、部署
  - 佣兵工会 → 影响佣兵法规

玩家可以利用团体：
  - 加入团体 → 获得团体支持 → 影响政策
  - 例：加入银行公会 → 公会支持你的经济改革提案
```

## 六、团体的生命周期

### 6.1 团体也会变化

```
团体不是永恒的，会随世界变化：

兴盛：
  - 成员增加 → 团体影响力扩大
  - 触发条件：某技能热门、某行业繁荣
  - 例：战争期间佣兵工会兴盛

衰落：
  - 成员减少 → 团体影响力下降
  - 触发条件：某技能冷门、某行业萧条
  - 例：和平时期角斗者协会衰落

分裂：
  - 内部矛盾 → 团体分裂成多个小团体
  - 触发条件：理念冲突、利益争夺
  - 例：商会因分配问题分裂成两派

消亡：
  - 成员全失 → 团体消失
  - 触发条件：长期无人加入、核心人物死亡
  - 例：某小村落的民兵队解散

新生：
  - 新需求出现 → 新团体诞生
  - 触发条件：新技能出现、新行业发展
  - 例：契约炼成师协会（新技术催生新团体）
```

### 6.2 团体与关键NPC的关系

```
团体与关键NPC的关系：

团体领袖型：
  - NPC是团体领导者
  - 例：契语 → 契约法师塔领袖
  - 玩家与NPC关系 = 与团体关系

团体成员型：
  - NPC是团体普通成员
  - 例：铁匠·老锤 → 铁匠工会成员
  - 玩家与NPC关系 ≠ 与团体关系（但NPC可推荐）

团体依赖型：
  - NPC依赖某团体但不属于它
  - 例：金河 → 依赖皇商公会但不直接领导
  - 玩家需要区分NPC个人 vs 团体
```

```typescript
interface Organization {
  id: string;
  name: string;
  type: 'guild' | 'academy' | 'union' | 'military' | 'network' | 'council';

  // 所属阵营
  faction?: 'canglong' | 'shuanglang' | 'jinque' | 'border' | 'international';

  // 组织形态
  structure: 'tight' | 'loose' | 'elite' | 'secret';

  // 总部位置
  headquarter: string;  // 城市或村落ID

  // 门槛条件
  requirements: {
    skills?: { skillId: string; minLevel: number }[];
    resources?: { type: string; amount: number }[];
    relations?: { npcId: string; minRelation: number }[];
    factionRep?: { faction: string; minRep: number }[];
    special?: string;  // 特殊条件描述
  };

  // 成员等级
  ranks: OrganizationRank[];

  // 提供的资源
  benefits: OrganizationBenefit[];

  // 成员义务
  obligations: OrganizationObligation[];

  // 当前状态
  status: {
    memberCount: number;
    influence: number;  // 团体影响力 0-100
    isActive: boolean;
  };

  // 关键人物
  keyMembers: {
    leader?: string;    // 团体领袖NPC ID
    managers: string[]; // 管理层NPC IDs
  };
}

interface OrganizationRank {
  id: string;
  name: string;
  level: number;        // 等级高低
  title: string;        // 对应头衔

  // 晋升条件
  promotionRequirements: {
    skills?: { skillId: string; minLevel: number }[];
    contribution?: number;  // 财务/战功贡献
    time?: number;          // 在团体时间（游戏月）
    special?: string;
  };

  // 该等级的权利
  privileges: string[];

  // 该等级的义务
  duties: string[];
}

interface OrganizationBenefit {
  type: 'skill_learning' | 'resource_access' | 'information' | 'task_channel' |
        'social_network' | 'protection' | 'title' | 'political_influence';
  description: string;
  rankRequired: number;  // 需要达到的等级
}

interface OrganizationObligation {
  type: 'fee' | 'service' | 'teaching' | 'confidentiality' | 'loyalty';
  description: string;
  consequence: string;   // 违反后果
}
```

## 九、团体叙事设计

### 9.1 团体相关的事件类型

```
团体可以触发的事件：

入会事件：
  "你申请加入铁匠工会，需要完成入会考验..."
  → 玩家完成任务 → 成功加入

团体内部事件：
  "铁匠工会召开年度大会，你作为成员被邀请参加..."
  → 团体聚会 → 玩家可以社交、学习

团体间事件：
  "铁匠工会与木匠工会发生资源争夺纠纷..."
  → 玩家可以选择立场

团体与阵营事件：
  "银行公会向朝廷施压，要求降低利率..."
  → 玩家可以参与游说

团体危机事件：
  "佣兵工会经费紧张，可能削减福利..."
  → 玩家可以选择帮助或退出
```

### 9.2 团体叙事示例

```
【加入铁匠工会】

事件触发：
  玩家向铁匠·老锤询问："有没有铁匠的组织？"

老锤回答：
  "有啊，铁匠工会。不过入会需要通过考验。
   你得独立完成一件作品，让工匠长认可才行。
   我可以帮你推荐，但考验还是要你自己过。"
```
