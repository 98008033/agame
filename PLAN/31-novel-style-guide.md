### 1.2 语言风格规范

#### 句子长度规范

```
标准句长分布：
──────────────────────────────────────────────────────────────

短句（10字以内）：     30%  ─ 节奏把控、动作描写、强调
中句（10-30字）：      50%  ─ 主叙述、对话、说明
长句（30-50字）：      15%  ─ 场景描写、心理独白
超长句（50字以上）：   5%   ─ 复杂因果、史诗段落

示例：
──────────────────────────────────────────────────────────────

【错误】过于古典：
"时维九月，序属三秋，潦水尽而寒潭清，烟光凝而暮山紫..."
（太长、太古，现代读者不适应）

【正确】网文风格：
"秋风已凉。霜牙城外的冰原上，极光正缓缓飘动。
苍狼站在城头，目光向北——那里是传统的方向，
也是改革的起点。"
（节奏明快，信息明确，悬念自然）

【短句节奏】：
"天枢站起身。
他没有说话。
只是看向窗外。
那里，父皇的寝宫正在沉睡。"
（用短句制造紧张感）

【中句叙述】：
"金潮放下茶杯，手指轻叩桌面三下。
这是他的习惯——每当要做出重要决定，他总会这样做。"
（信息充分，节奏舒适）
```

#### 段落节奏规范

```
段落长度规范：
──────────────────────────────────────────────────────────────

基础段落：      3-5句，约50-100字
过渡段落：      1-2句，约20-40字（用于场景切换）
重点段落：      5-8句，约100-200字（关键剧情）
描写段落：      4-6句，约80-150字（环境/人物）

段落间隔策略：
──────────────────────────────────────────────────────────────

每3-5个基础段落后，需要一个：
- 过渡段落（节奏调整）
- 对话段落（人物互动）
- 或描写段落（氛围渲染）

示例段落节奏：
──────────────────────────────────────────────────────────────

[基础段落] 交代情境
[基础段落] 事件发展
[对话段落] 人物互动
[过渡段落] 场景切换
[描写段落] 环境渲染
[基础段落] 继续推进
[重点段落] 关键转折
```

#### 悬念密度规范

```
悬念设置频率：
──────────────────────────────────────────────────────────────

章内悬念：      每500-800字一个小钩子
节内悬念：      每节结尾一个中等悬念
章终悬念：      每章结尾一个强悬念

悬念类型：
──────────────────────────────────────────────────────────────

1. 疑问悬念："但苍狼不知道的是，血狼已经做出了决定。"
2. 预告悬念："三天后，一个改变联邦命运的消息将抵达霜牙城。"
3. 人物悬念："那个神秘访客，到底是什么身份？"
4. 冲突悬念："天枢能瞒过破军的眼睛吗？"
5. 物品悬念："这封信的内容，将决定一切。"

悬念强度等级：
──────────────────────────────────────────────────────────────

Level 1（微悬念）：随下一段解开
  "他看向窗外，那里有什么在等待他。"
  → 下一段揭晓："是父皇传来的密信。"

Level 2（小悬念）：数段后解开
  "金潮的眉头紧锁——这笔账算不通。"
  → 后续段落逐步揭示原因

Level 3（中悬念）：本节内解开
  "但最关键的情报，他还没收到。"
  → 本节末尾揭示情报

Level 4（强悬念）：跨章解开
  "这天夜里，一个身影悄无声息地离开了暮光村..."
  → 下章揭示去向和目的
```

## 二、四阵营叙事语气差异

### 2.1 苍龙帝国：史诗权谋风格

```

│                    苍龙帝国叙事语气                           │

│                                                              │

│                                                              │
│  人物称呼：                                                  │
│  - 皇室：殿下、陛下、阁下                                    │

│                                                              │
│  场景描写特征：                                              │
│  ───────────────────────────────────────────────────────    │
│  - 建筑：恢弘、肃穆                                          │
│  - 气氛：压抑、凝重                                          │
│  - 光线：昏暗、幽深                                          │

│                                                              │
```

#### 苍龙叙事示例

```
【标准苍龙段落】

帝都的夜色总是来得特别沉重。

天枢府中，烛火摇曳，映照着墙上那幅先帝亲笔题写的"隐忍"二字。
大皇子天枢端坐案前，手中握着一封密信——来自边境铁壁关的急报。

"父皇病重三月，"他低声自语，"而破军的手已经伸到了边境。"

他的声音平静如水，但那双眸子深处，暗流正在涌动。
这是夺嫡之争——一场没有硝烟、却比硝烟更致命的战争。

烛火跳动了一下。
天枢抬眼看向窗外。夜色中，帝都的轮廓若隐若现。
这座八百年的古城，此刻正屏息等待着某个命运的转折。

而他，将亲自书写那个转折。

【对话示例】

天枢： "阁下深夜来访，想必不是叙旧吧？"
谋士： "殿下明鉴。破军将军已开始布局，时机不容错过。"
天枢： "时机..."他轻叩桌面，"时机是等出来的，不是抢出来的。"
谋士： "殿下谨慎，在下佩服。但..."
天枢： "但什么？但父皇的病情？但朝臣的观望？还是...但我的犹豫？"
谋士： "殿下心中自有计较。在下只提醒一句：观望者，终将站队。"
天枢： （微笑）"你很聪明。但聪明人，往往短命。"
谋士： "殿下的仁慈，在下铭记于心。"
```

#### 霜狼叙事示例

```
【标准霜狼段落】

狼嚎谷的风从不停歇。

那声音像远古狼王的呼唤，穿透两座雪山的阻隔，直抵人心。
血狼部族的战士们三三两两走过红土广场，身上的伤疤是他们的勋章，
眼中的战意是他们的灵魂。

"改革？"一个壮汉啐了一口，"那是背叛！"

他的声音在风中炸开，引来一片附和。
血狼·屠灭站在血色角斗场的中央，双手负后，目光如刀。

他什么都没说。
但那沉默，比任何话语都更沉重。

因为所有人都知道——今晚的极光台，将是一场无声的决斗。
改革派和保守派，终将在这里碰撞。

而碰撞的结果，将是鲜血。

【对话示例】

血狼： "苍狼想改变霜狼？他疯了！"
烈火： "首领，时代变了。我们不能永远靠劫掠生存。"
血狼： "时代？"他冷笑，"时代是谁定的？是强者！"
烈火： "但强者的定义也在改变。"
血狼： "你小子，年轻气盛。走，去角斗场！"
烈火： "首领，角斗场解决不了问题。"
血狼： "角斗场解决一切！在霜狼，拳头就是道理！"
烈火： "道理...那是过去的道理。"
血狼： （怒吼）"在霜狼，过去就是现在！荣誉不死！"
```

```
【标准金雀花段落】

金雀花城的清晨总是从算盘声开始。

交易所的门还没开，就已经有人排队了。
经纪人、商人、投机者——他们手里的金币像心跳一样跳动，
等待着今天的开盘价。

"自由贸易法案！"一个年轻人高喊，"这是机会！"

金潮站在交易所二楼，俯视着这一切。
他的脸上没有表情，但那双眼睛正在计算——计算风险，计算收益，
计算这个法案能为他带来多少新的筹码。

在金雀花，一切皆有价格。
包括权力，包括荣誉，包括人心。

而金潮的价格，是整个王国。

【对话示例】

金潮： "自由贸易法案通过了。你准备好了吗？"
旧冠： "准备好了...让议会变成战场。"
金潮： "战场？"他轻笑，"阁下，这是市场。议会只是另一种交易所。"
旧冠： "交易所？"他皱眉，"比喻恰当，但...那是我们的议会！"
金潮： "议会是谁的？"他反问，"谁的票多，议会就是谁的。"
旧冠： "这是赤裸裸的金钱政治！"
金潮： "阁下，政治本来就是赤裸裸的。我只是让它更...高效。"
旧冠： "高效..."他咬牙，"你这是在毁掉金雀花的根基！"
金潮： "根基？"他看向窗外，"根基是我正在建的银行大厦。"
```

#### 边境叙事示例

```
【标准边境段落】

暮光村的暮光，是全世界最美的。

太阳还没落山，天空已经染成了金红紫色——
那是三国交界的地方，金雀花的金色、苍龙的红色、霜狼的紫色，
在这里融化成一片独一无二的暮光。

老根坐在大树下，手里捏着一杯奶茶，眼神平静。
他不看天空，看人——
暮色酒馆里进进出出的人，有商人、有士兵、有流浪者、有间谍...

他们都来边境，因为边境是自由。
而老根，是自由的守护者——至少他自认为是。

"三国都想这里，"他低声说，"但这里不属于三国。"

小苏在酒馆里擦着杯子，听见这话，笑了笑。
"老根叔，你又开始了。"
"我没开始，"老根摇头，"我只是提醒——自由的代价，很高。"

【对话示例】

小苏： "老根叔，龙使又在打听你了。"
老根： "打听我？"他喝茶，"打听我什么？"
小苏： "打听...你到底站哪边。"
老根： "哪边？"他看向窗外，"我站暮光村这边。"
小苏： "那三国都不满意。"
老根： "满意？"他笑了，"他们满意了，我就不满意了。"
小苏： "您这是...骑墙啊。"
老根： "骑墙？"他大笑，"小丫头，墙是用来骑的吗？墙是用来站的！"
小苏： "站着...也要选方向啊。"
老根： "方向？我选的是——让暮光村永远有暮光。"
```

## 三、人物出场描写模板库

### 3.1 主角NPC出场模板

#### 天枢（苍龙大皇子）

```
模板A：正式出场
──────────────────────────────────────────────────────────────
那人身形挺拔，眉目间带着三分沉稳、七分隐忍。
一袭青色长袍在烛光下泛着淡淡的光泽，腰间悬着一块玉佩——
那是先帝亲赐的嫡长子印信。

他负手而立，目光扫过众人，却像什么都没看。
嘴唇微微抿着，像是在等待什么，又像是在准备什么。

"天枢殿下。"有人低声唤道。

他没有立刻回应，只是缓缓转身，
那双眼睛在昏暗的光线中格外深邃——
像是八百年帝都的深夜，沉淀着无数未曾说出口的话。

模板B：日常场景
──────────────────────────────────────────────────────────────
天枢正在看书。

不是兵书，不是史书，是一本诗集。
烛光跳动，他的影子在墙上摇曳。

有人进门时，他只是抬眼，继续翻页。
那态度像在说：你来或不来，我都在读我的诗。
至于你带来的消息，早一秒晚一秒，没什么区别。

模板C：关键决策
──────────────────────────────────────────────────────────────
天枢站起身，走向窗边。

夜色中，帝都的轮廓若隐若现。
他的手扶着窗框，指尖轻叩——
这是他的习惯，每当要做出决定，指尖总会这样叩动。

三下。
停下。
再两下。

他转身的瞬间，眼神已经变了。
隐忍淡去，决断浮现。
"就这样做。"他说。
```

#### 苍狼（霜狼首领）

```
模板A：正式出场
──────────────────────────────────────────────────────────────
那个人站在冰原上，身形不像战士，像个商人。
但他的眼神是战士的——那双灰蓝色的眼睛，
像是冰原上最冷的风，却也带着最热烈的火焰。

他披着一件简朴的斗篷，腰间没有武器，
只有一串挂在脖上的狼牙项链——那是联邦首领的象征。

"苍狼。"有人呼唤他的名字，带着敬畏。

他微微点头，嘴角浮现一丝笑意。
那笑意温暖，却也坚定。
像是在说：变革的火，已经点燃。

模板B：演讲场景
──────────────────────────────────────────────────────────────
苍狼走上高台。

他没有华丽的开场，只是举起右手，
那手中握着一枚符文石——改革的符文石。

"霜狼的兄弟们！"
他的声音穿透风雪，像狼王的吼叫，
但内容不是战斗，是贸易。

"我们曾经靠刀剑生存，现在靠契约生存！"
台下一片哗然，他却继续说，
他的眼神始终坚定——像是看见了冰原上的新曙光。

模板C：沉思场景
──────────────────────────────────────────────────────────────
苍狼独自坐在火堆旁。

火焰跳动，映照着他脸上的疲惫。
他低头，看着手中的符文石，指尖摩挲那古老的纹路。

血狼在反对。灰熊在观望。雪鹰在沉默。
改革的路，比他想象的更难。

但他抬头时，那双灰蓝色的眼睛仍然燃烧。
火焰跳了一下，像是在回应他的决心。
```

#### 金潮（金雀花银行家）

```
模板A：正式出场
──────────────────────────────────────────────────────────────
那个人坐在交易所二楼，手边是一杯咖啡。
他的西装是金雀花城最精致的剪裁，
脸上的表情是金雀花城最标准的表情——
计算、等待、下一个机会。

有人走近时，他没有抬头，
只是手指轻叩桌面三下。
这是他的习惯，一个关于交易的习惯。

"金潮阁下。"

他终于抬眼，那目光像金色的秤——
在衡量你，衡量这笔交易的收益与风险。

模板B：谈判场景
──────────────────────────────────────────────────────────────
金潮放下茶杯。

那杯茶还热着，但他不急着喝。
谈判桌上，他总是最耐心的人。

"阁下的提议，"他缓缓开口，
"我听明白了。但阁下没说清楚——代价。"

他的手指在桌面轻叩，三下、两下、一下。
节奏像算盘，像心跳。

"在金雀花，一切皆有价格。"
他说这话时，眼睛始终盯着对方。
那眼神不像威胁，像提醒——提醒交易的基础规则。

模板C：幕后布局
──────────────────────────────────────────────────────────────
金潮站在窗前，俯视着金雀花城。

港口的船只来来往往，交易所的人流川流不息，
银行的窗口前排着长队。

这是他的王国。
他没戴皇冠，但手里握着的账簿，比皇冠更重。

他转身时，嘴角浮现一丝笑意。
那笑意不温暖，但也不冷漠，
只是精明——像一个成功商人完成交易后的满足。
```

#### 老根（边境联盟村长）

```
模板A：正式出场
──────────────────────────────────────────────────────────────
那老人坐在千年古树下，手里捏着一杯奶茶。
他的身形瘦削，脸上的皱纹像是暮光村每一年的风霜，
眼睛却清澈——像是那古树下的泉水。

有人走近时，他抬眼，微微点头。
那态度不像迎接贵客，像招呼邻居。

"老根叔。"那人唤他。

他笑了，那笑意温暖，却也带着某种深意。
像是说：边境的自由，是我守护的东西。
你来或走，都是客人。但别忘了规矩。

模板B：村议场景
──────────────────────────────────────────────────────────────
老根站起身，走向村心。

暮光正洒在他身上，金色、紫色、红色。
他抬头看了一眼天空，然后开口。

"三国都在争我们，"
他的声音不洪亮，但传得很远。
"但我们不争任何人。"

他说这话时，眼睛扫过三国使者的位置。
龙使、北风、金掌柜——他们都在角落里，都在观望。

老根的视线在每个人身上停留一秒，
然后收回，看向天空。

"我们是边境。"他说。
"自由的地方，不需要主人。"

模板C：沉思场景
──────────────────────────────────────────────────────────────
老根独自坐在树下。

暮光渐暗，他的影子在地上拉得很长。
他低头，看着那棵古树的根——那是千万根须，
盘根错节，像边境的历史。

三国争他。村民看他。自己...累了。

但暮光重新洒在他脸上时，
那双眼睛仍然清澈。
他抬头，看了一眼天空。
暮光还在。
自由还在。
```

## 五、对话风格规范

### 5.1 对话节奏规范

```
对话节奏控制：
──────────────────────────────────────────────────────────────

【紧张对话】
短句、快节奏、少停顿
──────────────────────
天枢： "说。"
谋士： "破军动手了。"
天枢： "何时？"
谋士： "今夜。"
天枢： （停顿一秒）"知道了。"

【谈判对话】
中句、节奏稳健、有停顿
──────────────────────
金潮： "阁下的提议，我听明白了。"
      （停顿）
      "但阁下没说清楚——代价。"
旧冠： "代价？"
金潮： "一切皆有价格。阁下，这是金雀花。"

【日常对话】
灵活节奏、轻松、可有插科打诨
──────────────────────
小苏： "老根叔，龙使又在打听你了。"
老根： "打听我什么？"
小苏： "打听...你站哪边。"
老根： （大笑）"我站暮光村这边！"
```

### 5.2 对话暗示技巧

```
潜台词技巧：
──────────────────────────────────────────────────────────────

【话中有话】
表面意思 + 暗示信息 + 人物反应

示例：
天枢： "父皇身体还好吗？"（表面关心）
谋士： （低头）"殿下，您该知道的。"（暗示病情）
天枢： （沉默）"知道了。"（接受暗示）

【反话正说】
正面表达 + 负面暗示

示例：
金潮： "阁下很聪明。"
      （微笑）
      "但聪明人，往往短命。"（表面夸奖，暗含威胁）

【沉默暗示】
不回应 + 动作暗示

示例：
血狼： （沉默，看向窗外）
      （手指轻叩桌面）
烈火： （领会）"首领，我明白了。"（从沉默中读懂）
```

## 六、错误示例与修正

### 6.1 语言风格错误

```
错误：过于古典
──────────────────────────────────────────────────────────────

【原文】
"时维九月，序属三秋。天枢端坐于府邸之中，
烛火摇曳，映照墙上先帝亲笔之'隐忍'二字..."

【问题】
- 过多古典句式
- "之"字使用过多
- 描写节奏拖沓

【修正】
帝都的秋夜总是沉重。
天枢府中，烛火摇曳。
墙上那幅"隐忍"，是先帝亲笔。

──────────────────────────────────────────────────────────────

错误：过于现代口语
──────────────────────────────────────────────────────────────

【原文】
"苍狼站在台上说：'兄弟们，我们要搞贸易，
不搞劫掠了。改革势在必行，时代变了！'"

【问题】
- 完全现代口语，无历史感
- "搞"字不适合架空世界
- 缺少人物气质

【修正】
苍狼走上高台，举起手中的符文石。
"霜狼的兄弟们！"
他的声音穿透风雪。
"我们曾经靠刀剑生存，现在——靠契约生存！"
```

### 6.2 悬念设置错误

```
错误：悬念过度
──────────────────────────────────────────────────────────────

【原文】
"但金潮不知道的是，旧冠已经决定...
三天后，在交易所...
一个神秘的访客...
而那个访客的身份...
将改变一切..."
（每句都有悬念，节奏混乱）

【问题】
- 悬念密度过高
- 悬念强度不一致
- 读者疲劳

【修正】
但金潮不知道的是，旧冠已经做出了决定。
三天后，那个决定将在交易所揭晓。
至于具体是什么——金潮很快就会知道。
```
