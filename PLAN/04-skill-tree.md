# 技能体系

> 所有人从同一个中心出发——基本生存技能。然后向不同方向发展。走战斗线就变成战士，走种田线就变成农夫，走商线就变成商人。每条线走到深处会和其他线交汇，交汇处产生新的高级技能。整个世界就是一棵大树，每个人在不同的枝条上。

## 从中心向外，一层一层的技能

### 中心 · 基本生存（所有人自动获得）

走路、吃饭、睡觉、砍柴、生火、搭帐篷、基本交流

---

### 第一层 · 6条起始路线（选择一条开始）

| 路线 | 入门技能 | 干什么 |
|------|----------|--------|
| 战斗 | 挥剑、闪避 | 拿起武器保护自己 |
| 魔法 | 感知魔力、基础施法 | 发现自己能用魔法 |
| 锻造 | 认识矿石、使用工具 | 学会做基本工具 |
| 经济 | 看秤、算账、讨价还价 | 学会买卖东西 |
| 社交 | 察言观色、套话 | 学会和人打交道 |
| 农牧 | 播种、浇水、喂牲口 | 学会种地养动物 |

---

### 第二层 · 深化（沿选定路线深入）

```
战斗线深化：
  ├─ 剑术          精通近战武器
  ├─ 格斗          空手搏击、摔跤
  └─ 远程射击      弓箭、投掷

魔法线深化：
  ├─ 火系法术      火球、灼烧、照明
  ├─ 水系法术      冰霜、治愈、净化
  ├─ 土系法术      护盾、地震、石化
  └─ 木系法术      植物操控、自然治愈

锻造线深化：
  ├─ 冶炼          把矿石变成金属
  ├─ 锻造          把金属变成武器/工具
  └─ 铭刻          在物品上刻符文

经济线深化：
  ├─ 贸易          低买高卖、跑商
  ├─ 价格操控      操纵市场、囤积居奇
  └─ 合约          签合同、管账目

社交线深化：
  ├─ 外交谈判     代表势力谈判
  ├─ 谍报侦查     搜集情报、跟踪
  └─ 识人         判断人品、预测行为

农牧线深化：
  ├─ 精耕          提高粮食产量
  ├─ 养殖          养牛养马养羊
  └─ 种药          种植草药、基础药理
```

### 第三层 · 跨线交汇（两条路线的技能组合，产生新技能）

| 组合 | 技能名 | 干什么 |

| 水系法术+种药 | 治愈师 | 治疗各种伤病 |

## 学习规则

| 规则 | 说明 |
|------|------|
| 中心 | 自动获得 |
| 第一层 | 选一条路线开始，可以后期补学其他路线 |
| 第二层 | 在选定路线内深入，跨路线学习需2倍经验 |
| 第三层 | 需要两条不同路线的第二层技能达标 |
| 第四层 | 需要在一条路线上深度投入 |
| 第五层 | 需要阵营声望≥效忠 + 特定前置技能 |
