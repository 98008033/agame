# Agame 技能体系（像素奇幻版）

> **核心理念**：技能不是数值，是"能参与的故事类型"。你的技能决定了你能看到什么样的世界。
>
> **视觉隐喻**：技能树以像素风格的"世界之树"呈现——所有人从同一个根出发，向不同方向生长。

## 技能层级详解

### 中心层 · 基本生存（所有人自动获得）

**像素视觉**：以树根形式呈现，深褐色像素根须

| 技能 | 等级 | 描述 |
|------|------|------|
| **基础生存** | Lv.1 | 走路、吃饭、睡觉、砍柴、生火、搭帐篷、基础交流 |

**解锁内容**：
- 野外生存事件
- 基础NPC对话
- 简单工作委托

---

### 第一层 · 入门技能（选择一条路线开始）

**像素视觉**：三根主树干向上生长，分别为青绿色（谋略）、铁锈色（武力）、金黄色（经营）

#### 【谋略线】情报分析

```
情报分析 (Intelligence Analysis)
├── Lv.1 察言观色
│   └── 能感知他人情绪和立场
├── Lv.2 流言搜集
│   └── 能获得市井传言和线索
└── Lv.3 线索整合
    └── 能从碎片信息中发现真相
```

**解锁事件类型**：
- 旁听对话、发现异常
- 解读密信、识别谎言
- 追踪线索、建立情报网

**像素图标**：32×32 放大镜与卷轴组合

#### 【武力线】战斗技巧

```
战斗技巧 (Combat Technique)
├── Lv.1 基础格斗
│   └── 能自保，对付普通强盗
├── Lv.2 武器掌握
│   └── 熟练使用一种武器
└── Lv.3 战术意识
    └── 理解战场态势和阵型
```

**解锁事件类型**：
- 街头格斗、护送任务
- 武器对决、小型战斗
- 精英对决、竞技场挑战

**像素图标**：32×32 交叉剑与盾牌

#### 【经营线】商业贸易

```
商业贸易 (Commerce)
├── Lv.1 识货估价
│   └── 能识别商品价值
├── Lv.2 讨价还价
│   └── 能在交易中获得优势
└── Lv.3 市场洞察
    └── 能预测价格波动
```

**解锁事件类型**：
- 小商贩、跑腿送货
- 开设店铺、组织商队
- 跨区域贸易、商业联盟

**像素图标**：32×32 金币与天平

### 第二层 · 精通技能（入门技能达到3级后解锁）

**像素视觉**：树枝分叉，更加粗壮，带有阵营色彩

#### 【谋略线】政治操控

```
政治操控 (Political Manipulation)
├── Lv.4 利益交换
│   └── 能在谈判中达成交易
├── Lv.5 派系建立
│   └── 能组建自己的政治势力
└── Lv.6 格局改变
    └── 能影响国家政策走向
```

**解锁事件类型**：
- 参与高层会议、影响决策
- 策划政变、扶持代理人
- 制定国家政策、改变政治格局

**前置要求**：情报分析 Lv.3

**像素图标**：32×32 权杖与天平

#### 【武力线】军事指挥

```
军事指挥 (Military Command)
├── Lv.4 小队指挥
│   └── 能指挥小队作战
├── Lv.5 战术布局
│   └── 能策划复杂军事行动
└── Lv.6 战役决策
    └── 能决定大规模会战走向
```

**解锁事件类型**：
- 指挥小队作战、攻城战
- 大规模会战、战略规划
- 决定国家命运的战役

**前置要求**：战斗技巧 Lv.3

**像素图标**：32×32 军旗与号角

#### 【经营线】产业管理

```
产业管理 (Industry Management)
├── Lv.4 资源配置
│   └── 能有效配置人力物力
├── Lv.5 产业扩张
│   └── 能建立跨地区商业网络
└── Lv.6 经济影响
    └── 能影响地区乃至国家经济
```

**解锁事件类型**：
- 管理矿山/农场、雇佣工人
- 建立产业网络、创新工艺
- 跨国商业帝国、制定行业标准

**前置要求**：商业贸易 Lv.3

**像素图标**：32×32 工厂与齿轮

## 技能获取途径

### 获取方式对比

| 方式 | 说明 | 效率 | 像素表现 |
|------|------|------|----------|
| **事件实践** | 参与相关事件获得经验 | ★★★★★ | 经验值粒子飞入技能图标 |
| **NPC传授** | 向有技能的NPC学习 | ★★★★☆ | NPC头像旁出现教学动画 |
| **自我修炼** | 消耗时间和资源修炼 | ★★★☆☆ | 角色进入修炼状态动画 |
| **特殊际遇** | 奇遇或重大转折 | ★★★★★ | 特殊光效和粒子特效 |

## 像素美术规范

### 技能图标设计

**尺寸**：32×32 像素（标准），48×48 像素（大师级）

**风格**：
- 入门技能：单色剪影风格
- 精通技能：彩色+简单高光
- 大师技能：彩色+发光效果+粒子装饰

**颜色编码**：
```
谋略线：青绿色系 #2E7D32 → #4CAF50
武力线：铁锈色系 #8D6E63 → #A1887F
经营线：金黄色系 #F57C00 → #FFB74D
```

### 技能树视觉呈现

**背景**：深空黑 #1A1A2E

**节点样式**：
- 未解锁：灰色轮廓 #4A4A5A
- 可解锁：闪烁边框，对应技能线颜色
- 已解锁：实心填充，对应技能线颜色
- 精通：节点周围有粒子光环

**连线样式**：
- 未激活：虚线，深灰色
- 已激活：实线，对应技能线颜色
- 组合激活：发光连线，金色

## 技术实现建议

### 数据结构

```typescript
// 技能定义
interface Skill {
  id: string;
  name: string;
  category: 'strategy' | 'combat' | 'business';
  maxLevel: number;
  prerequisites?: string[];
  unlocksNarrative: string[];
  pixelIcon: string;  // 图标资源路径
}

// 玩家技能数据
interface PlayerSkill {
  skillId: string;
  level: number;
  experience: number;
  unlockedAt: Date;
}

// 叙事分支条件
interface NarrativeBranch {
  id: string;
  description: string;
  requiredSkills: {
    skillId: string;
    minLevel: number;
  }[];
}
```

### 技能检查逻辑

```typescript
// 检查玩家能否看到某个叙事选项
function canAccessNarrativeBranch(
  playerSkills: PlayerSkill[],
  branch: NarrativeBranch
): boolean {
  return branch.requiredSkills.every(req => {
    const playerSkill = playerSkills.find(s => s.skillId === req.skillId);
    return playerSkill && playerSkill.level >= req.minLevel;
  });
}
```

## 与其他系统的关联

```
技能系统 ←→ 事件系统：技能决定能看到的事件选项
技能系统 ←→ NPC系统：高技能等级吸引相关NPC关注
技能系统 ←→ 阵营系统：某些高级技能需要阵营声望
技能系统 ←→ 叙事系统：技能是解锁叙事分支的钥匙
技能系统 ←→ 视觉系统：像素风格的技能树展示
```

## 总结

### MVP核心要点

1. **3条线 × 2个技能 = 6个专业技能 + 1个通用技能**
2. **技能等级最高6级**（入门1-3级，精通4-6级）
3. **技能直接决定可见的叙事分支**，而非提供数值加成
4. **技能获取主要通过参与事件**，符合游戏"叙事优先"的核心理念
5. **像素风格呈现**，与游戏整体视觉风格统一

### 后续扩展方向

- **大师级技能**（第7-9级）：MVP后逐步开放
- **传奇级**（第10级）：赛季制，极少数玩家能达到
- **阵营独有技能**：每个阵营的专属高级技能
- **更多跨线组合**：开发更多技能组合的独特叙事
