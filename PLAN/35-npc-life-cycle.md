# Agame NPC人生成长系统

> **核心理念**：NPC不只是功能性的"服务提供者"，而是有完整人生轨迹的真实人物。他们会经历出生、成长、学习、工作、结婚、生子、衰老、死亡——对应现实社会中人的成长历程，让NPC有真实的"人生感"。

---

## 一、人生阶段系统

### 1.1 七大人生阶段

NPC的一生分为七个阶段，每个阶段有不同的能力、责任和可能性：

| 阶段 | 年龄范围 | 称号 | 能力特点 | 社会角色 | 游戏时间占比 |
|------|----------|------|----------|----------|--------------|
| **幼年期** | 0-6岁 | 童稚 | 无行动能力，被动成长 | 家庭抚养对象 | ~10% |
| **少年期** | 7-14岁 | 少年 | 开始学习基础技能 | 学徒/学生 | ~15% |
| **青年期** | 15-30岁 | 青年 | 快速成长，建立职业基础 | 新手/初入社会 | ~25% |
| **壮年期** | 31-50岁 | 壮年 | 能力巅峰，承担主要社会职责 | 成熟职业者/领导者 | ~30% |
| **中年期** | 51-65岁 | 中年 | 能力稳定，经验丰富 | 专家/导师/管理者 | ~15% |
| **老年期** | 66-80岁 | 老年 | 能力衰退，智慧传承 | 退休/顾问/传承者 | ~5% |
| **终末期** | 80岁+ | 晚年 | 高死亡概率，临终传承 | 临终者 | 可变 |

### 1.2 各阶段详细特征

**幼年期（0-6岁）**
```
特征：
- 完全依赖父母或监护人
- 无独立行动能力
- 属性开始形成（遗传父母基因）
- 性格萌芽阶段

玩家可见内容：
- NPC介绍时显示："XX的孩子，5岁"
- 可能出现在父母身边的场景中
- 不参与任何事件

系统行为：
- 每月自动成长
- 属性受父母基因影响（加权平均±随机）
- 性格由LLM生成，受家庭环境影响
```

**少年期（7-14岁）**
```
特征：
- 开始学习基础技能（L1-L2）
- 可以做简单帮工
- 性格基本定型
- 开始建立社交圈（同龄人）

玩家可见内容：
- 可以与少年NPC对话
- 可以委托简单任务（如送信、打扫）
- 可以成为学徒关系的候选人

系统行为：
- 开始学习父母或导师的技能倾向
- 每月自动获得少量技能经验
- 可能被玩家或NPC收为学徒
- 性格特征明确化
```

**青年期（15-30岁）**
```
特征：
- 属性快速增长期
- 建立职业基础（L2-L4技能）
- 可独立承担社会角色
- 开始考虑婚姻和家庭

玩家可见内容：
- 正常的NPC交互
- 可以雇佣、合作、竞争
- 可以建立深层关系（朋友、对手、恋人）
- 可以参与所有事件

系统行为：
- 主动学习技能（根据职业倾向）
- 可能主动寻找工作机会
- 可能主动建立社交关系
- 可能提出婚姻请求（与同年龄段NPC）
```

**壮年期（31-50岁）**
```
特征：
- 属性达到峰值
- 技能等级可达L4-L6（大师级）
- 承担主要社会职责（领导、管理、经营）
- 家庭稳定期（养育子女）

玩家可见内容：
- 核心NPC层（城市管理者、商会领袖、军事指挥）
- 高技能等级，提供高级服务/教学
- 关系经营的主要对象

系统行为：
- 承担城市/组织的管理职责
- 主动做出决策（Agent NPC）
- 培养后代和弟子
- 积累社会影响力
```

**中年期（51-65岁）**
```
特征：
- 体魄、敏捷开始下降
- 智慧、感知、意志达到峰值
- 技能稳定或略有下降
- 开始培养继承人

玩家可见内容：
- 专家型NPC（高级教学、顾问）
- 可能主动传授技能
- 关系网成熟，可提供深层帮助

系统行为：
- 减少主动冒险行为
- 专注传承和教学
- 可能指定继承人
- 可能退居幕后角色
```

**老年期（66-80岁）**
```
特征：
- 属性明显衰退（尤其是体魄、敏捷）
- 战斗能力大幅下降
- 智慧、意志可能继续提升（经验补偿）
- 高死亡率阶段

玩家可见内容：
- 顾问/导师角色
- 可能提供历史知识和故事
- 死亡风险提示（玩家可以看到NPC的健康状态）

系统行为：
- 减少日常活动
- 可能主动安排遗产传承
- 死亡概率每游戏年递增（见死亡系统）
```

**终末期（80岁+）**
```
特征：
- 高死亡概率（每年40%+）
- 可能临终传承
- 可能留下遗嘱和遗产

玩家可见内容：
- 可能见证NPC临终场景
- 可能继承NPC遗产（如果有关系）
- 可能被委托传承任务

系统行为：
- 高概率自然死亡
- 临终前可能触发特殊事件
- 死亡后遗产分配
```

## 二、职业发展系统

### 2.1 NPC职业类型

NPC的职业不是预设的，而是根据技能学习自然形成的：

**基层职业**（Template NPC为主）
```
服务类：
├── 商贩（商业L1-L2）
├── 店主（商业L2-L3）
├── 钱庄职员（商业L2）
├── 账房（商业L2+学识L1）
└── 集市管事（商业L2+行政L1）

生产类：
├── 农夫（农耕L1-L2）
├── 矿工（采矿L1-L2）
├── 木匠（工艺L1-L3）
├── 铁匠（锻造L1-L3）
├── 裁缝（工艺L1-L2）
└── 药剂师（炼金L1-L2）

军事类：
├── 士兵（战斗L1-L2）
├── 卫兵（战斗L2）
├──斥候（战斗L1+情报L1）
├── 保镖（战斗L2-L3）
└── 民兵队长（战斗L2+军事指挥L1）

医疗类：
├── 医师（治愈L1-L3）
├── 药师（治愈L1+学识L1）
└── 军医（治愈L2-L3）

情报类：
├── 信使（情报L1）
├── 消息贩子（情报L1-L2）
└── 探子（情报L2-L3）
```

**中层职业**（Agent NPC / 高级Template）
```
管理者：
├── 村长（行政L2+关系经营）
├── 店铺掌柜（商业L3+管理L2）
├── 镖局镖头（战斗L3+管理L2）
├── 军队队长（战斗L3+军事指挥L2）
└── 矿场管事（采矿L2+管理L2）

专家：
├── 高级铁匠（锻造L3-L4）
├── 高级医师（治愈L3-L4）
├── 商人（商业L3-L4）
├── 教头（战斗L3+教学L2）
└── 学者（学识L3-L4）
```

**高层职业**（Agent NPC）
```
领袖：
├── 城主/将军（军事指挥L4+）
├── 商会会长（商业L4+）
├── 部族首领（战斗L4+或政治L4+）
└── 学院山长（学识L4+）

决策者：
├── 部长/尚书（政治操控L4+）
├── 首辅（政治操控L5+）
├── 军团统帅（军事指挥L5+）
└── 金融巨头（产业管理L5+）
```

### 2.2 职业发展路径

NPC的职业发展遵循自然成长规律：

**基层NPC发展示例**
```
铁匠学徒 → 铁匠 → 高级铁匠 → 铁匠大师 → 铁匠村村长

阶段1：少年期（12-18岁）
  身份：铁匠学徒
  技能：锻造L1
  活动：帮师傅打杂，学习基础锻造

阶段2：青年期（18-25岁）
  身份：独立铁匠
  技能：锻造L2
  活动：开始独立接活，开设小铺

阶段3：壮年期（25-40岁）
  身份：高级铁匠
  技能：锻造L3-L4
  活动：承接高级订单，可能收徒

阶段4：中年期（40-55岁）
  身份：铁匠大师
  技能：锻造L4-L5
  活动：专注传承，可能成为铁匠村技术权威

阶段5：老年期（55岁+）
  身份：铁匠村村长/顾问
  技能：锻造L5（能力下降）
  活动：退居幕后，培养继承人
```

**军事NPC发展示例**
```
士兵 → 小队长 → 队长 → 将军 → 边境统帅

阶段1：青年期（15-22岁）
  身份：新兵
  技能：战斗L1
  活动：参与训练和基础任务

阶段2：青年期（22-30岁）
  身份：精锐士兵
  技能：战斗L2-L3
  活动：参与战斗，积累军功

阶段3：壮年期（30-40岁）
  身份：队长
  技能：战斗L3+军事指挥L2
  活动：带领小队，参与战役决策

阶段4：壮年期（40-50岁）
  身份：将军
  技能：军事指挥L3-L4
  活动：指挥大规模作战

阶段5：中老年（50岁+）
  身份：边境统帅
  技能：军事指挥L4-L5
  活动：战略规划，培养接班人
```

**商人类发展示例**
```
跑腿 → 小商贩 → 店主 → 商人 → 商会会长

阶段1：少年期（14-18岁）
  身份：店铺帮工
  技能：商业L1
  活动：送货、记账帮手

阶段2：青年期（18-28岁）
  身份：小商贩/店主
  技能：商业L2
  活动：独立经营小生意

阶段3：壮年期（28-40岁）
  身份：商人
  技能：商业L3-L4
  活动：扩张贸易网络，可能加入商会

阶段4：壮年期（40-55岁）
  身份：商会代表
  技能：商业L4+产业管理L3
  活动：参与商会决策，影响地区经济

阶段5：中老年（55岁+）
  身份：商会会长
  技能：产业管理L4-L5
  活动：商会领袖，培养继承人
```

### 2.3 职业晋升机制

**晋升条件**
```
晋升需要同时满足：
1. 技能等级达标（如：队长需要战斗L3）
2. 年龄适宜（通常青年期后才可担任中层）
3. 关系网支持（需要有推荐人或足够声望）
4. 职位空缺（上级职位有空位）

晋升方式：
├── 自然晋升（积累经验达到条件）
├── 推荐晋升（上级NPC推荐）
├── 竞争晋升（多人竞争同一职位）
├── 突发晋升（上级死亡/离职导致空缺）
└── 玩家干预（玩家推荐或任命）
```

**晋升事件示例**
```
场景：铁匠铺学徒晋升为独立铁匠

系统检测：
- 年龄：18岁 ✓
- 技能：锻造L2 ✓
- 师傅评价：良好 ✓
- 铺位空缺：城镇有可用铺位 ✓

触发事件：
"铁匠·老锤的学徒·小锤已完成学徒期。
师傅建议他开设自己的铁匠铺。
"

玩家可见选项：
[推荐小锤开设铺位]（需要声望）
[提供资金支持]（需要金钱）
[保持观望]
[介绍小锤给其他铁匠铺]
```

## 三、技能学习系统

### 3.1 NPC技能获取途径

NPC的技能获取遵循与玩家相同的原则，但有自动化的成长路径：

| 途径 | 适用场景 | 效率 | 说明 |
|------|----------|------|------|
| **遗传倾向** | 幼年期 | 高 | 继承父母的技能倾向（基因） |
| **师徒传授** | 少年期-青年期 | ★★★★★ | 向父母或导师学习基础技能 |
| **工作实践** | 青年期-壮年期 | ★★★★☆ | 通过日常工作积累技能经验 |
| **自我提升** | 全年龄段 | ★★★☆☆ | 主动阅读、训练、研究 |
| **事件触发** | 任何时期 | ★★★★★ | 参与重大事件获得大量经验 |

### 3.2 技能遗传与培养

**基因遗传**
```
父母技能倾向影响子女初始技能潜力：

技能潜力计算：
子代潜力 = (父代技能倾向 × 0.4 + 母代技能倾向 × 0.4 + 随机值 × 0.2)

示例：
父亲：铁匠（锻造倾向高）
母亲：裁缝（工艺倾向中）
子女：锻造潜力60% + 工艺潜力30% + 随机潜力10%
      → 更可能成为铁匠或相关工艺职业
```

**师徒培养**
```
NPC可以成为其他NPC（或玩家）的学徒：

学徒条件：
- 年龄：少年期或青年期
- 与导师关系：良好以上
- 导师技能等级：L3以上
- 导师愿意收徒

培养过程：
- 每月获得技能经验（导师技能等级 × 10）
- 可能学习导师的独门技巧
- 培养期通常3-5年
- 培养结束后可能继承导师职位

培养事件：
"秋实的弟子·望舒正在学习政治操控技巧。
当前进度：政治操控L2 → L3
预计完成时间：2年
"
```

### 3.3 技能成长时间表

```
少年期（7-14岁）：
├── 基础技能学习（L1-L2）
├── 受父母影响选择技能方向
└── 培养期，无独立技能使用

青年期（15-30岁）：
├── 技能快速增长（L2-L4）
├── 开始独立使用技能工作
├── 可能更换技能方向（转型期）
└── 建立职业基础

壮年期（31-50岁）：
├── 技能达到峰值（L4-L6）
├── 可能收徒传授技能
├── 技能组合产生新能力
└── 技能突破概率最高

中年期（51-65岁）：
├── 技能稳定或略降
├── 专注传承而非提升
├── 可能创造新技术
└── 技能突破概率降低

老年期（66岁+）：
├── 技能使用能力下降（体力限制）
├── 技能知识可传承
├── 技能等级可能降低（不使用）
└── 临终可能传授独门技巧
```

## 四、婚姻与家庭系统

### 4.1 NPC婚姻机制

NPC会主动考虑婚姻和家庭：

**婚姻条件**
```
年龄条件：
├── 最低年龄：16岁（部分地区可能不同）
├── 最佳婚龄：20-35岁
└── 晚婚：35岁以上（可能有社会压力）

社会条件：
├── 有足够经济基础（至少能养家）
├── 有稳定职业或技能
├── 家庭同意（父母或监护人）
└── 可能需要彩礼/嫁妆

关系条件：
├── 与配偶关系达到"友好"以上
├── 可能是自由恋爱
├── 可能是父母安排
└── 可能是政治/商业联姻
```

**婚姻类型**
```
自由恋爱：
├── NPC主动追求其他NPC
├── 需要积累好感度
├── 玩家可能被NPC追求（青年期NPC）
└── 玩家可以撮合NPC婚姻

父母安排：
├── NPC父母主动安排子女婚姻
├── 考虑家族利益
├── 可能违背子女意愿
└── 玩家可能影响父母决策

政治/商业联姻：
├── 高层NPC常见（贵族、商人）
├── 考虑家族/组织利益
├── 可能跨越阵营
└── 玩家可能促成或阻止

临终遗愿：
├── 老年NPC可能安排子女婚姻
├── 作为遗产的一部分
└── 玩家可能见证或干预
```

### 4.2 家庭结构

**家庭类型**
```
核心家庭：
├── 父母 + 子女（最常见）
├── 通常2-4个子女
└── 子女成年后可能离开

扩展家庭：
├── 父母 + 子女 + 孙子女
├── 老年NPC与成年子女同住
└── 常见于农村或小城镇

单亲家庭：
├── 一方父母去世或离婚
├── 子女由单方抚养
└── 可能由祖父母辅助

无子女家庭：
├── 未婚NPC
├── 已婚但无子女
└── 可能收养孤儿或弟子
```

**子女抚养**
```
生育：
├── NPC婚后可能生育子女
├── 生育概率受年龄影响（青年期最高）
├── 子女数量受经济条件影响
└── 可能生育多个子女（间隔2-3年）

抚养：
├── 幼年子女完全依赖父母
├── 少年子女开始学习父母技能
├── 父母为子女选择教育和职业方向
└── 玩家可能影响NPC子女的发展

教育：
├── 父母传授基础技能
├── 可能送子女学徒或上学
├── 高层NPC可能聘请导师
└── 玩家可能成为NPC子女的导师
```

### 4.3 家庭事件示例

```
【婚姻事件】
"铁匠·老锤的女儿·针线（18岁）与裁缝·剪子（22岁）表达了结婚意愿。

老锤希望女儿嫁给一个有稳定职业的人。
剪子刚开设裁缝铺，职业前途看好。

老锤询问你的意见（关系友好）。
"

选项：
[支持这桩婚姻]（老锤好感+10）
```

## 五、社会地位系统

### 5.1 地位层级

NPC的社会地位由多种因素决定：

| 层级 | 称号 | 决定因素 | 人数占比 | 影响力范围 |
|------|------|----------|----------|------------|
| **底层** | 平民 | 无特殊身份，依赖生存技能 | 60% | 个人/家庭 |
| **中层** | 专业人士 | 技能等级L3+，有稳定职业 | 30% | 本地/本行业 |
| **上层** | 领导者 | 技能等级L4+，管理职位 | 8% | 区域/组织 |
| **顶层** | 决策者 | 技能等级L5+，高层职位 | 2% | 国家/阵营 |

### 5.2 地位变化因素

**地位上升因素**
```
├── 技能提升（技能等级达到L3+）
├── 获得职位（被任命或推举）
├── 婚姻联姻（与高层NPC结婚）
├── 重大成就（完成任务、创新、战役）
├── 玩家帮助（玩家推荐或提拔）
└── 继承遗产（继承父母职位或财产）

地位上升事件：
"铁匠·小锤的锻造技艺达到L3。
镇上商会邀请他加入，成为技术顾问。
他的地位从'普通铁匠'提升到'技术专家'。
"
```

**地位下降因素**
```
├── 技能衰退（不使用导致等级下降）
├── 失去职位（被罢免、辞职）
├── 家庭变故（离婚、配偶死亡）
├── 社会事件（犯罪、失信、丑闻）
├── 经济失败（破产、负债）
└── 年龄衰老（老年后地位可能下降）

地位下降事件：
"商人·丝绸商的店铺破产。
他从'商会成员'降为'负债者'。
其他商人开始避开他。
"
```

### 5.3 地位影响

**高地位NPC特点**
```
玩家交互：
├── 更难建立关系（需要更高声望）
├── 提供更高级服务/教学
├── 可能主动联系玩家（如果玩家有足够地位）
└── 可能参与高层事件

社会行为：
├── 参与决策会议
├── 影响其他NPC命运
├── 可能推荐/任命其他NPC
└── 可能收徒或培养继承人
```

**低地位NPC特点**
```
玩家交互：
├── 较易建立关系
├── 提供基础服务
├── 可能寻求玩家帮助
└── 主要参与基层事件

社会行为：
├── 受地位高的NPC影响
├── 可能被雇佣或提拔
├── 可能寻求改变命运的机会
└── 可能成为高地位NPC的弟子
```

**死亡率**
```
自然死亡（老病）：
├── 60-65岁：每游戏年3%
├── 65-70岁：每游戏年8%
├── 70-75岁：每游戏年15%
├── 75-80岁：每游戏年25%
└── 80岁+：每游戏年40%

非自然死亡：
├── 战斗死亡（参与冲突）
├── 疾病死亡（瘟疫、重病）
├── 意外死亡（探索、事故）
└── 暗杀死亡（被玩家或NPC暗杀）
```

```
当人口过少时：
├── 提高生育率
├── 引入移民（事件触发）
├── 延长NPC寿命（医疗改善）
└── Chronos注入新人

当人口过多时：
├── 提高死亡率（疾病、战争）
├── NPC迁出（事件触发）
├── 降低生育率
└── Chronos平衡
```

**关键NPC更替**
```
当关键NPC（Agent类型）死亡时：

├── 无继承人时触发继承危机
└── 继承人可能能力不如前任

├── 关键NPC平均任职：15-25年
```

### 9.2 人生阶段更新逻辑

```typescript
function updateNPCLifeStage(npc: NPCLife): void {
  // 根据年龄更新阶段
  if (npc.currentAge < 7) npc.lifeStage = 'infant';
  else if (npc.currentAge < 15) npc.lifeStage = 'child';
  else if (npc.currentAge < 31) npc.lifeStage = 'youth';
  else if (npc.currentAge < 51) npc.lifeStage = 'adult';
  else if (npc.currentAge < 66) npc.lifeStage = 'middle';
  else if (npc.currentAge < 80) npc.lifeStage = 'elder';
  else npc.lifeStage = 'terminal';

  // 更新死亡概率
  npc.health.deathProbability = calculateDeathProbability(npc.currentAge, npc.health.current);

  // 触发人生阶段事件
  if (isLifeStageTransition(npc)) {
    triggerLifeStageEvent(npc);
  }
}
```

### 9.4 继承处理逻辑

```typescript
function handleNPCDeath(npc: NPCLife): void {
  // 通知玩家
  notifyDeath(npc);

  // 处理继承
  if (npc.inheritance.heir) {
    transferAssets(npc, npc.inheritance.heir);
  } else {
    triggerInheritanceCrisis(npc);
  }

  // 处理职位空缺
  if (npc.occupation.current) {
    handlePositionVacancy(npc);
  }

  // 处理家庭关系
  updateFamilyRelations(npc);
}
```

## 十、总结

### 10.1 NPC人生系统核心要点

1. **七大人生阶段**：从幼年到终末，每个阶段有不同特点
2. **职业自然发展**：职业由技能学习形成，非预设
3. **技能遗传与培养**：基因倾向 + 师徒传授 + 工作实践
4. **婚姻与家庭**：NPC会主动考虑婚姻和生育
5. **社会地位变化**：随技能、职位、成就动态变化
6. **人生关键时刻**：出生、入学、就业、结婚、晋升、退休、死亡
7. **玩家深度参与**：可影响NPC人生的多个阶段
8. **代际更替**：关键NPC死亡触发继承，基层NPC自动更替

### 10.2 与其他系统的关联

```
NPC人生系统 ←→ 技能系统：技能决定职业，职业决定地位
NPC人生系统 ←→ 属性系统：属性随年龄变化，影响能力
NPC人生系统 ←→ 关系系统：家庭、师徒、婚姻建立关系网
NPC人生系统 ←→ 事件系统：人生关键时刻触发事件
NPC人生系统 ←→ 叙事系统：NPC人生成为世界故事的一部分
NPC人生系统 ←→ Chronos：Chronos平衡人口和关键NPC分布
```

### 10.3 MVP简化建议

**必须实现（P0）**：
- 基础人生阶段（青年、壮年、老年）

- 死亡与继承系统
- 基础婚姻事件

**延后实现（P1/P2）**：
- 完整幼年和少年期
- 复杂家庭结构
- 详细的人生关键时刻事件
- 人口平衡机制
