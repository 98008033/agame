## 一、效忠选择类（5个）

### 事件1：苍龙使节的邀请

**触发条件**:
- 玩家影响力 ≥ 30
- 与苍龙阵营声望 ≥ 20
- 未效忠任何阵营
- 游戏日 ≥ 10

**事件描述**:
天都来的使节在暮光村的酒馆里等你。他穿着苍龙帝国的官服，腰间挂着六部令牌。

"陛下闻边境有能者，特遣在下前来相邀。苍龙帝国沃野千里，正是英雄用武之地。若阁下愿为帝国效力，六部之中必有你的位置。"

他放下一个檀木盒子，里面是一枚青铜腰牌。

"这是吏部预授的九品官身。接受它，你便是帝国的人。"

**选项**:

| 选项 | 描述 | 后果 |
|------|------|------|
| A | 接受效忠 | 金币 +200，苍龙声望 +50，获得"苍龙属臣"身份，解锁苍龙阵营任务，霜狼/金雀花声望 -10 |
| B | 婉言拒绝 | 苍龙声望 -20，获得"独立"标签，影响力 +10，触发后续"被监视"事件概率+30% |
| C | 拖延观望 | 苍龙声望 +5，获得3天时间考虑，3日内再次触发此事件 |

### 事件2：霜狼战士的决斗邀约

**触发条件**:
- 玩家武力属性 ≥ 60
- 与霜狼联邦声望 ≥ 10
- 游戏日 ≥ 15

**事件描述**:
一个披着狼皮的霜狼战士堵在村口。他的符文剑插在土里，双手抱胸。

"听说你是个有本事的。冰爪·铁心大人派我来——霜狼联邦不需要软蛋，但尊重强者。"

他拔出剑，指向你。

"两种选择：跟我打一场，赢了就能见铁心大人，也许能加入我们的战团。不打也行，但霜狼只看得起敢拔剑的人。"

**选项**:

| 选项 | 描述 | 后果 |
|------|------|------|
| A | 接受决斗 | 50%胜率判定：胜利→霜狼声望+40，获得"战团预备"身份；失败→金币-50（医药费），霜狼声望+20，获得"有胆量"标签 |
| B | 拒绝决斗 | 霜狼声望 -30，获得"懦夫"标签（持续30天），被霜狼NPC轻视 |
| C | 提议以智谋比试 | 需要外交≥50：成功→霜狼声望+30；失败→霜狼声望-15 |

### 事件3：金雀花的商业契约

**触发条件**:
- 玩家商业属性 ≥ 50
- 金币 ≥ 500
- 与金雀花声望 ≥ 0

**事件描述**:
金算盘亲自来找你，手里拿着一份羊皮纸契约。

"金雀花王国在找灰烬谷的商业伙伴。这是议会的贸易特许——你在灰烬谷收的所有货，我们可以溢价20%收购。"

他眯起眼睛。

"条件是：你每季度的贸易额的10%要上缴金雀花国库，算是'保护费'。签了它，你就是王国的特许商人。"

**选项**:

| 选项 | 描述 | 后果 |
|------|------|------|
| A | 签约效忠 | 金币 -100（签约费），获得"金雀花特许商人"身份，所有贸易收入+20%，解锁金雀花商路，每季度自动扣除贸易额10% |
| B | 讨价还价 | 商业≥60可触发：成功→签约费降至50金币，贸易溢价降至15%；失败→金雀花声望-10 |
| C | 拒绝 | 金雀花声望 -15，金算盘关系-20，获得"不识抬举"标签 |

### 事件4：边境联盟的独立号召

**触发条件**:
- 玩家在边境联盟声望 ≥ 40
- 与老根关系 ≥ 30
- 未效忠任何阵营

**事件描述**:
老根请你到他的大树下谈话。村长们围坐一圈，气氛凝重。

"三国都在拉拢我们，但烈焰说得对——灰烬谷应该属于灰烬谷的人。"

老根咳嗽几声。

"我们想成立真正的'边境联盟'，不是松散的村子联合，是一个真正的国家。你愿意加入吗？不是效忠哪一国，是效忠这片土地。"

**选项**:

| 选项 | 描述 | 后果 |
|------|------|------|
| A | 加入独立运动 | 影响力 +50，边境声望 +30，老根关系+25，获得"独立派"标签，触发"三国敌视"状态 |
| B | 保持中立 | 边境声望 +10，老根关系+5，获得"谨慎"标签 |
| C | 告密给苍龙 | 需要与苍龙有接触：金币 +300，苍龙声望+20，边境声望-50，老根关系-40，获得"叛徒"标签 |

### 事件5：背叛的机会

**触发条件**:
- 已效忠某一阵营 ≥ 30天
- 与该阵营关系出现危机（声望下降或接到可疑命令）

**事件描述**:
深夜，一个蒙面人潜入你的住所。他摘下头套——是敌对阵营的密探。

"我们知道你在那边过得不如意。他们给你什么？一点金币？一个虚衔？"

他推过来一个沉甸甸的钱袋。

"转投我们，金币翻倍，地位更高。而且..."他压低声音，"我们有你效忠阵营的黑料，如果你配合，可以让他们的信任瞬间崩塌。"

**选项**:

| 选项 | 描述 | 后果 |
|------|------|------|
| A | 接受策反 | 金币 +500，原阵营声望归零，新阵营声望+40，获得"背叛者"标签（永久），原阵营发布追杀令 |
| B | 假装接受，实则告发 | 需要外交≥70：成功→原阵营声望+50，金币+200，获得"忠诚"标签；失败→被双方追杀 |
| C | 拒绝并驱逐 | 原阵营声望+15，获得"忠诚"标签，密探被驱逐，触发"被报复"事件概率+20% |

### 事件6：灾民的求助

**触发条件**:
- 随机触发
- 玩家金币 ≥ 100

一群衣衫褴褛的灾民聚集在村口。他们来自北方的霜狼边境，据说那里发生了雪灾。

为首的难民跪在你面前："大人，求您施舍些粮食和钱财。我们的孩子快饿死了。"

村里的人小声议论："给了他们，我们自己的储备就少了。"

**选项**:

| 选项 | 描述 | 后果 |
|------|------|------|
| A | 慷慨救济 | 金币 -200，影响力 +30，获得"仁慈"标签，灾民中有1人成为可招募NPC |
| B | 少量施舍 | 金币 -50，影响力 +10 |
| C | 拒绝援助 | 金币不变，影响力 -10，边境声望 -5，获得"冷血"标签 |
| D | 上报苍龙 | 需要苍龙效忠：金币 +50（帝国奖励），影响力 -20，获得"帝国走狗"标签（边境） |

- 效忠某阵营

你的效忠阵营派来税官，表情严肃。

"国家危难之际，陛下下令紧急征税。作为帝国的属臣/王国的商人/联邦的战士，你有义务上缴500金币。"

税官压低声音："我知道这不少。但拒绝的话...可能会被视作不忠。"

**选项**:

| 选项 | 描述 | 后果 |
|------|------|------|
| A | 足额上缴 | 金币 -500，阵营声望 +25，获得"忠良"标签 |
| B | 部分缴纳 | 金币 -200，阵营声望 +5，触发"被怀疑"事件 |
| C | 请求宽限 | 需要外交≥50：成功→获得7天时间筹款；失败→阵营声望-10 |
| D | 拒绝缴纳 | 阵营声望 -30，触发"叛国嫌疑"调查 |

### 事件8：投资机遇

**触发条件**:
- 玩家金币 ≥ 400
- 商业属性 ≥ 40

**事件描述**:
商贸村的金算盘找你谈生意。

"有个机会。霜狼联邦急需粮食，如果我们从江南郡收粮，运到铁炉堡卖，利润至少翻倍。"

他递给你一份账本。

"需要你投资400金币。一个月后，还你600。当然...也有风险。如果路上被土匪抢了或者被苍龙截查，可能血本无归。"

**选项**:

| 选项 | 描述 | 后果 |
|------|------|------|
| A | 全额投资 | 金币 -400，30天后：70%概率金币+600（盈利），30%概率金币+0（亏损），影响力+15 |
| B | 要求担保 | 商业≥60：成功→降低风险至10%亏损概率，但利润降至150%；失败→机会被取消 |
| C | 小额投资 | 金币 -200，利润按比例分配，影响力+5 |
| D | 拒绝 | 金算盘关系-10，无其他影响 |

### 事件9：情报交易

**触发条件**:
- 玩家情报资源≥1（P1阶段）或有情报来源
- 与至少两个NPC关系≥30

小苏在酒馆里悄悄拉你到角落。

"有个消息，值大价钱。苍龙帝国的三皇子文曲...好像在秘密联络金雀花的某人。"

她眨眨眼。

"我可以告诉你详情，但你得付200金币。或者...你用你知道的秘密来换。我听说你和霜狼的人打过交道？"

**选项**:

| 选项 | 描述 | 后果 |
|------|------|------|
| A | 花钱购买 | 金币 -200，获得"文曲密谋"情报，可用于后续事件或出售 |
| B | 情报交换 | 需要持有其他情报：交换情报，小苏关系+10 |
| C | 拒绝交易 | 小苏关系-5，无其他影响 |
| D | 威胁告发 | 需要武力≥70或势力≥50：小苏被迫透露情报，但小苏关系-30，获得"威胁者"标签 |

### 事件10：资源分配困境

**触发条件**:
- 玩家拥有领地或村庄管理权
- 金币 ≥ 200

**事件描述**:
你的领地遭遇了旱灾。粮食减产，人心惶惶。

三个村长都来找你：
- 铁匠村需要钱修水利（300金币）
- 药草村需要种子渡过难关（200金币）
- 商贸村说如果不减税就搬走（减免50金币/月的税收）

"我们只能满足其中一方。"你的助手低声说。

**选项**:

| 选项 | 描述 | 后果 |
|------|------|------|
| A | 投资铁匠村 | 金币 -300，铁匠村发展+1，厚土关系+20，其他两村不满 |
| B | 投资药草村 | 金币 -200，获得"草药供应" buff（30天），白芷关系+20 |
| C | 给商贸村减税 | 金币收入 -50/月（持续3个月），金算盘关系+15，领地繁荣度+1 |
| D | 谁都不帮 | 所有村长关系-15，领地稳定度-1 |

## 三、人际冲突类（5个）

### 事件11：两难的调解

**触发条件**:
- 玩家外交属性 ≥ 50
- 与两个不同NPC关系≥20

**事件描述**:
烈焰和老根同时来找你。

烈焰怒气冲冲："老根反对独立，他要把我们出卖给苍龙！你站在哪边？"

老根叹气："烈焰太激进了，会把我们都害死。我们需要的是稳定，不是战争。你劝劝她。"

两人同时看着你，等待你的回答。

**选项**:

| 选项 | 描述 | 后果 |
|------|------|------|
| A | 支持烈焰 | 烈焰关系+25，老根关系-20，获得"激进派"标签 |
| B | 支持老根 | 老根关系+20，烈焰关系-30，获得"保守派"标签 |
| C | 调解双方 | 外交≥70判定：成功→双方关系+10，触发"联合方案"事件；失败→双方关系-10 |
| D | 保持中立 | 双方关系-5，获得"墙头草"标签 |

你的好友（关系≥40的NPC）脸色铁青地来找你。

"我听说你和那个混蛋（敌对NPC）走得挺近。你是不是把我这里的情报都告诉他了？"

| 选项 | 描述 | 后果 |

| B | 坦白部分真相 | 关系-25，但保留基本信任 |
| C | 指责对方多疑 | 关系-40，可能断交 |

### 事件13：求援的两难

**触发条件**:
- 同时收到两个NPC的求助请求
- 时间限制只能帮助一人

**事件描述**:
今天早上，两封信同时送到：

信一（来自苍龙旧识）："我卷入了一起政治案件，需要有人作证我当时不在现场。你是我唯一信任的人。"

信二（来自霜狼新交）："我的商队在边境被劫，我知道你有那里的关系。帮我追查货物，必有重谢。"

两封信都要求你三日内赶到。

**选项**:

| 选项 | 描述 | 后果 |
|------|------|------|
| A | 帮助苍龙旧识 | 旧识关系+30，新交关系-15，触发"作证"事件链 |
| B | 帮助霜狼新交 | 新交关系+25，旧识关系-20，金币+150（找回货物后） |
| C | 两边都敷衍 | 两边关系各-10，可能两边都失败 |
| D | 找第三人帮忙 | 影响力-30，两边都获得部分帮助，关系各+5 |

### 事件18：瘟疫阴影

**触发条件**:
- 随机触发（低概率）
- 玩家所在区域有药草村

**事件描述**:
药草村传来坏消息。

白芷面色凝重："一种奇怪的疫病正在蔓延。症状是高热、昏睡...已经死了三个人了。"

她看着你："我需要帮助。要么派人去隔离病人，要么去采购药材，要么...有些人建议把病人赶出村子。"

村民们恐惧地议论着。

**选项**:

| 选项 | 描述 | 后果 |
|------|------|------|
| A | 全力救治 | 金币 -300（药材），白芷关系+25，获得"救命恩人"标签，30天内该区域健康+1 |
| B | 隔离疫区 | 金币 -100，控制疫情蔓延，获得"理智"标签，部分村民不满 |
| C | 驱逐病人 | 金币不变，疫情结束更快，但获得"冷血"标签，白芷关系-30 |
| D | 逃离区域 | 你的领地不受影响，但获得"懦夫"标签，影响力-20 |

你被选中主持一场重要会谈。

苍龙、霜狼、金雀花的代表都来到了灰烬谷。议题是：边境贸易区的归属权。

"我们相信你的公正。"三国代表异口同声地说——虽然眼神里都没有信任。

老根私下告诉你："这是个烫手山芋。偏向任何一方，另外两方都会记恨你。但处理好了...你就是灰烬谷真正的领袖。"

## 五、事件使用指南

### 5.1 事件触发权重

```
效忠选择类：20%（游戏前期更高）
资源抉择类：30%（日常事件）
人际冲突类：25%（与NPC关系密切时更高）
危机应对类：25%（边境区域更高）
```

### 5.2 事件冷却机制

- 同一事件类型：7天内不重复触发
- 同一NPC相关事件：3天内不重复触发
- 失败的事件：30天后可再次触发

### 5.3 与叙事系统的结合

**每个事件都应生成叙事描述**：

```typescript
interface EventNarrative {
  eventId: string;
  preface: string;      // 事件前置叙述（触发时的场景）
  choices: string[];    // 每个选项的叙事描述
  outcomes: string[];   // 每个结果的叙事反馈
  aftermath: string;    // 事件后续影响（在晨报中提及）
}
```

**示例**（事件6：灾民的求助）：

```
preface: "清晨，你打开门，发现村口聚集了一群衣衫褴褛的人..."

choiceA_outcome: "你打开粮仓，亲自分发粮食..."

aftermath: "有传言称，暮光村的新领主（玩家）是个仁慈的人。一些流离失所的难民开始向你的领地聚集。"
```
