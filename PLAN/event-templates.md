# 通用事件模板库

> **MVP阶段通用事件模板**（20个完整版）
> 用于快速生成玩家可参与的标准事件
> 分类：效忠选择(5) + 资源抉择(5) + 人际冲突(5) + 危机应对(5)

---

## 一、效忠选择类（5个）

### 事件1：边境使者的邀请

**触发条件**：
- 玩家等级≥5
- 无效忠阵营
- 边境地区活跃度≥3天

**事件描述**：
暮光村的村口来了三位使者，各自代表着不同的势力。

苍龙帝国的使者穿着墨绿色官服，手持盖有玉玺的文书："帝国需要你这样的人才。效忠苍龙，你将获得正式的官职和俸禄。"

霜狼联邦的使者身披狼皮斗篷，腰挂符文护符："苍狼首领听说了你的事迹。霜狼需要勇士，不是奴才。"

金雀花王国的使者一袭锦缎长袍，手指上的戒指闪烁着魔法光泽："商会的金潮先生愿意与你谈谈。在金雀花，金币决定地位。"

**选项**：

| 选项 | 后果 | 代价/说明 |
|------|------|----------|
| 接受苍龙 | 苍龙声望+30，获得"帝国见习军官"身份，每日金币+30 | 霜狼、金雀花声望-10 |
| 接受霜狼 | 霜狼声望+30，获得"联邦勇士"身份，每日影响力+10 | 苍龙、金雀花声望-10 |
| 接受金雀花 | 金雀花声望+30，获得"商会代理人"身份，贸易收益+20% | 苍龙、霜狼声望-10 |
| 保持独立 | 边境声望+20，获得"自由民"身份 | 无阵营专属任务和资源支持 |

### 事件2：旧主的召唤

**触发条件**：
- 已效忠某阵营≥7天
- 另一阵营声望≥40

**事件描述**：
你正在自己的领地处理事务，一名风尘仆仆的信使送来了一封密信。

信是[原阵营]的[高层NPC]写的。他在信中回忆了你过去的功绩，然后笔锋一转："我知道你近期与[另一阵营]往来密切。我理解你的考虑，但请你记住——是谁在你最困难的时候伸出了援手。"

信的最后附着一张小纸条："如果你愿意 reaffirm 你的忠诚，来[地点]见我。如果你选择离开……我也理解，但请不要成为我们的敌人。"

**选项**：

| 选项 | 后果 | 说明 |
|------|------|------|
|  reaffirm 忠诚 | 原阵营声望+20 | 另一阵营声望-15 |
| 两边讨好 | 需谋略L3+检定 | 成功维持现状，失败双方声望-20，可能获"两面派"标签 |
| 转投新阵营 | 新阵营声望+25 | 原阵营声望-80，可能遭追杀 |
| 选择自立 | 双方声望-10 | 获"独行者"标签，触发"建立独立势力"事件链 |

### 事件3：背叛的诱惑

**触发条件**：
- 已效忠阵营≥14天
- 持有重要情报
- 敌对阵营声望≥30

**事件描述**：
深夜，一个黑影潜入你的住所。来人摘下面罩，你认出他是[敌对阵营]的高级间谍。

"我知道你手里有[原阵营]的布防图，"他直截了当地说，"我们也知道你在[原阵营]并不受重视。"

他推过来一个箱子，里面装满了金币："这是定金。只要你把情报给我们，还有三倍。而且[敌对阵营首领]答应给你高官职位。"

**选项**：

| 选项 | 后果 | 代价/风险 |
|------|------|----------|
| 出卖情报 | 金币+500，敌对阵营声望+40 | 原阵营声望-100，获"叛徒"标签，被追杀 |
| 假装答应并告发 | 原阵营声望+50 | 需谋略L4+，失败则被双方视为叛徒 |
| 严词拒绝 | 原阵营声望+30，获"正直"标签 | 间谍逃走，未来可能遭报复 |
| 杀死间谍 | 无即时后果 | 敌对阵营派更多人调查，压力持续 |

### 事件4：阵营内部的派系之争

**触发条件**：
- 已效忠阵营≥21天
- 阵营内部发生权力斗争

**事件描述**：
[效忠阵营]内部出现了分裂。[保守派领袖]和[改革派领袖]正在争夺[关键职位/政策方向]。

作为阵营内的一员，你无法置身事外。保守派的人找到你："[改革派]的激进方案会毁了我们的传统。支持我们，你将获得传统势力的友谊。"

改革派的人也来了："时代在变，固步自封只有死路一条。支持我们，你将是新秩序的奠基人。"

**选项**：

| 选项 | 后果 | 风险 |
|------|------|------|
| 支持保守派 | 保守派关系+40，改革派关系-30 | 改革派获胜则你被边缘化 |
| 支持改革派 | 改革派关系+40，保守派关系-30 | 保守派获胜则你被清算 |
| 保持中立劝和 | 影响力需50+，双方关系+10 | 成功获"和事佬"声望 |
| 待价而沽 | 获双方情报 | 被发现则双方关系-20 |

### 事件5：自立为王的契机

**触发条件**：
- 等级≥30
- 影响力≥200
- 拥有领地
- 至少一个阵营声望≥60

**事件描述**：
多年的经营，你已经不是当初那个无名小卒了。你拥有[领地名称]，手下有[士兵数量]的私兵，[NPC名单]等人才愿意追随你。

最重要的是——[某阵营]正深陷内乱，无力顾及边境地区。这是千载难逢的机会。

你的谋士[谋士NPC]进言："主公，时机已至。我们可以宣布独立，建立[势力名称]。"

你的老友[NPC]警告："三思啊！一旦走上这条路，就没有回头路了。"

**选项**：

| 选项 | 后果 | 代价/要求 |
|------|------|----------|
| 宣布独立 | 获"[势力名]领主"身份，解锁建设系统 | 原阵营关系-100，需武力L5+应对讨伐 |
| 暂缓独立 | 影响力+30 | 机会窗口可能关闭 |
| 与另一阵营结盟 | 独立成功率+30% | 未来需向盟友让渡利益 |
| 放弃野心 | 原阵营声望+50，获"忠诚"标签 | 失去独立可能性（本周期内） |

## 二、资源抉择类（5个）

### 事件6：饥荒中的抉择

**触发条件**：
- 游戏世界进入"饥荒"状态
- 玩家拥有粮食储备

**事件描述**：
连续三年的干旱终于酿成大祸。饥荒席卷了[地区名称]，粮食价格飞涨，民不聊生。

你的领地[领地名称]因为之前修建了粮仓，尚有一定储备。但现在你面临艰难选择：

村口聚集了大量难民，他们的眼神中满是绝望。你的管家报告："如果救济灾民，可以救活很多人，但我们自己的存粮也只够三个月。如果不救济……恐怕会民变。"

**选项**：

| 选项 | 后果 | 代价 |
|------|------|------|
| 开仓放粮 | 影响力+50，获"仁慈领主"标签 | 粮食清空，未来3个月收入-50% |
| 限量救济 | 影响力+20，民心维持稳定 | 粮食-50%，未来1个月收入-20% |
| 囤积居奇 | 金币潜力+800（粮价峰值） | 极高民变风险，获"奸商"标签 |
| 出售给商人 | 金币+500（三倍市价） | 民心-30，可能触发"民变"事件 |

"前期投入[价格]，每年可产出价值[年产量]的资源，三年回本，之后都是净利。"

**选项**：

| 选项 | 后果 | 代价/优势 |
|------|------|----------|
| 全额投资 | 获开采权，年收益+[年产量价值] | 独自承担所有风险 |
| 联合投资 | 与[商人NPC]合伙，年收益+[年产量价值/2] | 风险共担，[商人NPC]关系+30 |

更糟糕的是，[债权人NPC]派人传话："月底前必须还清[金额]，否则……"

你不得不向朋友们求助。

| 选项 | 后果 | 代价 |

### 事件9：资源争夺战

**触发条件**：
- 与其他玩家/NPC同时争夺同一资源点

**事件描述**：
[地点]的[资源点]突然变得炙手可热。传闻那里发现了[稀有资源]，各方势力都在觊觎。

你派出的人马刚刚到达，就发现[竞争对手]的人已经在那了。双方剑拔弩张，冲突一触即发。

对方首领[竞争对手NPC]喊话："这个资源点是我们先发现的！识相的退开！"

| 选项 | 后果 | 要求/风险 |

| 高价收购 | 获完整资源点 | 金币-[价格]，资金压力 |
| 暗中破坏 | 无即时成本 | 触发"破坏行动"事件，削弱对方 |

因为连续的高额税收，村民们不堪重负。今天，他们在[地点]聚集，要求减税。

领头的是[村民代表NPC]，他红着眼睛说："大人，我们真的活不下去了！求您大发慈悲，减减税吧！"

## 三、人际冲突类（5个）

你收到一封密信，是你的老朋友[老友NPC]写的。你们曾经一起[共同经历]，情同手足。

但现在的局势让你们站在了对立面。他是[敌对阵营]的重要人物，而你效忠[我方阵营]。

信中写道："我知道接下来会发生的事。作为朋友，我警告你——[即将发生的事件]。我不能背叛我的阵营，但也不想看到你送死。尽快离开[地点]。"

**选项**：

| 选项 | 后果 | 代价/风险 |
|------|------|----------|
| 相信老友撤离 | 避免[事件]的伤害 | 放弃[地点]利益，声望-10 |
| 不相信准备迎敌 | 如为真则防御+30% | 如为假则中埋伏 |
| 利用情报反制 | 挫败敌方计划 | 需谋略L4+，[老友NPC]关系-40 |

## 四、危机应对类（5个）

- 随机触发
- 或特定剧情阶段

恐慌像野火一样蔓延。

一种奇怪的瘟疫在[地区]爆发，患者会[症状描述]。更可怕的是，这种瘟疫似乎会通过[传播途径]传染。

你的领地[领地名称]也出现了病例。你的医师[医师NPC]束手无策："大人，这病我从未见过。我们需要[治疗所需]。"

**选项**：

| 选项 | 后果 | 代价/风险 |
|------|------|----------|
| 封锁领地 | 瘟疫传播-70% | 民心-20，经济损失，被封锁者可能死亡 |

## 五、事件使用指南

### 5.1 事件触发权重

```
效忠选择类：20%（游戏前期更高）
资源抉择类：30%（日常事件）
人际冲突类：25%（与NPC关系密切时更高）
危机应对类：25%（边境区域更高）
```

### 5.2 事件冷却机制

- 同一事件类型：7天内不重复触发
- 同一NPC相关事件：3天内不重复触发
- 失败的事件：30天后可再次触发

### 5.3 与叙事系统的结合

**每个事件都应生成叙事描述**：

```typescript
interface EventNarrative {
  eventId: string;
  preface: string;      // 事件前置叙述（触发时的场景）
  choices: string[];    // 每个选项的叙事描述
  outcomes: string[];   // 每个结果的叙事反馈
  aftermath: string;    // 事件后续影响（在晨报中提及）
}
```

**示例**（事件6：灾民的求助）：

```
preface: "清晨，你打开门，发现村口聚集了一群衣衫褴褛的人..."

choiceA_outcome: "你打开粮仓，亲自分发粮食..."

aftermath: "有传言称，暮光村的新领主（玩家）是个仁慈的人。一些流离失所的难民开始向你的领地聚集。"
```
