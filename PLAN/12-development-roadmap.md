# 开发路线

## 十二、开发路线

### Phase 1: MVP（6-8周）
- 基础UI框架 + 时间周期系统
- 资源管理 + 基础建设（5种建筑）
- 单个国家Agent的基本逻辑
- 基础事件系统（预设+简单LLM生成）
- 四阵营选择流程
- **成功标准**：世界能自动运转10个游戏日

### Phase 2: 核心体验（6-8周）
- 三国家Agent + Chronos主脑
- NPC Agent（每阵营3个关键NPC）
- 外交系统 + 效忠/背叛
- 事件链系统
- 技能树基础版
- **成功标准**：NPC对话自然，选择有真实影响

### Phase 3: 内容丰富（持续）
- 50+事件模板
- 20+独特NPC
- 完整战斗系统
- 经济系统
- 赛季系统
- **成功标准**：玩家愿意持续游玩

### Phase 4: 微信上线（4周）
- 微信登录 + 分享
- 排行榜
- 支付集成
- 审核准备
