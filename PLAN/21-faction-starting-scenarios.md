# Agame 阵营开局剧本

> **设计说明**：四个阵营的开局剧本，每个剧本为玩家提供独特的起始体验和初始故事线
>
> **重要更新**：根据最新角色系统设计，所有玩家从**普通老百姓**开始，无预设职业。职业由技能学习自然形成。

---

## 一、苍龙帝国开局剧本

### 1.1 剧本标题
**《龙吟九霄·天命之争》**

### 1.2 阵营背景

你身处帝都，苍龙帝国的心脏。

这座八百年的古城此刻笼罩在不安的阴云中。皇帝病重已有三月，三位皇子各据一方，暗中角力。大皇子天枢隐忍待发，二皇子破军掌控军权，三皇子文曲得母后支持。六部大臣们各自站队，朝中暗流涌动。

你是帝国的一名**普通村民**——生活在帝都城郊的小村庄，靠耕种或手艺维持生计。你不曾想过自己会卷入这场权力之争，但命运往往在意想不到的地方转弯。你的选择，将影响这场夺嫡之争的走向——也可能让你付出生命的代价。

### 1.3 初始角色设定

**玩家身份**：
- 出身：农家子弟/手艺人学徒/小商贩之子（三选一）
- 初始等级：1级（无名之辈）
- 初始地点：帝都城郊·青田村
- 初始状态：普通村民，无特殊身份

**初始技能**：
- 所有玩家初始拥有：**生存能力 L1**（通用技能）
- 根据出身，额外获得一项基础技能：
  | 出身 | 初始技能 | 说明 |
  |------|----------|------|
  | 农家子弟 | 农耕 L1 | 种植作物、养殖牲畜 |
  | 手艺人学徒 | 工艺 L1 | 基础手工制作 |
  | 小商贩之子 | 商业 L1 | 简单交易买卖 |

**初始属性**（基于十维属性系统）：
| 属性 | 初始值 | 说明 |
|------|--------|------|
| 体魄 | 40 | 常年劳作的体格 |
| 敏捷 | 35 | 普通人的反应 |
| 智慧 | 30 | 基础教育水平 |
| 意志 | 35 | 朴实的坚韧 |
| 感知 | 35 | 对环境的敏感 |
| 魅力 | 30 | 普通人的亲和 |
| 名望 | 0 | 无名之辈 |
| 恶名 | 0 | 无 |

**初始关系**：
- 与青田村村民关系 +20（邻里乡亲）
- 与帝都权贵关系均为 0（素未谋面）
- 玩家需要通过行动建立关系，而非预设

### 1.4 起始场景

**场景：帝都城郊·青田村**

> 青田村位于帝都以东二十里，洛水的一条支流从村旁流过。村里有三十多户人家，以种植水稻和养蚕为生。
>
> 你在这里长大，熟悉每一条田埂、每一棵老树。你的日子本该像父辈一样平淡——耕种、娶妻、生子、老去。
>
> 但今天，一切都将改变。
>
> 清晨，你正在田间劳作，忽然听到村口传来喧哗。一队骑士从帝都方向疾驰而来，马蹄踏碎了村庄的宁静。他们不是在赶路，而是在找人——找你。

**场景人物**：
- 村长老李头：青田村的村长，看着你长大
- 神秘骑士：来自帝都的使者，带来改变你命运的邀请

### 1.5 第一个事件：命运的转折

**事件名称**：《来自帝都的邀请》

**事件类型**：初始事件/效忠选择前置

**事件描述**：

那队骑士在村口停下，为首的骑士翻身下马，目光扫过聚集的村民，最终落在你身上。

"你就是[玩家名字]？"他的声音不带感情，"有人要见你。跟我们走一趟。"

村长老李头紧张地站出来："这位大人，这孩子犯了什么事？"

骑士摇头："不是问罪，是机会。"他看着你，"三皇子文曲殿下正在招募有识之士。有人推荐了你。"

村民哗然。你一个普通村民，怎么会入皇子的眼？

骑士不解释，只是递给你一块玉牌："去不去，你自己决定。如果去，明天日出前到帝都听雨楼。如果不去……"他顿了顿，"就当我们没来过。"

骑士们上马离去，留下你在村民复杂的目光中握着那块玉牌。

**可选行动**：

| 选项 | 前置条件 | 结果 |
|------|----------|------|
| [接受邀请，前往帝都] | 无 | 触发「初入帝都」事件链，开始接触权力中心 |
| [拒绝邀请，留在村里] | 无 | 触发「村庄守护者」支线，走本地发展路线 |
| [询问村长老李头的意见] | 无 | 老李头建议你先去探探，获得「谨慎的村民」标签 |
| [打听是谁推荐了你] | 感知≥35 | 发现是村里一位隐居老者年轻时的旧识，获得隐藏信息 |
| [观察骑士的装备细节] | 感知≥40 | 发现骑士装备精良但无皇家徽记，可能是某势力的私人卫队 |

**事件结果**：
- 接受邀请：进入帝都权力圈，可能卷入夺嫡之争
- 拒绝邀请：留在村庄，走基层发展路线，但仍可能被卷入大事件
- 注意：这不是效忠选择，只是故事的开始。真正的阵营选择将在后续展开

### 1.6 初始资源

| 资源 | 数值 | 说明 |
|------|------|------|
| 金币 | 20 | 普通村民的积蓄 |
| 影响力 | 0 | 尚未建立声望 |
| 苍龙帝国声望 | 0 | 平民身份 |
| 个人物品 | 农具/工具 | 根据出身不同 |

### 1.7 发展方向

**从平民开始的多种可能**：

```
路线A：进入权力中心
  └─ 接受邀请 → 进入听雨楼 → 接触三皇子 → 学习谋略技能
  └─ 可能身份：谋士、官员、密探

路线B：留在村庄发展
  └─ 拒绝邀请 → 帮助村庄 → 被推举为村长 → 学习经营技能
  └─ 可能身份：村长、地方豪强、独立势力

路线C：军事路线
  └─ 无论是否去帝都，都可以选择参军
  └─ 可能身份：士兵、军官、将军

路线D：商业路线
  └─ 从摆摊开始 → 开设店铺 → 建立商队
  └─ 可能身份：商人、商会成员、商业巨擘

路线E：混合路线
  └─ 学习战斗+商业 → 武装商人
  └─ 学习谋略+经营 → 政治商人
  └─ ...无限组合
```

**核心体验**：从无名小卒开始，通过选择和学习，逐步塑造自己的身份

## 二、霜狼联邦开局剧本

### 2.1 剧本标题
**《狼啸冰原·变革之火》**

### 2.2 阵营背景

你站在霜牙城的冰渊桥上，脚下是深不见底的冰川裂谷。

霜狼联邦正处在一个十字路口。新任首领苍狼推行改革，试图用贸易取代掠夺，用契约取代刀剑。但传统派并不买账——血狼部族首领公开反对，认为这是对先祖荣耀的背叛。

你是联邦的一名**普通牧民**——生活在驯鹿原的一个小部落，靠放牧驯鹿和打猎为生。你不曾想过自己会卷入这场变革之争，但命运往往在意想不到的地方转弯。现在，你可以选择：是支持苍狼的改革，还是坚守血狼的传统——或者，走出一条自己的路。

### 2.3 初始角色设定

**玩家身份**：
- 出身：牧民之子/矿工之子/猎人之后（三选一）
- 初始等级：1级（无名之辈）
- 初始地点：驯鹿原·白鹿部落
- 初始状态：普通牧民，无特殊身份

**初始技能**：
- 所有玩家初始拥有：**生存能力 L1**（通用技能）
- 根据出身，额外获得一项基础技能：
  | 出身 | 初始技能 | 说明 |
  |------|----------|------|
  | 牧民之子 | 驯兽 L1 | 驯养牲畜、驾驭坐骑 |
  | 矿工之子 | 采矿 L1 | 矿石识别、基础开采 |
  | 猎人之后 | 追踪 L1 | 野外追踪、狩猎技巧 |

**初始属性**（基于十维属性系统）：
| 属性 | 初始值 | 说明 |
|------|--------|------|
| 体魄 | 45 | 冰原生活的强健体格 |
| 敏捷 | 35 | 普通人的反应 |
| 智慧 | 25 | 基础教育水平 |
| 意志 | 40 | 冰原磨练的坚韧 |

| 名望 | 0 | 无名之辈 |

**初始关系**：
- 与白鹿部落村民关系 +25（部落成员）
- 与联邦权贵关系均为 0（素未谋面）
- 玩家需要通过行动建立关系，而非预设

### 2.4 起始场景

**场景：驯鹿原·白鹿部落**

> 白鹿部落是驯鹿原上众多小型部落之一，有四十多顶帐篷，两百多口人。这里的牧民世代以驯养驯鹿为生，夏天在草原放牧，冬天迁入山谷避寒。
>
> 你在这里长大，学会了驾驭驯鹿、识别天气、在冰原生存。你的日子本该像父辈一样——放牧、参与部落议事、在合适的时候娶妻生子。
>
> 但今天，一切都将改变。
>
> 傍晚，部落外传来马蹄声。一队来自霜牙城的使者抵达，带来首领苍狼的召集令：所有成年男子都可以前往霜牙城，参与联邦的未来决策。

**场景人物**：
- 部落长老·白须：白鹿部落的长老，看着你长大
- 传令使者：来自霜牙城的信使，带来改变命运的邀请

### 2.5 第一个事件：冰原的召唤

**事件名称**：《来自霜牙城的召集》

**事件类型**：初始事件/效忠选择前置

**事件描述**：

传令使者在部落中央的火堆旁宣读召集令：

"首领苍狼谕令：联邦正处于变革之际，需要每一位勇士的声音。无论你是否赞同改革，都可以前往霜牙城，亲眼见证，亲口发声。"

部落长老白须看着你，目光复杂："孩子，这是机会，也是危险。去了霜牙城，你可能卷入那些大人物的争斗。但留在部落，你的声音永远传不到外面。"

传令使者补充道："不去也无妨，召集令不是强制。但记住——联邦的命运，最终由每一个选择参与的人决定。"

**可选行动**：

| [询问长老的详细建议] | 无 | 长老讲述改革派和传统派的矛盾，获得「知情者」标签 |
| [观察传令使者的态度] | 感知≥40 | 发现使者本人倾向于改革，获得隐藏信息 |
| [向使者展示你的驯鹿] | 驯兽L1 | 使者对你的驯鹿技艺印象深刻，承诺引荐给草原·驯风 |

**事件结果**：
- 前往霜牙城：进入联邦权力圈，可能卷入改革与传统之争
- 留在部落：走基层发展路线，发展部落成为地方势力
- 注意：这不是效忠选择，只是故事的开始

| 资源 | 数值 | 说明 |

| 影响力 | 0 | 尚未建立声望 |

### 2.7 发展方向

## 三、金雀花王国开局剧本

### 3.1 剧本标题
**《金潮涌动·商海沉浮》**

### 3.2 阵营背景

你站在金雀花城的港口，看着来来往往的商船。

这里是大陆的金融心脏。银行家金潮掌控着王国的命脉，而传统贵族们却在暗中谋划——他们认为商人永远不该染指权力。议会大厦与金库隔街相望，象征着新旧势力的对峙。

你是王国的一名**普通市民**——生活在金雀花城贫民区的小商贩之子，靠帮父亲摆摊或打零工维持生计。你不曾想过自己会卷入上流社会的博弈，但命运往往在意想不到的地方转弯。在这个"一切皆可交易"的城市，哪怕是一个小人物，也可能通过智慧和机遇改变自己的命运。

### 3.3 初始角色设定

**玩家身份**：
- 出身：小商贩之子/工匠学徒/码头苦力（三选一）
- 初始等级：1级（无名之辈）
- 初始地点：金雀花城·金河区
- 初始状态：普通市民，无特殊身份

**初始技能**：
- 所有玩家初始拥有：**生存能力 L1**（通用技能）
- 根据出身，额外获得一项基础技能：
  | 出身 | 初始技能 | 说明 |
  |------|----------|------|
  | 小商贩之子 | 商业 L1 | 基础交易、货物识别 |
  | 工匠学徒 | 工艺 L1 | 手工制作、修理 |
  | 码头苦力 | 劳作 L1 | 体力劳动、搬运 |

**初始属性**（基于十维属性系统）：
| 属性 | 初始值 | 说明 |
|------|--------|------|
| 体魄 | 35 | 城市生活的普通体格 |
| 敏捷 | 35 | 普通人的反应 |
| 智慧 | 35 | 城市环境的见闻积累 |
| 意志 | 30 | 贫困中磨练的韧性 |
| 感知 | 40 | 对机会的敏感 |
| 魅力 | 30 | 市井中的普通存在 |
| 名望 | 0 | 无名之辈 |
| 恶名 | 0 | 无 |

**初始关系**：
- 与金河区邻居关系 +15（邻里关系）
- 与上流社会关系均为 0（素未谋面）
- 玩家需要通过行动建立关系，而非预设

### 3.4 起始场景

**场景：金雀花城·金河区·鱼市码头**

> 金河区的鱼市码头每天清晨最热闹。渔民们卸下刚捕获的海产，商贩们吆喝着叫卖，苦力们扛着货物穿梭在狭窄的巷子里。
>
> 你在这里长大，熟悉每一个摊位、每一条捷径。你的日子本该像父辈一样——劳作、攒钱、在合适的时候租一个小铺子。
>
> 但今天，一切都将改变。
>
> 一个身穿华服的商人正在码头中央大声宣布："金潮阁下正在招募有潜力的年轻人！不论出身，只要有才能，就有机会进入交易所学习！"
>
> 人群哗然。金潮阁下是王国最有权势的人之一，他的招募令意味着什么，每个人都清楚。

**场景人物**：
- 父亲/导师：养育你的人，对你的未来有不同期望
- 招募官员：来自金潮阁下的使者，带来改变命运的机会

### 3.5 第一个事件：商海的召唤

**事件名称**：《码头的机会》

**事件类型**：初始事件/发展选择前置

**事件描述**：

招募官员站在码头中央的木箱上，目光扫过人群：

"金潮阁下相信，才能不分贵贱。无论你是商贩之子、工匠学徒，还是码头苦力，只要有头脑、有勇气，都可以来试试。"

他顿了顿："明天 sunrise，在交易所门口有入门测试。通过者可以进入金潮阁下的学徒计划。"

与此同时，你注意到人群中还有另一股势力——一个旧冠家族的代理人正在私下接触有潜力的年轻人，承诺提供更稳定的传统路径。

你的父亲/导师看着你："孩子，这是机会，但也可能是陷阱。金潮阁下的世界充满算计，一不小心就会粉身碎骨。但留在金河区，你的天花板就在头顶。"

**可选行动**：

| 选项 | 前置条件 | 结果 |
|------|----------|------|
| [参加金潮的学徒招募] | 无 | 触发「交易所入门」事件链，进入商业世界 |
| [接触旧冠家族的代理人] | 无 | 触发「传统路径」支线，走贵族庇护路线 |
| [两边都了解，暂不决定] | 无 | 获得「谨慎者」标签，先收集更多信息 |
| [观察测试现场的情况] | 感知≥40 | 发现金潮的测试其实更看重应变能力而非出身，获得隐藏信息 |
| [向招募官员展示你的商业头脑] | 商业L1 | 官员对你印象深刻，承诺在测试中给你额外机会 |

**事件结果**：
- 参加金潮招募：进入商业资本运作世界，体验金融战争与政治博弈
- 接触旧冠家族：走传统贵族庇护路线，获得稳定但受限的发展
- 两边都了解：保持选择权，但需要承担犹豫的代价
- 注意：这不是效忠选择，只是发展路径的初步选择

### 3.6 初始资源

| 资源 | 数值 | 说明 |
|------|------|------|
| 金币 | 30 | 普通市民的积蓄 |
| 影响力 | 0 | 尚未建立声望 |
| 金雀花王国声望 | 0 | 平民身份 |
| 个人物品 | 货物/工具 | 根据出身不同 |

### 3.7 发展方向

**从平民开始的多种可能**：

```
路线A：商业巨头路线
  └─ 参加学徒招募 → 进入交易所 → 学习商业技能 → 成为金融巨擘
  └─ 可能身份：银行家、商会领袖、商业政治家

路线B：传统贵族路线
  └─ 投靠旧冠家族 → 获得庇护 → 学习政治技能 → 成为贵族顾问
  └─ 可能身份：政客、贵族代理人、传统势力代表

路线C：自主创业路线
  └─ 留在金河区 → 从摆摊开始 → 逐步扩张 → 成为独立商人
  └─ 可能身份：商人、工厂主、贸易代表

路线D：混合路线
  └─ 学习商业+谋略 → 商业间谍
  └─ 学习商业+战斗 → 武装商人
  └─ ...无限组合
```

**核心体验**：从市井小民开始，通过智慧和机遇，逐步塑造自己的身份

## 四、边境联盟开局剧本

### 4.1 剧本标题
**《灰烬余生·自由之路》**

### 4.2 阵营背景

你站在暮光村的村中心，身后是那棵千年古树，面前是三界交汇的暮光山丘。

这里是灰烬谷，一个不属于任何国家的地方。暮光村是联盟最古老的村落之一，也是你的家乡。村长老根在这里维持着脆弱的和平——苍龙、霜狼、金雀花，三国的势力都在渗透这片土地。

你是暮光村的一名**普通村民**——靠耕种、打猎或帮工维持生计。你不曾想过自己会改变什么，但命运往往在意想不到的地方转弯。在这个自由但危险的地方，哪怕是一个小人物，也可能通过自己的选择影响三国的平衡。

### 4.3 初始角色设定

**玩家身份**：
- 出身：暮光村本土人/苍龙逃难者/霜狼流浪者/金雀花破产商（四选一）
- 初始等级：1级（无名之辈）
- 初始地点：暮光村·暮色酒馆附近
- 初始状态：普通村民，无特殊身份

**初始技能**：
- 所有玩家初始拥有：**生存能力 L1**（通用技能）
- 根据出身，额外获得一项基础技能：
  | 出身 | 初始技能 | 说明 |
  |------|----------|------|
  | 暮光村本土人 | 采集 L1 | 野外采集、识别资源 |
  | 苍龙逃难者 | 学识 L1 | 基础教育、文字能力 |
  | 霜狼流浪者 | 战斗 L1 | 基础格斗、野外生存 |
  | 金雀花破产商 | 商业 L1 | 交易基础、货物识别 |

**初始属性**（基于十维属性系统）：
| 属性 | 初始值 | 说明 |
|------|--------|------|
| 体魄 | 40 | 边境生活的强健体格 |
| 敏捷 | 35 | 普通人的反应 |
| 智慧 | 30 | 基础教育水平 |
| 意志 | 40 | 边境磨练的坚韧 |
| 感知 | 40 | 对危险的敏感 |
| 魅力 | 30 | 边境社区中的普通成员 |
| 名望 | 0 | 无名之辈 |
| 恶名 | 0 | 无 |

**初始关系**：
- 与暮光村村民关系 +20（邻里乡亲）
- 与老根关系 +10（村长认识你）
- 与三国势力关系均为 0（素未谋面或仅有传闻）

### 4.4 起始场景

**场景：暮光村·村中心大树下**

> 暮光村的村中心有一棵千年古树，村民们习惯在这里议事、交易、休息。这里是灰烬谷的消息集散地，三国的商队、使者、间谍都会经过这里。
>
> 你在这里长大（或来到这里），熟悉村里的每一个人、每一条小路。你的日子本该像其他人一样——劳作、生活、在边境的夹缝中求存。
>
> 但今天，一切都将改变。
>
> 村口传来消息：一队神秘的武装人员出现在边境附近，身份不明。村长决定召集村民商议对策。而你，被意外地邀请参加。

**场景人物**：
- 老根：暮光村村长，睿智而务实
- 小苏：酒馆老板娘，消息灵通
- 同村伙伴：与你一起长大的村民

### 4.5 第一个事件：边境的抉择

**事件名称**：《村长的召集》

**事件类型**：初始事件/效忠选择前置

**事件描述**：

村民们聚集在古树下，议论纷纷。

老根站在众人面前，声音沉稳："边境出现不明武装，可能是匪帮，也可能是某国的先锋。我们需要决定：是加强自卫，还是向某一方寻求庇护。"

人群中传出不同声音：
- 有人主张保持中立，依靠联盟自己的力量
- 有人建议投靠苍龙，获得帝国的保护
- 有人提议联络霜狼，用武力保卫家园
- 有人认为应该接受金雀花的援助，换取商业利益

老根看向你："年轻人，你怎么看？"

与此同时，你注意到人群边缘有三个人在暗中观察——龙使、北风、金掌柜，三国的代表都在等待你的表态。

**可选行动**：

| 选项 | 前置条件 | 结果 |
|------|----------|------|
| [支持老根，加强自卫] | 无 | 老根好感+30，获得「守村者」标签；触发「村庄防御」任务链 |
| [建议向苍龙求助] | 无 | 龙使好感+20；触发「龙之子」任务线（可后续选择是否效忠） |
| [建议向霜狼求助] | 无 | 北风好感+20；触发「狼之血」任务线（可后续选择是否效忠） |
| [建议向金雀花求助] | 无 | 金掌柜好感+20；触发「金之路」任务线（可后续选择是否效忠） |
| [收集更多情报] | 感知≥40 | 发现武装人员其实是匪帮而非某国军队，获得关键信息 |
| [组织村民准备防御] | 战斗L1 或 意志≥40 | 老根好感+25，获得「预备防御者」临时身份 |
| [保持沉默，先观察] | 无 | 各方好感+5，但被视为缺乏主见；后续选择机会减少 |

**事件结果**：
- 支持自卫：走联盟独立发展路线，保持中立但需自行承担风险
- 建议求助某国：开启该国任务线，但不等于效忠，仍可后续选择
- 收集情报：获得信息优势，可在关键时刻影响决策
- 注意：这不是最终的效忠选择，只是应对当前危机的初步态度

## 六、开局事件设计要点

### 6.1 共同设计原则

1. **强制选择**：每个开局都有一个核心选择，决定玩家的初始阵营倾向
2. **技能差异化**：不同技能等级解锁不同选项，体现技能系统的价值
3. **后续触发**：初始选择会触发不同的任务链，形成分支剧情
4. **信息差**：部分关键信息需要特定技能才能发现，鼓励多样化发展

### 6.2 与许愿系统的关联

玩家在开局后的**第一次许愿**（次日）将由对应层级的Agent计算：

| 阵营 | 许愿Agent | 许愿内容示例 |
|------|-----------|--------------|
| 苍龙 | 天命司·Chronos | "我该如何在夺嫡之争中保全自己？" |
| 霜狼 | 先祖议会·Chronos | "我应该支持改革还是传统？" |
| 金雀花 | 黄金议会·Chronos | "我该如何在商业竞争中获胜？" |
| 边境 | 谷主意志·Chronos | "灰烬谷的未来在哪里？" |

### 6.3 与技能系统的关联

开局事件是玩家**第一次体验技能系统**的场景：

- 技能等级决定可见选项数量
- 技能组合可能解锁特殊选项
- 技能选择影响后续事件触发

### 6.4 与叙事系统的关联

开局事件将作为玩家角色的**起源故事**，被记录在叙事系统中：

- 玩家的选择会被写入每日小说章节
- NPC会根据玩家的开局选择调整态度
- 主角NPC（天枢/苍狼/金潮/老根）会在后续剧情中提及玩家的起源
