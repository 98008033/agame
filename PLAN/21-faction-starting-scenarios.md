# 《三界风云录》阵营开局剧本

> **设计说明**：四个阵营的开局剧本，每个剧本为玩家提供独特的起始体验和初始故事线

---

## 一、苍龙帝国开局剧本

### 1.1 剧本标题
**《龙吟九霄·天命之争》**

### 1.2 阵营背景

你身处天都，苍龙帝国的心脏。

这座八百年的古城此刻笼罩在不安的阴云中。皇帝病重已有三月，三位皇子各据一方，暗中角力。大皇子天枢隐忍待发，二皇子破军掌控军权，三皇子文曲得母后支持。六部大臣们各自站队，朝中暗流涌动。

你是帝国的一名**新晋士绅**——通过科举或军功获得地位，却尚未在权力的棋盘上占据要位。你的选择，将影响这场夺嫡之争的走向。

### 1.3 初始角色设定

**玩家身份**：
- 出身：寒门学子/退伍军官/商贾之子（三选一）
- 初始地位：士绅（11级）
- 初始地点：天都·洛水河畔

**初始技能**（选择一项主修）：
| 路线 | 初始技能 | 特色 |
|------|----------|------|
| 文官路线 | 情报分析 L1 | 擅长收集信息、洞察人心 |
| 武将路线 | 战斗技巧 L1 | 擅长单挑、保护重要人物 |
| 商贾路线 | 商业贸易 L1 | 擅长经济运作、资源调配 |

**初始关系**：
- 与六部中的某一部尚书关系 +20（玩家选择效忠对象）
- 与三位皇子关系均为 0（尚未站队）

### 1.4 起始场景

**场景：天都·洛水河畔·听雨楼**

> 洛水穿城而过，将天都分为南北两半。北城是皇城与贵族府邸，南城是平民与商贾的居所。听雨楼位于南北交界处，是各方势力交换消息的秘密场所。
>
> 今夜，你收到一封没有署名的请柬，邀你至此。窗外雨声淅沥，楼内烛火摇曳。你知道，这是一个选择站队的机会——也是一个可能丧命的危险之地。

**场景人物**：
- 听雨楼掌柜：消息灵通的中年人，八面玲珑
- 神秘信使：递给你请柬的人，身份不明

### 1.5 第一个事件：三子争锋

**事件名称**：《洛水密会》

**事件类型**：效忠选择

**事件描述**：

你在听雨楼的雅间中等候，窗外雨声渐大。

不多时，三人依次抵达——他们竟然都收到了同样的请柬，以为只有自己在场。

- **大皇子天枢**：一袭青衫，气度沉稳。他看着你，目光中带着审视："阁下年纪轻轻便获士绅之位，必有过人之处。若愿助我，将来封侯拜相，不在话下。"

- **二皇子破军**：身披轻甲，腰悬长剑。他拍案而起："废话少说！我需要的是能打仗的人，不是耍嘴皮子的！跟了我，战场上的功劳全归你！"

- **三皇子文曲**：手执折扇，温文尔雅。他轻声道："两位兄长未免太过功利。我只问一句——阁下可愿为苍生谋一份太平？"

三人的目光同时落在你身上。

**可选行动**（根据技能等级解锁）：

| 选项 | 前置条件 | 结果 |
|------|----------|------|
| [选择支持天枢] | 无 | 获得天枢好感+30，破军好感-10，文曲好感-5；触发「隐忍者的试探」后续事件 |
| [选择支持破军] | 无 | 获得破军好感+30，天枢好感-15，文曲好感-5；触发「军中的考验」后续事件 |
| [选择支持文曲] | 无 | 获得文曲好感+30，天枢好感-10，破军好感-10；触发「改革者的理想」后续事件 |
| [察言观色] | 情报分析L1+ | 发现三皇子文曲的扇子上有皇后凤仪的徽记，意识到文曲与皇后关系匪浅 |
| [保持中立] | 无 | 三方好感各+5，被视为"待价而沽"，后续可能收到更多拉拢 |
| [试图调解] | 政治操控L4+ | 提出一个让三方都暂时停战的方案，获得「调解者」称号，影响力+20 |
| [以武服人] | 战斗技巧L3+ | 展示武力震慑众人，要求他们听你说完，触发特殊对话分支 |

**事件结果**：
- 无论选择哪一方，都将进入对应皇子的故事线
- 保持中立则会触发更多阵营间的事件，直到被迫站队

### 1.6 初始资源

| 资源 | 数值 | 说明 |
|------|------|------|
| 金币 | 200 | 士绅的基础积蓄 |
| 影响力 | 30 | 新晋士绅的初始声望 |
| 天都声望 | +10 | 作为本地士绅的基础声望 |

### 1.7 开局特色

**核心体验**：权谋斗争、站队抉择、政治博弈

**推荐玩家**：喜欢策略、权谋、政治剧情的玩家

**故事基调**：压抑中蕴含机遇，每一步都需谨慎

## 二、霜狼联邦开局剧本

### 2.1 剧本标题
**《狼啸冰原·变革之火》**

### 2.2 阵营背景

你站在霜牙城的冰渊桥上，脚下是深不见底的冰川裂谷。

霜狼联邦正处在一个十字路口。新任首领苍狼推行改革，试图用贸易取代掠夺，用契约取代刀剑。但传统派并不买账——血狼部族首领公开反对，认为这是对先祖荣耀的背叛。

你是联邦的一名**新晋战士**——在边境冲突中立下功劳，获得了进入霜牙城的资格。现在，你必须选择：是支持苍狼的改革，还是坚守血狼的传统？

### 2.3 初始角色设定

**玩家身份**：
- 出身：牧民之子/矿工伤员/流浪战士（三选一）
- 初始地位：平民（5级）
- 初始地点：霜牙城·狼魂广场

**初始技能**（固定）：
- 战斗技巧 L2
- 生存能力 L1

**初始关系**：
- 与苍狼关系 +10（认可你的战功）
- 与烈风（苍狼之弟）关系 +15（引荐你进城的人）
- 与血狼·屠灭关系 0（尚未接触）

### 2.4 起始场景

**场景：霜牙城·狼魂广场**

> 广场中央矗立着十二根符文石柱，这是联邦议会的所在地。此刻，广场上挤满了人——首领苍狼即将发表重要讲话。
>
> 你站在人群边缘，身边是烈风。他拍了拍你的肩膀："小子，今天之后，联邦可能会变天。你想站在哪一边？"
>
> 话音未落，广场另一侧传来一阵骚动。血狼·屠灭带着他的人马来了，气氛瞬间剑拔弩张。

**场景人物**：
- 烈风：首领之弟，性格直率，赏识你的勇武
- 血狼·屠灭：血狼部族首领，强硬派领袖
- 古木：首席谋士，正在努力维持局面

### 2.5 第一个事件：冰原抉择

**事件名称**：《狼魂广场的对峙》

**事件类型**：效忠选择 + 危机应对

**事件描述**：

苍狼站在符文石柱下，声音沉稳而有力：

"诸位！先祖以刀剑开辟这片土地，但刀剑不能让我们度过下一个严冬。霜狼需要改变——不是背叛传统，而是让传统延续下去！"

话音刚落，血狼·屠灭大步走出人群，他的声音如雷鸣般炸响：

"说得好听！但改变就是软弱！苍龙人正在边境集结，而你却想和他们做生意？这是把联邦卖给敌人！"

人群开始骚动。有人支持苍狼，有人支持血狼。你意识到，这是一个必须表态的时刻。

**可选行动**：

| 选项 | 前置条件 | 结果 |
|------|----------|------|
| [支持苍狼] | 无 | 苍狼好感+25，血狼好感-30，获得「改革派」标签；触发「新商路」任务链 |
| [支持血狼] | 无 | 血狼好感+30，苍狼好感-20，获得「传统派」标签；触发「边境巡逻」任务链 |
| [保持观望] | 无 | 双方好感各+5，被视为谨慎之人；但烈风会对你失望（-10） |
| [提出折中方案] | 情报分析L2+ 或 政治操控L3+ | 建议先解决边境威胁再谈改革，获得古木赏识（+20），触发「双面间谍」特殊线 |
| [挑战血狼决斗] | 战斗技巧L3+ | 以决斗解决争端，胜利则血狼暂时退让，失败则重伤并失去选择机会 |
| [联络烈风] | 与烈风关系≥20 | 烈风会在关键时刻帮你说话，增加选择筹码 |

**事件结果**：
- 支持改革派：进入苍狼的现代化故事线，体验经济与外交
- 支持传统派：进入血狼的荣耀故事线，体验战斗与征服
- 折中方案：获得双面角色身份，可两边摇摆但风险更高

### 2.6 初始资源

| 资源 | 数值 | 说明 |
|------|------|------|
| 金币 | 80 | 战士的微薄积蓄 |
| 影响力 | 15 | 边境战功带来的名声 |
| 霜狼声望 | +15 | 作为战士的基础声望 |
| 符文石 | 1 | 烈风赠送的见面礼 |

### 2.7 开局特色

**核心体验**：武力荣耀、改革与保守的冲突、部落荣誉

**推荐玩家**：喜欢战斗、直率剧情、团队荣誉的玩家

**故事基调**：豪爽中蕴含危机，每一次选择都是荣耀的考验

## 三、金雀花王国开局剧本

### 3.1 剧本标题
**《金潮涌动·商海沉浮》**

### 3.2 阵营背景

你站在金雀花城的金库穹顶下，头顶是纯金打造的穹顶，在阳光下耀眼夺目。

这里是大陆的金融心脏。银行家金潮掌控着王国的命脉，而传统贵族们却在暗中谋划——他们认为商人永远不该染指权力。议会大厦与金库隔街相望，象征着新旧势力的对峙。

你是王国的一名**新晋商人**——通过一笔精明的交易或一个冒险的赌局，获得了进入上流社会的门票。但在这个"一切皆可交易"的城市，你的位置并不稳固。

### 3.3 初始角色设定

**玩家身份**：
- 出身：小商贩之子/破产贵族/海外归侨（三选一）
- 初始地位：士绅（11级）
- 初始地点：金雀花城·金河运河畔

**初始技能**（选择一项主修）：
| 路线 | 初始技能 | 特色 |
|------|----------|------|
| 金融路线 | 商业贸易 L2 | 擅长资本运作、投资分析 |
| 实业路线 | 产业管理 L1 | 擅长生产运营、资源调配 |
| 外交路线 | 情报分析 L1 | 擅长信息收集、关系运作 |

**初始关系**：
- 与金潮关系 +15（看好你的潜力）
- 与旧冠关系 -10（传统贵族对新兴商人的轻视）
- 与铜臭（商团首领）关系 +10（同行互助）

### 3.4 起始场景

**场景：金雀花城·交易所大厅**

> 交易所是王国的心脏，每分钟都有数以万计的金币在这里流动。交易员们的叫喊声此起彼伏，空气中弥漫着紧张与贪婪的气息。
>
> 你站在二楼回廊，俯瞰着这场金融战争。金潮就在你身旁，他的目光穿透人群，仿佛能看到未来的走向。
>
> "有趣的交易。"他轻声说，"旧冠刚刚抛售了他家族的矿业股份——显然是为政治斗争筹集资金。你怎么看？"

**场景人物**：
- 金潮：银行家/议长，王国的实际掌控者
- 交易所主持：冷静公正的交易仲裁者
- 旧冠的代理人：正在执行抛售任务

### 2.5 第一个事件：金河暗战

**事件名称**：《交易所的博弈》

**事件类型**：资源抉择 + 人际冲突

**事件描述**：

金潮递给你一份文件——那是旧冠家族矿业的详细账目。

"旧冠正在筹集资金，准备在议会上提出一项限制商人从政的法案。"金潮的声音平静，"如果我们收购这些股份，就能在关键时刻否决他的提案。但......"

他顿了顿："这需要大量资金。你愿意投资吗？"

与此同时，你的随从带来消息：旧冠派人传话，如果你保持中立，他愿意让你以优惠价格购买他领地内的贸易权——那可是通往东方的必经之路。

**可选行动**：

| 选项 | 前置条件 | 结果 |
|------|----------|------|
| [投资金潮] | 金币≥100 | 花费100金币，获得矿业股份；金潮好感+30，旧冠好感-40；触发「商战」任务链 |
| [接受旧冠提议] | 无 | 获得东方贸易权（价值150金币）；旧冠好感+20，金潮好感-30；触发「贵族游戏」任务链 |
| [两边下注] | 情报分析L3+ | 同时答应双方，风险极高但收益巨大；触发「双面商人」特殊线 |
| [提出合作方案] | 商业贸易L3+ 或 政治操控L4+ | 建议三方合资开发新商路，获得「和事佬」称号；如果成功，三方好感各+15 |
| [观望] | 无 | 暂时不表态，但可能错失机会；双方好感各-5，被视为不可靠 |
| [调查旧冠] | 情报分析L2+ | 发现旧冠其实也在向霜狼走私武器，获得把柄；可用来要挟或揭露 |

**事件结果**：
- 投资金潮：进入商业资本故事线，体验金融战争与政治博弈
- 接受旧冠提议：进入贵族社交故事线，体验权力交易与阴谋
- 两边下注：高风险高回报，但可能被两边同时抛弃

### 3.6 初始资源

| 资源 | 数值 | 说明 |
|------|------|------|
| 金币 | 300 | 商人起步资金 |
| 影响力 | 25 | 商业圈子的初步认可 |
| 金雀花声望 | +10 | 作为商人的基础声望 |
| 贸易许可证 | 1 | 可在金雀花境内自由贸易 |

### 3.7 开局特色

**核心体验**：资本运作、利益博弈、商业战争

**推荐玩家**：喜欢经济系统、策略经营、利益权衡的玩家

**故事基调**：光鲜中暗藏凶险，每一笔交易都可能是陷阱

## 四、边境联盟开局剧本

### 4.1 剧本标题
**《灰烬余生·自由之路》**

### 4.2 阵营背景

你站在暮光村的村中心，身后是那棵千年古树，面前是三界交汇的暮光山丘。

这里是灰烬谷，一个不属于任何国家的地方。暮光村是联盟最古老的村落，也是你的家乡。村长老根在这里维持着脆弱的和平——苍龙、霜狼、金雀花，三国的势力都在渗透这片土地。

你是暮光村的一名**年轻村民**——直到某一天，你发现自己卷入了一场改变命运的冲突。现在，你必须选择：是守护这片自由的故土，还是投靠某个阵营寻求庇护？

### 4.3 初始角色设定

**玩家身份**：
- 出身：暮光村本土人/苍龙逃难者/霜狼流浪者/金雀花破产商（四选一）
- 初始地位：平民（3级）
- 初始地点：暮光村·暮色酒馆

**初始技能**（根据出身不同）：
| 出身 | 初始技能 | 特色 |
|------|----------|------|
| 本土人 | 生存能力 L2 | 熟悉灰烬谷地形，擅长野外生存 |
| 苍龙逃难者 | 情报分析 L1 | 带来外部世界的知识与信息 |
| 霜狼流浪者 | 战斗技巧 L1 | 具备基础战斗能力 |
| 金雀花破产商 | 商业贸易 L1 | 了解商业运作，有经商头脑 |

**初始关系**：
- 与老根关系 +30（村长是你的保护人）
- 与小苏关系 +20（酒馆老板娘，情报来源）
- 与龙使/北风/金掌柜关系 0（三国代表，尚未接触）

### 4.4 起始场景

**场景：暮光村·暮色酒馆**

> 暮色酒馆是灰烬谷的情报中心，三国的消息在此交汇。今夜，酒馆里比往常更加热闹——边境巡逻队在村外发现了一队来历不明的武装人员。
>
> 老根坐在你对面，眉头紧锁。他已经六十有三，但眼神依然锐利。
>
> "孩子，"他轻声说，"世道要变了。那队武装人员，可能是苍龙的密探，也可能是霜狼的先锋，甚至可能是金雀花雇佣的佣兵。不管是谁，他们的目标都是灰烬谷。"
>
> 他顿了顿，目光落在你的脸上："你想站在哪一边？"

**场景人物**：
- 老根：暮光村村长，联盟的守护者
- 小苏：酒馆老板娘，消息灵通
- 烈焰：赤焰村村长，主张独立建国

### 4.5 第一个事件：暮光抉择

**事件名称**：《暮色酒馆的密会》

**事件类型**：效忠选择 + 危机应对

**事件描述**：

酒馆的灯火摇曳，四个人围坐在你的桌前——

- **老根**："我已经守护暮光村四十年。我不属于任何国家，我只属于这片土地。如果你愿意，可以帮我一起守护这份自由。"

- **烈焰**（赤焰村村长）："老根太软弱了！灰烬谷要么独立建国，要么早晚被吞并。我需要年轻人加入我的民兵，为自由而战！"

与此同时，酒馆角落里有三双眼睛在注视着你——

- **龙使**（苍龙使节）：礼貌地点头，暗示可以为你提供庇护
- **北风**（霜狼代表）：直率地举起酒杯，表示欢迎勇士
- **金掌柜**（金雀花商人）：微笑着展示一袋金币，意思不言自明

**可选行动**：

| 选项 | 前置条件 | 结果 |
|------|----------|------|
| [帮助老根维持中立] | 无 | 老根好感+40，获得「守村者」称号；触发「外交平衡」任务链 |
| [加入烈焰的独立运动] | 无 | 烈焰好感+35，老根好感-15（担心激化矛盾）；触发「建国之路」任务链 |
| [投靠苍龙] | 无 | 龙使好感+30，获得苍龙平民身份；触发「龙之子」任务链 |
| [投靠霜狼] | 无 | 北风好感+30，获得霜狼战士身份；触发「狼之血」任务链 |
| [投靠金雀花] | 无 | 金掌柜好感+30，获得100金币启动资金；触发「金之路」任务链 |
| [收集情报] | 情报分析L1+ | 发现那队武装人员其实是金雀花雇佣的佣兵，准备嫁祸给苍龙；获得重要情报 |
| [组织村民防御] | 战斗技巧L2+ 或 生存能力L2+ | 老根好感+25，村民支持度+20；获得「守护者」临时称号 |
| [保持观望] | 无 | 各方好感+5，但被视为墙头草；后续选择机会减少 |

**事件结果**：
- 帮助老根：进入联盟自治故事线，体验外交平衡与多方周旋
- 加入烈焰：进入独立建国故事线，体验武装斗争与建国理想
- 投靠某一国：获得该国阵营身份，但失去联盟中立优势
- 收集情报：获得信息优势，可在关键时刻影响局势

## 六、开局事件设计要点

### 6.1 共同设计原则

1. **强制选择**：每个开局都有一个核心选择，决定玩家的初始阵营倾向
2. **技能差异化**：不同技能等级解锁不同选项，体现技能系统的价值
3. **后续触发**：初始选择会触发不同的任务链，形成分支剧情
4. **信息差**：部分关键信息需要特定技能才能发现，鼓励多样化发展

### 6.2 与许愿系统的关联

玩家在开局后的**第一次许愿**（次日）将由对应层级的Agent计算：

| 阵营 | 许愿Agent | 许愿内容示例 |
|------|-----------|--------------|
| 苍龙 | 天命司·Chronos | "我该如何在夺嫡之争中保全自己？" |
| 霜狼 | 先祖议会·Chronos | "我应该支持改革还是传统？" |
| 金雀花 | 黄金议会·Chronos | "我该如何在商业竞争中获胜？" |
| 边境 | 谷主意志·Chronos | "灰烬谷的未来在哪里？" |

### 6.3 与技能系统的关联

开局事件是玩家**第一次体验技能系统**的场景：

- 技能等级决定可见选项数量
- 技能组合可能解锁特殊选项
- 技能选择影响后续事件触发

### 6.4 与叙事系统的关联

开局事件将作为玩家角色的**起源故事**，被记录在叙事系统中：

- 玩家的选择会被写入每日小说章节
- NPC会根据玩家的开局选择调整态度
- 主角NPC（天枢/苍狼/金潮/老根）会在后续剧情中提及玩家的起源
