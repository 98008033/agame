# 四阵营设定

## 三、四阵营设定

> **设计原则**：每个阵营是一个能自运转的社会。社会由NPC构成，NPC之间有等级、有关系、有利益冲突。没有玩家，这些社会也会自己运转——玩家只是一个外来的变量。

---

### 前置知识：一个前工业社会需要什么

根据历史研究，一个能自运转的前工业社会需要以下要素：

**人口职业分布**（参考中世纪欧洲/古代中国）：
```
农业人口    70-85%    生产粮食，养活所有人
手工工匠    5-10%     制造工具、武器、日用品
商贾贸易    3-5%      流通物资，连接供需
军队        2-5%      国防和治安
宗教/魔法   1-3%      精神需求、医疗、预言
行政管理    1-2%      收税、司法、政令
其他服务    2-5%      医疗、教育、娱乐、情报
```

**社会运转的九大支柱**：
```
1. 食物生产与分配    没有粮食 → 社会崩溃
2. 贸易与货币经济    没有流通 → 专业化不可持续
3. 治理与税收        没有公共事务管理 → 秩序瓦解
4. 军事防御          没有安全 → 一切归零
5. 知识传承          没有教育 → 技术退化
6. 宗教/魔法体系     没有精神纽带 → 社会凝聚力下降
7. 医疗与卫生        没有医疗 → 瘟疫灭国
8. 司法与冲突解决    没有仲裁 → 内战
9. 继承与权力过渡    没有继承机制 → 权力真空导致动荡
```

**关键角色类型**（每个社会必须覆盖）：
```
统治层    国王/首领 + 继承人 + 核心幕僚
军事层    统帅 + 将领 + 基层军官 + 精锐部队
行政层    宰相/总管 + 税务官 + 法官 + 地方长官
经济层    大商人/银行家 + 行会首领 + 粮商 + 武器商
宗教层    大祭司 + 治愈师 + 预言者 + 魔法研究者
知识层    学者 + 医生 + 教师 + 记录官
基层      村长/里正 + 工匠首领 + 农业管理者
灰色地带  间谍 + 走私者 + 土匪 + 地下势力
```

### 3.1 苍龙帝国

**关键词**：秩序、礼法、精密工艺、元素魔法

#### 国家概况

- **地理位置**：东部平原与河谷，沃野千里
- **人口**：800万
- **政体**：中央集权制，皇帝制度，六部内阁
- **经济结构**：
  - 农业人口：75%（稻米、丝绸、茶叶）
  - 手工业：8%（精密机关、丝绸纺织、陶瓷）
  - 商业：5%（国内贸易为主，相对封闭）
  - 军队：4%（正规军30万 + 边防驻军5万）
  - 行政/学者：4%（科举制度培养的官僚）
  - 宗教/魔法：2%（五行术士、道士）
  - 其他：2%
- **魔法体系**：元素操控术——金木水火土五行相生相克，讲究精确的法阵和仪式
- **科技**：精密机械工艺、水利工程、大型建筑技术
- **文化**：重礼法，等级森严，讲究"名正言顺"

#### 地理与城市

> **城市NPC规则**：每个城市有1名城主（Governor）+ 1-2名关键市民。城主是该城市的实际管理者，关键市民是玩家在该城市能交互的重要角色。其余职位（守卫队长、酒馆老板、铁匠等）由系统按城市类型自动生成，可被LLM动态命名。

#### 各城风土

**天都**
- 地理：坐落于龙脊山脉环抱的盆地中，三面环山，一面朝东面向平原。城墙由黑色玄武岩砌成，高三十丈，城墙上刻满五行法阵，夜间微微发光。
- 风土：城中居民以官僚、学者、侍从为主，街道按五行方位规划，东市卖丝绸茶叶，西市卖兵器铁器，南市卖珠宝香料，北市卖粮食物资。城中禁止大声喧哗，违者罚银。
- 景观：皇城金顶（远眺可见金光）、五行观星台（大国师玄清修炼处）、千阶白玉梯（通往皇宫的台阶，象征步步高升）
- 传说：相传建城时，始祖艾尔德兰的秩序之火坠落于此，化为城中心的"天火石"，帝国以此石为正统之证。谁掌握天火石，谁就是天命所归。
- 历史：建城已八百余年，历经十七朝。上一次皇位争夺在六十年前，导致"九龙之乱"，半座城市被焚毁，后重建。

**九龙城**
- 地理：位于天都以东的河谷平原，洛水穿城而过，将城市分为南北两半。北半为皇族封地，南半为平民居住区。
- 风土：皇族子弟众多，竞争气氛浓烈。北城街道安静肃穆，南城反而热闹繁华。城中流行"九龙赛马"——每年秋季，皇子们的马队在城外竞技。
- 景观：三座皇子府鼎足而立（天枢府庄严肃穆、破军府兵器林立、文曲府书卷满壁）、洛水上的九孔石桥（象征九龙）、马场（破军最爱的地方）
- 传说：洛水中曾有九条龙居住，大帝国开国皇帝在此驯服九龙，九龙化为九块龙鳞，分赐九子，后来九龙之乱即九子争位。
- 历史：此城本为皇族狩猎行宫，后因皇子成年后必须出宫就藩，逐渐演变为皇子聚居之城。夺嫡之争的序幕往往在此城拉开。

**铁壁关**
- 地理：建在龙脊山脉最北端的隘口，两侧是千仞绝壁，中间仅容五马并行。城墙直接利用山体，被称为"天造铁壁"。
- 风土：城中几乎全是军人和军属，女人也会舞刀弄枪。冬季大雪封山，守军常驻半年与外界隔绝。士兵之间流行"血酒"——用热血和烈酒混合，出征前必饮。
- 景观：铁壁主墙（传说有百丈之厚，从未被攻破）、点将台（将军誓师之处）、北望楼（可远眺霜狼联邦的冰原）
- 传说：建关之初，工匠首领以身殉墙，将自己铸入城墙之中，此后城墙坚不可摧。每年冬至，守军会在城墙上洒酒祭奠那位工匠。
- 历史：五十年前，霜狼联邦曾以十万大军猛攻铁壁关，围城三个月，最终因粮尽撤退。此战让铁壁关名扬天下，"铁壁不可破"成为帝国军民的信念。

**玉泉城**
- 地理：位于两条山脉交汇处的温泉地带，地下涌出的泉水含五行元素矿质，呈现五种颜色。城市沿温泉分布建造，到处蒸汽缭绕。
- 风土：城中居民以术士、学者、药剂师为主，街上常能看到衣着奇异的魔法师。城规极严：禁止在城内私斗魔法，违者逐出师门。
- 景观：五色温泉（金泉银泉蓝泉红泉黄泉，各有不同功效）、天师府（灵元的居所，外人不得入内）、五行学院（帝国最大的魔法学府，千余名学生）
- 传说：五色温泉据说是始祖之力渗入地下的痕迹。在五泉交汇处泡浴一整夜的人，据说能获得五行兼修的天赋——但上一个尝试的人疯了。
- 历史：天师府在此建府已有三百年，历代掌教都是帝国最强大的五行术士。灵元是第十七代掌教，也是最保守的一代。

**丝绸港**
- 地理：位于帝国东海岸的天然港湾，三面环山挡住海风，港湾内水深浪静。港外有暗礁群，不熟悉航路的船很容易触礁。
- 风土：城中鱼龙混杂——水手、商贩、走私者、异国旅人。港口区的酒馆彻夜不休，流行喝一种叫"海鬼"的烈酒（据说是用海兽胆汁酿的）。
- 景观：灯塔山（港口最高点，夜间有五行法阵指引航路）、船坞区（帝国唯一的造船厂，但规模远不及金雀花）、海上集市（渔民直接在船上卖鱼）
- 传说：港外的暗礁群下埋藏着大帝国时代的沉船宝藏。曾有人声称在水下看到了发光的金箱子，但下去打捞的人都消失了。
- 历史：丝绸港本是一个小渔村，一百年前帝国开放海禁后才逐渐发展。港务司·潮声是第一个从商人转为官员的人，打破了"商人不得为官"的传统。

**洛水城**
- 地理：位于洛水上游的丘陵地带，城市沿山坡而建，层层叠叠。城中到处是书院、藏书阁和科举考场。
- 风土：城中居民以读书人为主，街上随处可见手捧书卷的学子。城规：读书人过桥免费，非读书人过桥收三文钱——目的是鼓励读书。每年科举季，全城戒严。
- 景观：太学（帝国最高学府，千年古建筑）、千卷阁（藏书百万卷，帝国最大图书馆）、状元桥（据说过了这座桥的考生中状元的概率最高）
- 传说：千卷阁的最顶层有一本"天书"，据说是始祖留下的知识之源。只有学识达到极致的人才能读懂。但三百年前的一场大火后，顶层被封印了。
- 历史：洛水城出了三十七位首辅、一百五十二位尚书，被称为"宰相之城"。秋实就是洛水城出身。

**江南郡**
- 地理：位于帝国东南的大平原，河网密布，水田连绵至天际。春季油菜花金黄一片，秋季稻浪翻滚。
- 风土：城中居民以农民和粮商为主，生活节奏跟着节气走。民风淳朴但精明——江南郡的粮商出了名的会算计。节日时流行放河灯，祈求丰收。
- 景观：千亩稻田（帝国最大单片农田）、丰收塔（每年秋收时在此祭天）、粮仓群（帝国战略储备粮仓，据说够全帝国吃三年）
- 传说：江南郡地下埋着远古的"丰壤"，可以让庄稼永远丰收。但丰壤被一条沉睡的土龙守护着，惊醒土龙会导致大地震。
- 历史：江南郡曾是大帝国的粮仓，大帝国崩溃后一度荒废。苍龙帝国建国后重新开垦，用五行土系魔法改良土壤，使亩产翻倍。丰收郡守就是靠这个功绩上位的。

**灰烬谷前哨**
- 地理：建在灰烬谷入口的一座矮山上，三面悬崖，只有一条窄路通向灰烬谷。城中以木石混合建筑为主，简陋但实用。
- 风土：驻军和密探为主，几乎没有平民。士兵们以轮值方式驻防，每三个月换一批。城中唯一的酒馆是影蝶经营的，实际上是情报站。
- 景观：了望塔（可远眺灰烬谷全貌）、密道（织影修建的地下通道，据说通向灰烬谷深处）、血碑（纪念在此战死的将士）
- 传说：前哨地下有大帝国时代的遗迹入口，但入口被强大的封印封锁。曾有术士试图解开封印，结果被封印反噬而亡。
- 历史：前哨在三十年前建立，当时苍龙和霜狼在灰烬谷发生小规模冲突。此后这里成为帝国观察灰烬谷的"眼睛"，也是织影情报网的前沿。

#### 当前局势

- 老皇帝病重，三位皇子暗中夺嫡
- 主战派主张趁乱吞并灰烬谷，主和派担心内忧外患同时爆发
- 密探网络遍布灰烬谷，正在暗中培养亲苍龙势力

### 3.2 霜狼联邦

**关键词**：荣誉、武力、自然崇拜、符文战魂

#### 国家概况

- **地理位置**：北部冰原与山脉，资源贫瘠但矿藏丰富
- **人口**：200万
- **政体**：部落联盟制，各部族长老组成议会，首领由议会推选
- **经济结构**：
  - 畜牧/狩猎：60%（驯鹿、雪狼、冰原猎物）
  - 采矿/冶炼：15%（铁矿、寒冰矿脉、符文石）
  - 军事：10%（全民皆兵传统，可动员15万战士）
  - 符文/魔法：5%（符文铭刻、祖灵沟通）
  - 贸易：5%（矿石出口，近年发展商路）
  - 其他：5%
- **魔法体系**：符文战魂——在装备/身体上刻写符文赋予力量，祖灵在战斗中指引
- **科技**：简陋但实用，擅长冶炼、驯兽、寒地生存
- **文化**：崇尚武力、直率豪爽、重视荣誉和承诺

#### 地理与城市

#### 各城风土

**霜牙城**
- 地理：建在冰川裂谷的两侧崖壁上，两半城市由数条铁索桥连接。裂谷底部是深不见底的冰渊，据说通向地心。冬季极光出现时，整座城市被绿色和紫色的光芒笼罩。
- 风土：城中以战士、符文师和牧民为主，建筑用巨石和冻木搭建，风格粗犷。城中不设城门——裂谷本身就是天然屏障。居民习惯在屋顶挂兽骨风铃，风越大铃声越响，被称为"荣誉之声"。
- 景观：首领大帐（苍狼的居所，比普通帐篷大十倍）、符文石阵（议会所在地，十二根百尺石柱刻满符文）、冰渊桥（悬于裂谷之上，摇摇欲坠但从未断裂）
- 传说：冰渊底部住着远古冰龙的亡魂。霜狼的先祖曾与冰龙签订契约——冰龙守护霜牙城永不被外敌攻破，但每隔百年必须献祭一位最勇敢的战士。上次献祭是九十年前。
- 历史：霜牙城建城仅两百年，是霜狼联邦从游牧转为定居的标志。此前霜狼各部族逐水草而居，没有固定居所。苍狼的祖父是第一个主张建城的首领。

**铁炉堡**
- 地理：位于一座死火山的半山腰，火山口内有天然的地热和丰富的矿脉。城中到处是烟囱和锻造炉，天空常年灰蒙蒙的。
- 风土：灰熊部族的核心领地，城中居民以矿工和铁匠为主。城规第一条："不劳者不得食"。居民性格粗犷直率，说话像打铁一样干脆。城中最大的娱乐是"锻铁大赛"——用锤子敲出最精美的器物。
- 景观：火山口矿洞（帝国最大的矿脉入口，深入地下千尺）、百炉广场（一百座锻造炉同时燃烧，夜间火光冲天）、符文锻造台（只有最顶级的铁匠才能使用）
- 传说：火山深处有一块"星铁"——来自天外的陨石金属，用星铁锻造的武器无坚不摧。但星铁被火山之心的火灵守护，接近它的人会被烧成灰烬。
- 历史：铁炉堡发现矿脉是在一百五十年前。铜须家族世代管理矿脉，灰熊部族因此成为联邦最富裕的部族。矿脉分配一直是联邦内部矛盾的焦点。

**狼嚎谷**
- 地理：位于两座雪山之间的狭长山谷，山谷入口窄、内部宽，形如狼口。谷内终年狂风呼啸，声音如狼嚎。
- 风土：血狼部族的大本营，城中弥漫着好战气息。城规简单粗暴："强者为尊"。城中最大的建筑是角斗场，部族内部的争端用决斗解决。居民身上普遍有战斗疤痕，视之为荣誉。
- 景观：血色角斗场（用红色岩石建造，据说染过千人之血）、狼王崖（血狼部族首领宣誓就职的地方，站在崖边可俯瞰全谷）、铁笼监狱（关押战俘和异见者）
- 传说：狼嚎声不是风声，而是远古战死的霜狼战士的灵魂在嚎叫。在月圆之夜，如果你独自站在狼王崖上回应狼嚎，就能获得先祖战士的战斗直觉。
- 历史：狼嚎谷曾是霜狼与苍龙五十年前那场大战的主战场。谷中的红土据说就是被鲜血染红的。血狼·屠灭的父亲就战死于此，这也是他仇恨苍龙的根源。

**极光台**
- 地理：建在冰原最高的一座平顶山上，四面开阔无遮挡，是观赏极光的最佳位置。山顶有一座天然的水晶岩台，极光出现时水晶会发出共鸣。
- 风土：雪鹰部族的圣地，城中以符文师和萨满后裔为主（编者注：此处的"萨满"指古老传承的称呼，非魔法体系）。居民安静内敛，日出而作日落而息。城中禁止杀生——连猎食都在城外进行。
- 景观：极光水晶台（极光折射形成的七彩光柱）、符文图书馆（收藏联邦最古老的符文典籍）、天语塔（先知·天语的居所，常有神秘现象）
- 传说：极光是始祖沃坎的灵魂在天空跳舞。如果在极光最盛时在水晶台上冥想，可以获得变革魔潮的碎片力量——但这种力量不稳定，使用者可能获得超能力，也可能疯掉。
- 历史：极光台作为符文研究圣地已有三百年。雪鹰部族的历代符文大师都出自此地。雪鹰·望月在此修炼三十年，成为联邦最强大的符文师。

**驯鹿原**
- 地理：冰原南部的一片广袤草原，夏季短暂但草木茂盛，是冰原上唯一的绿色地带。冬季被大雪覆盖，驯鹿群在此迁徙。
- 风土：冰鹿部族的放牧领地，城中以牧民和商人为主。城中最热闹的地方是贸易集市——霜狼与苍龙、金雀花的商路在此交汇。居民热情好客，习惯用奶茶和烤肉招待客人。
- 景观：驯鹿迁徙路（每年秋季数十万头驯鹿从此经过，壮观至极）、贸易集市（冰原上最大的交易市场）、风马场（训练冰原战马的地方）
- 传说：驯鹿原地下有一条温泉暗河，是冰原唯一不结冰的水源。冰鹿部族的先祖发现这条暗河后才在此定居。据说暗河的尽头通向一个温暖的地下世界。
- 历史：驯鹿原是联邦最年轻的"城市"——十年前苍狼推行改革，鼓励发展贸易，冰鹿部族的草原·驯风在此建立了集市。集市的繁荣引发了保守派的不满，认为这会腐蚀联邦的传统。

**暗影哨**
- 地理：冰原最北端的一座石质哨塔，周围是无尽的冰原和永冻层。视野极佳但环境恶劣，冬季气温可达零下五十度。
- 风土：暗影部族的总部，城中以侦察兵和情报人员为主。居民沉默寡言，习惯独处。城中没有酒馆和集市——暗影部族认为这些都是"分散注意力的累赘"。
- 景观：千里眼（一座高百尺的了望塔，可以看到冰原尽头）、暗影密室（灰影处理情报的地方，外人不得入内）、冻土墓地（埋葬历代斥候，每座墓碑朝向不同的方向——象征他们守护的方位）
- 传说：冰原的尽头不是海，而是一道巨大的冰墙，冰墙后面是一片未知的黑暗。曾有斥候试图翻越冰墙，但再也没有回来。灰影相信，冰墙后面有某种存在正在观察这个世界。
- 历史：暗影哨建于三十年前，当时苍龙的斥候频繁出现在冰原上。灰影的父亲建立了这个哨所，灰影继承后将其扩展为联邦最大的情报据点。

### 3.3 金雀花王国

**关键词**：商业、契约、航海、炼金术

#### 国家概况

- **地理位置**：南部丘陵与海岸线，占据东西商路要道
- **人口**：500万
- **政体**：贵族议会制，实际上由商业寡头操控
- **经济结构**：
  - 农业：45%（葡萄酒、橄榄油、棉花）
  - 商业/金融：20%（大陆最大银行体系和商船队）
  - 手工业：12%（造船、炼金、奢侈品制造）
  - 军队：8%（海军为主，陆军较弱，依赖雇佣兵）
  - 航海/探索：5%
  - 宗教/魔法：5%（契约法师、炼金术士）
  - 其他：5%
- **魔法体系**：契约魔法——与异界存在签订契约，借用力量。效果强大但有代价
- **科技**：航海技术、炼金术、金融制度（银行、保险、期货雏形）
- **文化**：重利轻义、讲究契约精神、阶级流动相对自由

#### 地理与城市

#### 各城风土

**金雀花城**
- 地理：建在一个天然U形港湾的环抱中，城市沿山坡层层而上，最高处是贵族议会大厦，最低处是鱼市码头。城市被一条叫"金河"的人工运河分成东西两半。
- 风土：城中一切皆可交易——职位、情报、婚姻、甚至灵魂。城中流行"契约纹身"——把重要的契约条款纹在身上，以防对方反悔。市民见面打招呼不说"你好"，说"今天有什么好买卖"。
- 景观：金库穹顶（金潮的银行总部，屋顶是纯金打造，在阳光下耀眼夺目）、契约法师塔（十三座尖塔组成的建筑群，每座塔代表一位顶级法师）、赌船（停在港湾里的大型赌场，法律管不到的地方）
- 传说：金河运河底下埋着一箱"第一金币"——王国铸造的第一批金币，据说持有者会财运亨通。每隔几年就有人试图打捞，但河水会突然暴涨将人冲走。
- 历史：金雀花城原名"渔港镇"，两百年前被金潮的祖先买下，改造成商业中心。金潮家族用三代人的时间控制了王国的银行业，实现了"以商驭国"。

**黄金港**
- 地理：王国最大的天然深水港，港湾内可停泊千艘巨轮。港外有一座叫"金门"的两座小岛形成的天然门户，是商船进出的必经之路。
- 风土：城中居民以水手、船主、渔夫和码头工人为主。城中流行"海上赌命"——两个船长用各自的船和货物做赌注进行竞速赛。港口区的酒馆叫"醉鱼"，是全王国消息最灵通的地方。
- 景观：金门灯塔（照亮航路的巨大魔法灯塔，由契约法师维护）、千帆码头（可以看到各国商船的旗帜）、银帆旗舰（银帆的座舰"霜鲨号"，据说速度无人能及）
- 传说：金门两座小岛其实是远古海兽的两颗牙齿。海兽还活着，沉睡在海底。每当有海盗船试图强行通过金门，海兽就会在梦中翻身，掀起巨浪。
- 历史：黄金港在一百年前只是一个小渔港。金雀花的商人们发现这里是通往东方的最佳港口后，大量投资建设。银帆的祖父是第一任港务长，银帆家族从此与海洋结缘。

**炼金堡**
- 地理：建在一座富含矿石的丘陵上，城中到处是冒着彩色烟雾的烟囱。城外有一个巨大的陨石坑，据说是远古时代一颗魔力陨石坠落形成的。
- 风土：城中居民以炼金术士、契约法师和各种工匠为主。城中随时可能发生小型魔法爆炸，市民对此习以为常。城规：实验炸毁自己家可以，炸毁邻居的家要赔偿。
- 景观：陨石坑（中心区域被契约法师公会封锁，外人不得入内）、万药坊（全王国最大的药剂市场）、星屑的实验室（一间永远在变化的建筑，据说每天的格局都不一样）
- 传说：陨石坑中心有一块"原初之石"，是始祖力量的碎片。靠近它的人会看到自己最渴望的事物——但伸手去拿就会被弹开。星屑花了半辈子研究如何接近它。
- 历史：炼金堡的建立源于一百年前陨石的发现。契约法师公会在此建堡，将其发展为王国魔法研究的中心。契语是公会历史上最年轻的首席法师。

**葡萄丘**
- 地理：位于王国腹地的一片起伏丘陵，气候温和，土壤肥沃。漫山遍野都是葡萄园，秋季时整座山变成紫色。
- 风土：城中居民以酒农和传统贵族为主，生活方式古老保守。城中最有权势的不是官员，而是酒庄庄主。每年葡萄收获季，全城举办"酒神节"，连续狂欢七天。
- 景观：千年酒窖（地下挖出的巨型酒窖，据说里面有帝国时代的存酒）、酒神广场（节日庆典的中心）、旧冠的葡萄园（品质最好的葡萄产自旧冠家族的私人庄园）
- 传说：最好的葡萄酒需要用月光照射过的葡萄酿造。每年只有八月十五的葡萄才能达到这个条件，酿出的酒叫"月露"，一壶值千金。
- 历史：葡萄丘的传统贵族在这里经营酒庄已有五百年，比金雀花王国的历史还长。王国建立后，这些贵族被纳入议会体系，但他们始终怀念"贵族治国"的旧时光。

**香料城**
- 地理：位于王国最东端的山口，是从东方商路进入王国的第一个城市。城市建在两山之间的谷地，地形狭窄但位置关键。
- 风土：城中混居着苍龙人、金雀花人、甚至来自更遥远国度的旅人。街上能听到至少五种语言。城中最出名的是"百家宴"——各国移民各自做家乡菜，摆成一条长街。
- 景观：东门关（通往东方的山口，刻有"欢迎来发财"五个大字）、香料市场（各种奇异香料的味道混在一起，初来者会打喷嚏打一整天）、锦帛的货栈（堆满了来自东方的丝绸和瓷器）
- 传说：山口深处有一条秘密通道，据说是大帝国时代的走私路线。通过这条通道可以绕过所有关税。锦帛声称知道入口在哪里，但没人敢问他。
- 历史：香料城最初是东方商人的落脚点，后来逐渐形成城市。城中最有影响力的不是本地人，而是东方贸易商锦帛——他控制了王国七成的丝绸进口。

**银帆港**
- 地理：位于王国西南海岸的峡湾中，两侧是高耸的悬崖，峡湾内水深且隐蔽，是天然的军事港口。
- 风土：城中居民以造船工人和海军军官为主，气氛严肃。城规严格：军港内禁止饮酒、禁止赌博、禁止打架（违者罚划船一百海里）。唯一例外是凯旋归来那天。
- 景观：巨龙船坞（可以同时建造十艘战舰的巨型船坞）、海军学院（全王国最好的航海教育机构）、龙骨的工坊（造船大师龙骨的工作室，据说他造的船能在风暴中毫发无损）
- 传说：峡湾最深处沉睡着一艘大帝国时代的"龙骨战舰"，用整条远古海龙的骨架建造。龙骨大师造的每一艘船，都在模仿那艘沉船的结构。
- 历史：银帆港在五十年前只是一个修船的小港。金雀花决定建设海军后，大量投资扩建。铁锚是海军学院的第一届毕业生，也是唯一一个在学院里打败所有教官的学生。

**海盗湾**
- 地理：位于王国南部一群散乱的小岛之间，航道复杂，暗礁密布。不熟悉航路的人进去就出不来。
- 风土：城中没有法律，只有黑帆的规矩。居民以海盗、走私者、逃犯和亡命徒为主。城中唯一的货币是实力——你能打就能得到尊重。
- 景观：黑帆旗杆（海湾入口最高的旗杆上挂着黑帆的骷髅旗）、藏宝洞（据说黑帆把抢来的财宝藏在某个洞穴里）、决斗台（解决争端的地方，死了白死）
- 传说：海盗湾的前身是一个富庶的海港城市，三百年前被大帝国的舰队摧毁。沉没的城市在海底形成了一个巨大的迷宫，黑帆声称他找到了迷宫的入口。
- 历史：海盗湾的存在是公开的秘密。金雀花海军曾三次试图剿灭，但每次都因为航道复杂而失败。银帆发誓要亲手抓住黑帆。

- 海盗活动频繁，威胁商路安全

### 3.4 边境联盟

**关键词**：自由、多元、生存、左右逢源

#### 国家概况

- **地理位置**：灰烬谷地区，三国交界
- **人口**：5万，分散在数十个村落
- **政体**：松散的村落自治联盟，无统一领袖，重大事务由"谷主大会"商议
- **经济结构**：
  - 农业/狩猎：50%
  - 小型贸易：20%（三国商路中转站）
  - 手工业：10%（各取所长的杂学工匠）
  - 佣兵/冒险：10%（为三国提供雇佣服务）
  - 其他（情报/走私）：10%
- **魔法体系**：无统一魔法体系，但因地理位置可接触三系魔法碎片
- **科技**：各取所长，杂而不精
- **文化**：务实、灵活、对外来者包容

#### 对外定位：自由都市

> 边境联盟对外正式名称为**"灰烬谷自由都市"**，声称自己是一个独立、中立、开放的自治城邦。这个定位并非完全的谎言，也非完全的真相——联盟内部确实是各村自治，但远未达到一个统一"都市"的程度。
>
> "自由都市"的名号带来了实际利益：
> - 三国都愿意在此设使馆和贸易站（因为名义上不是敌国领土）
> - 商人愿意在此交易（因为"自由"意味着少税和少管）
> - 冒险者和佣兵愿意在此活动（因为没有复杂的法律）
>
> 但这个名号也带来了麻烦：
> - 三国都认为灰烬谷"理应"属于自己
> - 匪帮利用"自由"之名在此肆虐
> - 联盟内部有人想真正建国，有人只想维持现状

#### 各聚落风土

**暮光村**
- 地理：位于灰烬谷中央的一片缓坡上，周围是茂密的混交林。村后有一座小山丘，站在山顶可以看到灰烬谷全貌——东面是苍龙的山脉，北面是霜狼的冰原，南面是金雀花的丘陵。
- 风土：村中居民来源复杂——有逃难来的苍龙农夫、退伍的霜狼战士、破产的金雀花商人。大家都不问彼此的过去。村中最热闹的地方是小苏的酒馆"暮色"，三国的消息在此交汇。
- 景观：暮色酒馆（联盟情报中心）、老根的大树（村中心一棵千年古树，老根常坐在树下议事）、暮光山丘（可远眺三国地界）
- 传说：暮光村之所以叫"暮光"，是因为每天黄昏时分，夕阳同时照亮三个方向——东面金色（苍龙）、北面紫色（霜狼）、南面红色（金雀花）。老人们说，暮光村注定是三界交汇之地，谁也无法独占。
- 历史：暮光村建村于大帝国时代，是帝国旧都的郊区村落。帝国崩溃后，这里被遗忘了近百年，直到三国势力开始渗透灰烬谷，暮光村才重新进入视野。

**赤焰村**
- 地理：位于灰烬谷南部的一片红土地上，土壤富含铁矿，呈现赤红色。村中有几处天然温泉，水温极高。
- 风土：赤焰村是联盟中人口最多的村落，也是最激进的。居民以年轻冒险者和退伍军人为主，普遍对现状不满，认为联盟应该独立建国。烈焰是他们的精神领袖。
- 景观：赤焰温泉（水温可以煮熟鸡蛋）、誓言墙（村民用鲜血写下的独立誓言）、练兵场（烈焰组织的民兵训练地）
- 传说：赤焰村的红土地是远古时代一场天火留下的痕迹。据说天火中蕴含着始祖沃坎的变革之力——这也是赤焰村民性格激进的原因。
- 历史：赤焰村建村仅二十年，由一批从三国逃来的激进分子建立。烈焰是创始者之一，她梦想在灰烬谷建立一个不属于任何国家的独立国度。

**铁匠村**
- 地理：位于灰烬谷东北角的一条矿脉附近，村中有一座小型露天铁矿。村子被一圈简陋的石墙围住。
- 风土：铁匠村是联盟唯一的铁器锻造中心，村民以铁匠和矿工为主。性格沉默务实，不太关心政治——他们只关心有没有足够的铁矿和煤炭。厚土是村民中最受尊敬的人。
- 景观：百锤炉（村中心的大锻造炉，日夜不息）、矿洞入口（深入地下，产量有限但品质不错）、铁器集市（各国商人在此采购兵器和工具）
- 传说：铁匠村的矿脉尽头据说有一块"活铁"——一种会自己生长的金属矿石。如果能把活铁锻造成武器，武器会随主人的成长而变强。但矿洞太深，没人到达过尽头。
- 历史：铁匠村的前身是大帝国的一个军械作坊，帝国崩溃后工匠们留了下来，形成了村落。厚土的爷爷是第一批定居者。

**商贸村**
- 地理：位于灰烬谷最开阔的谷地上，周围地势平坦，适合商队扎营。村中有三条道路交汇——东通苍龙、北通霜狼、南通金雀花。
- 风土：商贸村是联盟最繁荣的村落，村民以商贩、旅店老板和车马行主为主。城中最重要的人不是村长，而是掌握货源的大商人。金算盘是其中最精明的一个。
- 景观：三岔路口（三条商路的交汇点，竖着三块指向不同方向的路牌）、货栈群（各国商人存放货物的仓库区）、算盘酒楼（金算盘的产业，也是商人们谈生意的地方）
- 传说：三岔路口的地下埋着大帝国时代的"商神像"——据说摸了它的人会财运亨通。但商神像的位置每年都会变，需要看风水才能找到。
- 历史：商贸村的存在是灰烬谷能维持和平的重要原因——三国都从这里的贸易中获利，谁也不愿打破现状。金算盘利用这一点，让商贸村成为联盟中最安全的地方。

**药草村**
- 地理：位于灰烬谷西侧的一片山谷中，溪水清澈，气候湿润，适合各种草药生长。村中到处是药圃和晾药架。
- 风土：药草村是联盟的医疗中心，村民以药农和治愈师为主。白芷是村中最出色的医者，三国都有人慕名前来求医。村中氛围安静祥和，与灰烬谷其他地方的紧张截然不同。
- 景观：百草园（种植上百种草药，部分有剧毒）、白芷的诊所（每天排着长队）、溪边采药小径（风景优美但有野兽出没）
- 传说：溪水上游有一个隐秘的洞穴，洞中长着一株"不死草"——据说吃了它可以起死回生。白芷曾试图寻找，但在洞口听到了可怕的低语声，退了回来。
- 历史：药草村最初是几个避世的医者建立的。白芷继承了师父的衣钵后，将小诊所发展成了联盟最大的医疗机构。三国都尊重药草村的中立地位——因为谁也不知道自己什么时候需要医者。

**侦察村**
- 地理：位于灰烬谷最外围的一座高地上，视野开阔，可以监视三条主要通路。村子被密林环绕，不易被发现。
- 风土：侦察村是联盟的"眼睛"，村民以猎人和侦察兵为主。猎鹰训练了一支精干的侦察队，每天轮班监视三国边境的动向。村中安静而警觉，连小孩都会辨认足迹。
- 景观：鹰眼台（最高的了望点，可以看到三国边境的军队调动）、陷阱迷宫（村子周围的防御陷阱）、猎鹰的藏书阁（意外地有一屋子关于三国军事的书籍）
- 传说：侦察村所在的高地在大帝国时代是帝国的"天眼"——一座用魔法驱动的远程监视塔。塔已经崩塌，但据说塔基的魔法阵还在运作，猎鹰的侦察队有时会收到奇怪的回声。
- 历史：侦察村建村十年，是猎鹰一手创建的。他原本是苍龙的斥候，因为一次任务失败被追杀，逃入灰烬谷后决定永远留在这里。他的侦察技能是联盟最宝贵的军事资源。

**灰烬市场**
- 地理：位于灰烬谷中心的一片空地上，没有固定建筑——只有搭了又拆的帐篷和摊位。市场周围的土地被踩得寸草不生。
- 风土：灰烬市场是联盟最大的集市，每月开市三次。市场上什么都有卖——粮食、兵器、魔法材料、情报、甚至人口。没有法律，只有"在市场里不许动刀"的默契。灰狐是市场中最神秘的商人。
- 景观：千帐灯（开市时上千盏灯笼同时点亮，夜间如星海）、灰狐的帐篷（永远在市场最中心，但帐篷里的东西每天都不一样）、决斗台（市场外用来解决纠纷的地方）
- 传说：灰烬市场的地下有一条远古的商道，据说是大帝国时代的贸易隧道。灰狐声称他在隧道中发现了通往三国腹地的秘密通道，但没人能证实。
- 历史：灰烬市场的历史比联盟还长——早在三国形成之前，灰烬谷就是各族交易的中转站。市场没有创立者，它是自然形成的。灰狐是最近十年才出现的，但已经控制了市场三成的交易量。
