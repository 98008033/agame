# 核心概念

## 一、核心概念

### 1.1 游戏定位

**一句话描述**：在一个由AI自主运转的乱世中，你选择效忠、背叛或自立，成为推动历史的那颗石子——或者被历史碾碎。

**核心卖点**：
- 世界在玩家不在线时依然运转——战争爆发、王朝更替、瘟疫蔓延
- 四个阵营，选择即命运：你可以选择效忠任何一国，也可以选择成为边境之王
- 真正的蝴蝶效应：小村庄的选择可能改变三国格局
- 隐藏的上帝之手：系统AI在幕后维持世界平衡，玩家只能感受到"时势"

**目标用户**：25-40岁，有碎片时间的已工作成年人，喜欢深度策略但厌倦重复操作

### 1.2 体验节奏

```
轻度入口 → 深度内核

日常：每日登录5-10分钟，查看世界晨报，做2-3个决策
深入：每周1-2次30分钟深度经营——领地建设、外交谈判、科技规划
极端：重大事件期间（战争/瘟疫/王位更迭）需要密集决策
赛季：每现实月为一个"赛季"，有赛季目标和排行，驱动长线留存
```
