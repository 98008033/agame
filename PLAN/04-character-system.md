# 角色体系与玩家体验

> **核心原则**：玩家和NPC在同一套规则下运行。属性、等级、技能、声望、效忠、背叛——都是世界通用的角色系统。玩家只是这个世界中一个被人类操控的角色。

---

## 四、角色体系

### 4.1 游戏开始：四选一

玩家进入游戏时选择阵营，这决定了起点体验：

| 阵营 | 起点体验 | 初始资源 | 初始挑战 |
|------|----------|----------|----------|
| **苍龙帝国** | 从帝国边陲小官做起，执行上级任务 | 有帝国后勤支持 | 官僚体系束缚，需要完成KPI |
| **霜狼联邦** | 从部落勇士做起，完成首领交代的使命 | 有部落战士支持 | 资源匮乏，需要靠实力说话 |
| **金雀花王国** | 从商人学徒做起，在灰烬谷建立贸易站 | 有启动资金 | 竞争对手多，需要商业头脑 |
| **边境联盟** | 暮光村新任村长，白手起家 | 几乎为零 | 自由度最高，但什么都要自己拼 |

**关键设计**：
- 阵营选择不是永久绑定
- 任何角色（玩家/NPC）都可以效忠、背叛、自立
- 每次效忠/背叛都有深远的后果——整个世界都会记住

### 4.3 等级与头衔（通用）

```
等级1-10：无名之辈
  └─ 村民、学徒、士兵

等级11-20：小有名气
  └─ 村长、队长、商人

等级21-30：一方豪强
  └─ 镇主、将军、富商

等级31-40：举足轻重
  └─ 城主、将军长、商会会长

等级41-50：影响格局
  └─ 藩王、元帅、财阀

等级51+：传奇人物
  └─ 国王、军阀、商业之王...
```

**升级方式（通用）**：
- 完成任务
- 达成成就
- 影响世界事件的结果
- 领地发展
- 战斗胜利

**NPC也会升级**：天枢如果在夺嫡中胜出，会从等级35提升到45+。苍狼如果改革成功，也会升级。NPC的成长和玩家的成长遵循同一套规则。

|-------|---------|----------|----------|

### 4.5 职业体系（技能决定职业）

**核心逻辑**：职业不是你选的，是你的技能组合自然决定的。你学了剑术和冶炼，你就是铸剑师。你学了贸易和谍报，你就是商业间谍。技能树上的每个节点都是一个可从事的角色。

#### 创新认证体系

**核心理念**：预设的技能树不是世界知识的全部——它只是"已被公认的部分"。角色（玩家或NPC）可以通过组合已有技能、在特定条件下尝试、被关键人物见证并认可，创造出全新的技术和职业。创新是世界进化的引擎。

##### 创新的三层结构

```
┌─────────────────────────────────────────────────────────┐
│  第三层：新职业                                          │
│  一种被公认的新社会角色，有明确的职责和地位                │
│  例："魔纹铁匠"——专门制造魔法武器的职业                  │
│  认证条件：需要权威机构（学院/行会/朝廷）正式承认         │
├─────────────────────────────────────────────────────────┤
│  第二层：新技术                                          │
│  一种前所未有的技能组合方式，可被他人学习                  │
│  例："五行附体"——将五行元素附着在自身而非武器上的技巧     │
│  认证条件：需要在关键场合展示并被专家认可                 │
├─────────────────────────────────────────────────────────┤
│  第一层：新尝试                                          │
│  一种个人的实验性组合，尚未被验证                         │
│  例：你尝试将情报分析的技巧应用到商业谈判中               │
│  结果：可能成功（解锁新技术），可能失败（浪费时间/信誉）  │
└─────────────────────────────────────────────────────────┘
```

##### 创新条件：天时 · 地利 · 人和

创新不是随便组合就能成功的，需要三个条件同时满足：

**天时——时机成熟**
```
- 世界状态满足特定条件（如：某项技术发展到瓶颈、战争催生需求）
- 角色自身状态满足（技能等级、属性达标、身体/精神状态良好）
- 概率因素（即使条件满足，也不是100%成功）

示例：
  你在和平年代尝试研究新的军事战术 → 成功率低（没有需求驱动）
  你在战争年代尝试研究新的军事战术 → 成功率高（天时已到）
```

**地利——环境契合**
```
- 需要在合适的环境中尝试（魔法实验需要魔力充沛的地方）
- 需要合适的材料和工具
- 某些创新只在特定阵营/地区可行

示例：
  在天都尝试政治操控创新 → 有官僚体系支持，成功率+20%
  在边境尝试军事指挥创新 → 实战环境丰富，成功率+15%
  在金雀花尝试商业贸易创新 → 有金融系统支持，成功率+15%
```

**人和——见证与认可**
```
这是最关键的条件：你的创新必须被"对的人"看到并认可。

见证者等级：
  - 普通人见证 → 只能传播小道消息，无法认证
  - 同行专家见证 → 可以验证技术真实性，但影响力有限
  - 权威人物见证 → 可以正式认证，让新技术被世界承认
  - 多位权威同时见证 → 高级认证，新技术立刻获得广泛认可

认可方式：
  - 公开演示：在众人面前成功施展，赢得喝彩
  - 对决验证：在战斗/比赛中使用，击败对手
  - 机构审查：提交给学院/行会/朝廷审查
  - 门派传承：被大师收为弟子，师徒共同完善

没有被见证的创新：
  - 你可能成功创造了新技术
  - 但世界不会知道——它只是你个人的秘密
  - 除非你找到见证者，或者你的弟子在公开场合使用
```

##### 认证流程

```
第一阶段：尝试（个人行为）
  角色选择2-3个已有技能进行组合尝试
  系统判定：天时 + 地利 + 角色属性 → 成功/失败/部分成功
  结果：
    完全失败 → 浪费材料/时间，可能受伤
    部分成功 → 获得不完整的新技术，需要继续完善
    完全成功 → 获得新技术（但尚未认证）

第二阶段：验证（需要见证）
  角色需要在见证者面前成功施展新技术
  见证者类型决定认证级别：
    ├─ 普通场景（如在酒馆里展示）→ 仅限口耳相传
    ├─ 公开场合（如在广场/擂台展示）→ 区域性认可
    ├─ 权威场合（如在学院/朝廷展示）→ 国家级认可
    └─ 跨阵营场合（如在三国共同的场合展示）→ 大陆级认可

第三阶段：传播（新技术扩散）
  认证后的技术可以被他人学习
  学习方式：
    ├─ 拜师学艺（向创造者直接学习，最快）
    ├─ 文献研究（如果有被记录下来，较慢）
    └─ 独立重现（其他人在知道原理后自行尝试，最慢）
```

##### 创新示例

**示例1：玩家创造新技术**
```
场景：你是一名边境指挥官（武力：军事指挥L4 + 经营：产业管理L3）
你尝试用商业物资调配的逻辑来优化军队后勤补给

尝试条件：
  天时：战争期间，补给紧张 ✓
  地利：在边境地区，后勤压力大 ✓
  角色：军事指挥L4 + 产业管理L3，属性达标 ✓

系统判定：成功率65% → 成功！

结果：你创造了"军商统筹术"——用商业贸易的方式管理军队后勤，
大幅提高补给效率。但这只是"未认证"状态——只有你自己能用。

后续：你需要找一个见证者来认证。
  - 在军队内部展示 → 口耳相传，边境驻军知道了
  - 在军演中展示给铁壁将军看 → 将军认可，成为边境驻军可学习的技能
  - 提交给兵部审查 → 如果通过，成为帝国军队标准后勤技能
```

**示例2（MVP后扩展）：NPC创造新职业**
```
场景：当MVP后引入魔法和锻造系统时
炼金堡的星屑（魔法L5）+ 契语的弟子灵约（契约术L3）

星屑和灵约合作研究，尝试将炼金术和契约术融合
经过多次失败后，他们创造了"契约炼成"

认证过程：
  1. 在炼金堡公开展示 → 炼金术士们震惊
  2. 契语（首席契约法师）亲自验证 → 认可技术真实性
  3. 金潮（议长）听闻后投资推广 → "契约炼成师"成为金雀花的新职业

影响：
  - 新职业"契约炼成师"出现在技能树中
  - 这个职业以前不存在——是被创造出来的
```

**示例3：未被见证的创新**
```
场景：荒原狼在荒野中独自作战时，无意中将战斗技巧和商业直觉融合
他创造出了一种"掠夺式贸易"——先武力威慑，再低价收购

但荒原狼没有找见证者，这件事只有他自己知道
"掠夺式贸易"成为他的个人秘密技能

后来：他的副手·独眼在一次行动中看到他使用了这个技巧
如果独眼学会并公开使用 → 这个技术才开始传播
如果独眼也保持沉默 → 这个技术仍然是秘密

极端情况：如果荒原狼突然死亡且没有传承 → 这个技术随他一起消失
```

##### 创新对世界的影响

```
技术创新 → 改变战争形态
  "军商统筹术"的出现 → 后勤效率成为战场胜负关键 → 推动军事与经济的深度融合

职业创新 → 改变社会结构
  "商业政治家"的出现 → 能用经济手段影响政治 → 金雀花的政商融合加深

失传技术 → 世界退步
  某位大师去世且没有传人 → 他的独门技术消失 → 世界失去一种可能性

技术扩散 → 阵营平衡变化
  苍龙的技术被霜狼学会 → 霜狼军事实力提升 → 三国平衡被打破
```

### 4.6 效忠与背叛系统（通用——NPC也会效忠和背叛）

这套系统对所有角色生效。NPC也会做出效忠和背叛的决定，并产生深远后果。

```
效忠流程：

  [角色当前状态] ──→ 可以向任意一国提出效忠
        │
        ▼
  [效忠条件审查] ──→ 该国是否需要你？你的价值是什么？
        │
        ▼
  [效忠代价] ──→ 需要完成效忠任务（可能是道德困境）
        │
        ▼
  [效忠获益] ──→ 获得该国资源、军事、科技支持
        │
        ▼
  [效忠约束] ──→ 必须执行该国的命令（即使你不想）
        │
        ▼
  [背叛条件] ──→ 随时可以背叛，但代价巨大
        │
        ▼
  [背叛后果] ──→ 被原阵营追杀，新阵营可能不信任你
                  所有角色记住这次背叛行为
                  可能获得"叛徒"声望标签
```

**效忠层次**：
```
疏离 → 中立 → 友好 → 效忠 → 核心 → 传奇

疏离：被视为外人
中立：正常贸易
友好：可以获得任务和优惠
效忠：获得正式身份和资源支持
核心：参与国家高层决策
传奇：成为国家历史的一部分
```

**背叛的类型和后果**（通用）：

| 背叛类型 | 后果 | 可恢复性 |
|----------|------|----------|
| 暗中转投 | 原阵营关系-50，新阵营+30但怀疑你 | 可通过长期忠诚修复 |
| 公开反叛 | 原阵营关系-100，被视为死敌 | 几乎不可逆 |
| 自立为王 | 所有阵营关系-30，被视为威胁 | 取决于实力 |
| 回归中立 | 脱离所有阵营，损失所有特权 | 可重新开始 |

**NPC的效忠/背叛**：
- 血狼·屠灭（霜狼强硬派）可能在某个事件中公开反叛苍狼
- 天枢如果被逼到绝境，可能暗中联络金雀花
- 织影（苍龙情报头子）可能同时为多方效力（双面间谍）
- 这些NPC的效忠/背叛由国家Agent和NPC Agent根据世界状态动态决定

---

### 4.7 成长的影响力（通用）

任何角色变强了，世界都知道。

| 成长阶段 | 世界如何响应 |
|----------|-------------|
| 无名之辈 | 无人关注 |
| 小有名气 | 同行开始注意，有机会接触更高层 |
| 一方豪强 | 国家层面开始关注，拉拢或警惕 |
| 举足轻重 | 重大决策需要考虑你的立场 |
| 影响格局 | 你的选择能改变一场战争的走向 |
| 传奇人物 | 你是历史的书写者 |

**反制机制**（由Chronos主脑Agent控制——对所有强大角色生效）：
- 某角色过于强大 → 其他势力会联合压制
- 某角色过于弱小 → 会有机遇出现
- 某阵营过于偏激 → 会遭遇系统性阻力

## 五、玩家体验层

> 以下是玩家视角独有的设计，不属于世界通用规则，而是"人类操控一个角色"时的体验设计。

### 5.1 日常体验循环

**短期循环**（每次登录5-10分钟）：
```
登录 → 读世界晨报（新鲜感/信息获取）
     → 收获定时奖励（领地产出/任务完成）
     → 做2-3个关键决策（掌控感）
     → 查看之前的决策后果（因果满足）
     → 离开 → 世界继续运转
```

**中期循环**（每周1-2次深度游玩30分钟）：
```
进入 → 查看一周世界大事
     → 规划领地建设（战略决策）
     → 推进外交行动（关系经营）
     → 完成一个事件链（叙事体验）
     → 感受到自己的影响在扩大
```

**长期循环**（每现实月一个"赛季"）：
```
赛季目标 → 完成赛季任务获得赛季奖励
         → 赛季排行榜
         → 赛季大事件（影响世界格局）
         → 新赛季开始
```

### 5.2 心流触发器

| 触发器 | 机制 | 效果 |
|--------|------|------|
| 因果满足 | 3天前的决策今天产生结果 | "我的选择真的有影响！" |
| 稀缺信息 | 情报只有特定时间/条件能获取 | "错过了就没了" |
| 社交压力 | NPC关系在变化，不维护会恶化 | "我得去看看老朋友" |
| 渐进解锁 | 每个阶段都有新内容 | "下一步会有什么？" |
| 身份认同 | 身份和头衔在变化 | "我是这个世界的一部分" |
| 悬念驱动 | 事件链中断，等待下一章 | "接下来会怎样？" |
| 荣誉竞争 | 赛季排行/成就/称号 | "我要比别人更强" |

---

### 5.3 玩家视角的特殊性

**玩家能看到而NPC看不到的**：
- 世界晨报（全大陆视角的新闻汇总）
- 赛季目标和排行榜
- 技能树全貌（NPC只知道自己的技能，不知道完整的技能树结构）
- 声望数值（NPC只能感知大致态度，不能看到精确数字）

**玩家做得到而NPC做不到的**：
- 存档/读档（NPC的世界是不可逆的）
- 选择阵营起点（NPC的出身是世界决定的）
- 随时切换操控方向（NPC被自己的性格和目标绑定）

**玩家做不到而NPC做得到的**：
- 玩家不能同时操控多个角色
- 玩家不能7x24小时在线（NPC在Agent调度下持续运转）
- 玩家需要通过界面交互（NPC之间可以直接通信）

这种不对称正是游戏乐趣的来源：你和NPC在同一个世界、同一套规则下竞争，但你有独特的优势（全知视角、灵活选择）和独特的劣势（时间有限、只能操控一人）。
