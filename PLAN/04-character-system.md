# 角色体系与玩家体验

> **核心原则**：玩家和NPC在同一套规则下运行。属性、等级、技能、声望、效忠、背叛——都是世界通用的角色系统。玩家只是这个世界中一个被人类操控的角色。

---

## 四、角色体系

### 4.1 游戏开始：四选一

玩家进入游戏时选择阵营起点，但**所有玩家都从普通老百姓开始**：

| 阵营 | 起点位置 | 初始状态 | 特色 |
|------|----------|----------|------|
| **苍龙帝国** | 帝都附近村庄/小镇 | 普通村民/流民/学徒 | 可选择向官僚、军事、商业、学术任意方向发展 |
| **霜狼联邦** | 霜牙城附近部落/定居点 | 普通牧民/矿工/学徒 | 可选择向战士、符文师、商人、工匠任意方向发展 |
| **金雀花王国** | 金雀花城附近城镇 | 普通市民/学徒/水手 | 可选择向商业、航海、炼金、政治任意方向发展 |
| **边境联盟** | 暮光村（或其他边境村落） | 普通村民/流浪者 | 自由度最高，可向任何方向发展 |

**核心设计原则**：
- **无预设职业**：玩家不是"小官"、"勇士"、"学徒"，而是**普通老百姓**
- **技能决定一切**：你想成为什么，取决于你学习了什么技能
- **动态发展**：今天你是农民，明天可能成为将军——只要你学会了军事技能
- **阵营可随时改变**：通过效忠/背叛/自立，不绑定初始选择

**起点示例**：
```
玩家A选择苍龙帝国：
  → 起点：帝都外某村庄的普通村民
  → 可以：种地、打猎、学习、找工作
  → 发展方向：完全自由
    - 学谋略+效忠官僚 → 可能成为官员
    - 学战斗+参军 → 可能成为将军
    - 学经营+做生意 → 可能成为富商
    - 学魔法+研究 → 可能成为法师

玩家B选择霜狼联邦：
  → 起点：驯鹿原的普通牧民
  → 可以：放牧、打猎、交易、学习符文
  → 发展方向：完全自由
    - 学战斗+决斗 → 可能成为战士
    - 学符文+修炼 → 可能成为符文师
    - 学经营+贸易 → 可能成为商人
```

### 4.2 角色属性系统（十维能力图）

> **设计理念**：属性不是数字，是"你是什么样的人"。每个属性都影响你能做什么、怎么做、以及世界如何回应你。

#### 4.2.1 十维能力图

```
                    [意志] 精神核心
                       △
                      /|\
                     / | \
        [智慧] ←———·———→ [魅力]
       认知能力    |    社交能力
                  |
    [体魄] ←———·———·———→ [敏捷]
   身体基础      |      身体控制
                |
    [幸运] ←———·———·———→ [感知]
   命运眷顾      |      信息获取
                |
    [名望] ←———·———→ [恶名]
   正面声誉      |      负面威慑
                |
           [阵营声望]
          各方势力态度
```

#### 4.2.2 属性详解与游戏作用

**【第一组：身体属性】—— 决定你能做什么**

| 属性 | 含义 | 影响的游戏内容 | 提升方式 |
|------|------|----------------|----------|
| **体魄** | 身体强度、耐力、生命力 | • 负重上限<br>• 生命值上限<br>• 疾病抵抗力<br>• 体力劳动效率 | • 战斗/劳作<br>• 体育锻炼<br>• 特殊训练<br>• 随着年龄增长而自然下降（30岁后） |
| **敏捷** | 反应速度、协调性、灵活度 | • 闪避率<br>• 行动顺序<br>• 精细操作成功率<br>• 逃跑成功率 | • 战斗/探险<br>• 特殊训练<br>• 年轻时期自然较高<br>• 随年龄缓慢下降 |

**身体属性 ↔ 技能关联**：
```
战斗技巧要求：体魄 ≥ 技能等级 × 15
锻造/工艺要求：敏捷 ≥ 技能等级 × 10
军事指挥无身体要求（脑力技能）
```

**【第二组：心智属性】—— 决定你能学会什么**

| 属性 | 含义 | 影响的游戏内容 | 提升方式 |
|------|------|----------------|----------|
| **智慧** | 理解力、记忆力、学习能力 | • 学习新技能速度<br>• 技能突破成功率<br>• 创新成功率<br>• 识别谎言/陷阱 | • 学习/阅读<br>• 解谜/探索<br>• 年龄增长（到40岁）<br>• 受伤/疾病可能下降 |
| **意志** | 精神强度、抗压能力、专注度 | • 技能持续训练不疲劳<br>• 拒绝诱惑/胁迫<br>• 承受痛苦能力<br>• 魔法/诅咒抵抗 | • 克服困难事件<br>• 坚持长期目标<br>• 特殊精神训练<br>• 重大挫折可能下降 |
| **感知** | 观察力、直觉、信息敏感度 | • 发现隐藏信息<br>• 察觉危险/敌意<br>• 情报质量<br>• 先攻判定 | • 侦察/探索<br>• 冥想/专注<br>• 经验积累<br>• 随着年龄增长（到50岁） |

**心智属性 ↔ 技能关联**：
```
谋略系技能要求：智慧 + 感知 ≥ 技能等级 × 20
魔法系技能要求：意志 ≥ 技能等级 × 15
学习速度 = 基础时间 × (100 / 智慧)
```

**【第三组：社交属性】—— 决定世界如何回应你**

| 属性 | 含义 | 影响的游戏内容 | 提升方式 |
|------|------|----------------|----------|
| **魅力** | 吸引力、说服力、领导力 | • 谈判成功率<br>• 招募NPC成功率<br>• 命令服从度<br>• 初次印象 | • 社交活动<br>• 成功领导事件<br>• 特殊训练（演讲/礼仪）<br>• 随着年龄变化（青年 peak） |
| **名望** | 正面声誉、英雄事迹 | • 获得帮助概率<br>• 交易优惠<br>• 被推举为领袖<br>• 正面事件触发 | • 完成公共任务<br>• 英雄行为<br>• 维护正义<br>• 背叛/恶行会下降 |
| **恶名** | 威慑力、恐惧、残忍名声 | • 威慑敌人<br>• 获得畏惧而非爱戴<br>• 黑暗势力拉拢<br>• 负面事件触发 | • 残忍行为<br>• 背叛盟友<br>• 暗杀/恐吓<br>• 行善/和解会下降 |

**社交属性 ↔ 技能关联**：
```
政治操控要求：魅力 ≥ 技能等级 × 12
外交/说服效果 = 基础效果 × (魅力 / 50)
名望/恶名决定某些事件的解锁（如：只有名望80+才能被推举为王）
```

**【第四组：命运属性】—— 决定机遇与风险**

| 属性 | 含义 | 影响的游戏内容 | 提升方式 |
|------|------|----------------|----------|
| **幸运** | 随机事件偏向、意外收获 | • 随机事件好坏倾向<br>• 意外发现概率<br>• 失败时的补救机会<br>• 赌博/抽奖 | • 不可主动训练<br>• 特定事件/仪式可能临时提升<br>• 长期好运/厄运会累积/消耗<br>• 不同阵营有基础幸运差异 |

**幸运的特殊机制**：
```
幸运不是固定值，而是"浮动值"：
- 基础幸运：角色固有（随机生成，30-70）
- 当前幸运：随事件波动
- 幸运耗尽：连续好运后必定厄运（回归均值）
- 幸运累积：厄运后必定有补偿

应用：
- 探险发现：幸运检定
- 战斗暴击：幸运加成
- 事件结果：幸运偏向
```

**【第五组：阵营声望】—— 决定你在世界的位置**

| 阵营 | 声望范围 | 声望影响 |
|------|----------|----------|
| 苍龙帝国 | -100 ~ +100 | 决定你在帝国的待遇、任务来源、晋升可能 |
| 霜狼联邦 | -100 ~ +100 | 决定部落对你的态度、交易价格、决斗挑战 |
| 金雀花王国 | -100 ~ +100 | 决定商业机会、法律保护、融资可能 |
| 边境联盟 | -100 ~ +100 | 决定村民的接纳度、任务委托、庇护提供 |

**声望 ↔ 游戏内容**：
```
声望 ≥ 80：被视为英雄/领袖，获得特殊权限
声望 20-80：友好，正常交易和互动
声望 -20~20：中立，需建立关系
声望 ≤ -20：敌对，可能被攻击/拒绝服务
声望 ≤ -80：死敌，悬赏追杀

效忠某阵营后：
- 该阵营声望自动提升到50+
- 敌对阵营声望可能下降
```

#### 4.2.4 属性改变方式详解

**自然变化**（随时间自动发生）：
```
年龄影响（每游戏年）：
- 15-25岁：体魄、敏捷自然上升
- 25-40岁：智慧、感知自然上升
- 40-55岁：意志、魅力达峰值
- 55岁后：体魄、敏捷自然下降
- 70岁后：智慧可能下降（老年痴呆）

长期行为影响（每游戏月）：
- 持续战斗 → 体魄+敏捷小幅上升
- 持续学习 → 智慧+意志小幅上升
- 持续社交 → 魅力+感知小幅上升
- 长期无所事事 → 全属性小幅下降
```

**主动训练**（玩家/NPC主动行为）：
```
体魄训练：
- 负重行走
- 高强度劳作
- 军事训练
- 效果：每月 +1-3，上限取决于年龄

智慧训练：
- 阅读书籍
- 解谜探索
- 向智者请教
- 效果：每月 +1-2，需要持续投入

意志训练：
- 坚持困难任务
- 拒绝诱惑
- 忍受痛苦
- 效果：事件触发，非持续

魅力训练：
- 公开演讲
- 社交活动
- 成功领导
- 效果：每次成功社交 +0.5-1
```

**事件触发**（重大事件影响）：
```
正面事件：
- 大难不死 → 意志+5，幸运+10（临时）
- 成功创新 → 智慧+3，名望+10
- 拯救村庄 → 名望+20，魅力+2
- 击败强敌 → 体魄感知+2，恶名+5

负面事件：
- 重伤不愈 → 体魄-5至-20
- 重大失败 → 意志-3，幸运-10（临时）
- 背叛盟友 → 恶名+15，名望-10
- 被囚禁折磨 → 意志+5（坚强）或-10（崩溃）
```

**外部干预**（他人/环境影响）：
```
魔法/药剂：
- 某些药剂可临时提升属性
- 某些魔法可永久改变属性（代价巨大）
- 契约魔法可借用属性（有代价）

装备影响：
- 某些装备可提升属性（穿戴时）
- 符文装备可永久提升（极少）

环境影响：
- 在智慧之地学习 → 智慧+20%（临时）
- 在战争之地 → 体魄+10%（临时），魅力-10%（临时）
```

- 魅力高的NPC在谈判中更难被说服
- 意志高的NPC不容易被胁迫/利诱背叛
- 恶名高的NPC可能主动提出非法合作

### 4.3 等级与头衔系统

#### 4.3.1 等级系统（1-100级）

**等级代表什么**：你在世界中的积累——经验、能力、影响力的综合体现。

| 等级段 | 称号 | 含义 |
|--------|------|------|
| 1-20级 | 无名小卒 | 刚开始，没人认识你 |
| 21-40级 | 略有名气 | 在本地有了一定声望 |
| 41-60级 | 一方人物 | 在区域内被认可 |
| 61-80级 | 举足轻重 | 你的决定影响他人命运 |
| 81-99级 | 影响格局 | 能改变势力走向 |
| 100级 | 传奇 | 成为历史的一部分 |

#### 4.3.2 等级提升方式

**核心原则**：等级提升不依赖单一途径，任何领域的成就都能积累经验。

```
【经验获取途径】

1. 【任务完成】
   - 完成任何任务（主线/支线/日常）
   - 经验 = 任务难度 × 完成质量
   - 例：成功护送商队（难度20）+ 无损失（质量1.5）= 30经验

2. 【技能提升】
   - 每次技能升级获得经验
   - 经验 = 技能等级 × 10
   - 例：战斗技巧从L2升到L3 = 30经验

3. 【成就达成】
   - 完成特定成就（首次击杀、首次交易、首次创新等）
   - 经验 = 成就等级 × 50
   - 例：首次击败BOSS = 100经验

4. 【事件参与】
   - 参与世界事件（战争、政变、瘟疫等）
   - 经验 = 事件影响 × 参与程度
   - 例：参与并影响一场战役 = 200-500经验

5. 【社会认可】
   - 获得新身份（被任命为村长、被推举为领袖等）
   - 经验 = 身份等级 × 100
   - 例：被任命为村长 = 200经验

6. 【创新/创造】（高难度）
   - 创造新技术/新职业/新方法
   - 经验 = 创新影响 × 200
   - 例：创造被认可的新技术 = 500-1000经验

7. 【教学传承】
   - 教导他人技能（成为导师）
   - 经验 = 学生进步 × 20
   - 例：教导的NPC技能提升 = 每次+20经验

8. 【领地发展】
   - 管理的领地发展壮大
   - 经验 = 领地发展指标 × 月度累计
   - 例：村庄人口增长、收入提升 = 每月+50-200经验
```

**不同发展路线的经验侧重**：

| 发展路线 | 主要经验来源 | 辅助经验来源 |
|----------|--------------|--------------|
| **战斗路线** | 任务(战斗)、战斗胜利 | 技能提升、成就 |
| **谋略路线** | 事件参与、社会认可 | 任务(谋略)、创新 |
| **经营路线** | 领地发展、社会认可 | 任务(商业)、成就 |
| **工匠路线** | 技能提升、创新 | 任务(制作)、教学传承 |
| **混合路线** | 均衡获取 | 根据当前行为动态变化 |

**等级提升速度**：
```
早期（1-30级）：快速成长期
  - 每次任务/技能提升都能明显感受到进度
  - 约游戏时间1-2个月达到30级

中期（31-60级）：稳步积累期
  - 需要持续投入，单次收益相对降低
  - 约游戏时间3-6个月达到60级

后期（61-90级）：艰难攀升期
  - 只有重大成就才能显著提升
  - 约游戏时间6-12个月达到90级

顶级（91-100级）：传奇铸就期
  - 必须改变世界格局的事件才能提升
  - 可能需要1-2年现实时间
```

#### 4.3.3 头衔系统

**头衔是什么**：社会对你的正式认可，通常与身份、技能、成就相关。

**头衔获取途径**：

```
1. 【身份头衔】—— 由效忠关系获得
   条件：被任命为某职位
   例：
   - 被苍龙任命为"边境守备" → 获得头衔
   - 被推举为"村长" → 获得头衔
   - 效忠金雀花成为"商会代表" → 获得头衔

   特点：
   - 失去身份即失去头衔
   - 背叛/被罢免后头衔消失
   - 可被剥夺

2. 【技能头衔】—— 由技能等级获得
   条件：某项技能达到特定等级
   例：
   - 战斗技巧L6 → "剑术大师"
   - 商业贸易L6 → "商业巨擘"
   - 锻造L6 → "神匠"

   特点：
   - 技能不遗忘则头衔保留
   - 可通过技能退化失去
   - 可被挑战/质疑（需维护）

3. 【成就头衔】—— 由特定成就获得
   条件：完成特定壮举
   例：
   - 单枪匹马击败千人军队 → "万人敌"
   - 创造被广泛认可的新技术 → "创新者"
   - 拯救一座城市免于毁灭 → "救世主"

   特点：
   - 永久保留（除非被证伪）
   - 可叠加
   - 有具体事迹支撑

4. 【声望头衔】—— 由名望/恶名获得
   条件：声望达到特定阈值
   例：
   - 名望≥80 → "受人尊敬的英雄"
   - 恶名≥80 → "令人恐惧的恶魔"
   - 名望≥50且恶名≥50 → "亦正亦邪的传奇"

   特点：
   - 随声望动态变化
   - 可能同时拥有多个
   - 影响NPC互动
```

**头衔示例表**：

| 类型 | 头衔 | 获取条件 | 效果 |
|------|------|----------|------|
| 身份 | 边境守备 | 苍龙任命的边境军官 | 可调动边境驻军 |
| 身份 | 村长 | 被村民推举/任命 | 管理村庄权限 |
| 身份 | 商会代表 | 金雀花商会认可 | 获得商会资源支持 |
| 技能 | 剑术大师 | 战斗技巧L6 | 可开设武馆收徒 |
| 技能 | 商业巨擘 | 商业贸易L6 | 交易价格优惠20% |
| 技能 | 神匠 | 锻造L6 | 可打造传说装备 |
| 成就 | 万人敌 | 单挑战胜1000人军队 | 威慑+50，敌军士气-20% |
| 成就 | 创新者 | 创造3项被认证的技术 | 创新成功率+20% |
| 成就 | 救世主 | 拯救10万+人口 | 全阵营声望+20 |
| 声望 | 受人尊敬的英雄 | 名望≥80 | NPC主动提供帮助 |
| 声望 | 令人恐惧的恶魔 | 恶名≥80 | NPC畏惧但敌视 |

**头衔叠加与冲突**：
```
可叠加：
- 身份头衔 + 技能头衔 + 成就头衔
  例："边境守备" + "剑术大师" + "万人敌"

冲突：
- 同一类型的头衔通常互斥（如不能同时是两个不同阵营的军官）
- 某些头衔组合会产生特殊效果
  例："村长" + "神匠" → 村庄可建造高级铁匠铺

特殊组合：
- "商业巨擘" + "令人恐惧的恶魔" → "黑市霸主"（隐藏头衔）
- "剑术大师" + "受人尊敬的英雄" → "骑士典范"（隐藏头衔）
```

#### 4.3.4 等级、技能、身份、头衔的关联

```
【四者关系图】

等级 ─────┬───── 技能
（积累）  │    （能力）
          │
          ▼
        身份
      （社会位置）
          │
          ▼
        头衔
      （社会认可）

【关联规则】

1. 等级 → 技能
   - 等级提供"技能学习上限"
   - 例：等级30最多学习技能到L4
   - 突破需要等级先提升

2. 技能 → 身份
   - 技能水平决定能获得什么身份
   - 例：战斗技巧L4 + 军事指挥L2 → 可申请"队长"
   - 身份申请需要对应技能证明

3. 身份 → 头衔
   - 身份自动带来对应头衔
   - 例："队长"身份 → "边境守备"头衔

4. 头衔 → 等级
   - 获得头衔带来等级经验
   - 例：获得"队长"头衔 = +200经验

5. 技能 → 头衔（直接）
   - 技能达到L6自动获得技能头衔
   - 无需身份中介

6. 等级 → 身份（间接）
   - 高等级更容易获得高级身份
   - 等级是身份申请的隐性条件
```

**实际案例**：
```
玩家发展路径示例：

起点：1级村民，无技能，无身份，无头衔

阶段1（1-20级）：
  - 学习战斗技巧到L2（+20经验）→ 升到5级
  - 完成剿匪任务（+30经验）→ 升到8级
  - 学习战斗技巧到L3（+30经验）→ 升到12级
  - 被军队招募为"士兵"（+100经验）→ 升到15级
  - 获得身份：士兵，头衔：帝国新兵

阶段2（21-40级）：
  - 参与战役并获胜（+200经验）→ 升到25级
  - 学习军事指挥到L2（+20经验）→ 升到28级
  - 被提拔为"队长"（+200经验）→ 升到32级
  - 获得身份：队长，头衔：边境守备

阶段3（41-60级）：
  - 技能提升 + 战役参与 → 升到50级
  - 被任命为"将军"（+500经验）→ 升到55级
  - 战斗技巧达到L6 → 获得头衔：剑术大师
  - 最终：55级，技能L6，身份将军，头衔"边境守备"+"剑术大师"
```

### 4.5 职业体系（技能决定职业，无预设路径）

**核心原则**：
- **没有"选择职业"这个环节**——职业是技能学习的自然结果
- **没有职业限制**——你可以同时学战斗和经商，成为"武装商人"
- **职业随时间变化**——今天的职业不是永远的标签
- **NPC和玩家一样**——都从普通人开始，通过技能获得身份

#### 职业是如何形成的

```
初始状态：普通老百姓（无任何职业标签）
    ↓
学习技能 → 技能组合 → 社会认可 → 获得身份
    ↓
职业不是"选择的"，而是"成为的"
```

**示例**：
```
玩家起点：边境村民（无职业）

路径A：想学战斗
  → 向铁匠学习基础格斗（战斗技巧L1）
  → 参与剿匪获得实战经验（战斗技巧L2）
  → 被军队招募（获得"士兵"身份）
  → 学习军事指挥（军事指挥L1）
  → 带兵打胜仗（晋升"队长"）
  → ...最终可能成为"将军"

路径B：想学经商
  → 从摆摊卖杂货开始（商业贸易L1）
  → 攒钱开小店（商业贸易L2）
  → 学习产业管理（产业管理L1）
  → 雇佣NPC帮工（获得"店主"身份）
  → 扩张多店经营（产业管理L2）
  → ...最终可能成为"商会会长"

路径C：混合发展
  → 同时学战斗和经商
  → 建立武装商队（战斗技巧L3 + 商业贸易L3）
  → 获得"武装商人"身份
  → 这个身份是独特的，不是预设的职业选项
```

#### 身份与职业的区别

| 概念 | 定义 | 特点 |
|------|------|------|
| **技能** | 你掌握的能力 | 可学习、可遗忘、可多线并行 |
| **职业** | 社会对你的功能定位 | 由技能组合自然形成 |
| **身份** | 你在社会结构中的位置 | 由效忠关系+地位决定 |
| **头衔** | 荣誉称号 | 由成就获得，可能有实际权力 |

**示例**：
```
铁壁：
  - 技能：战斗技巧L5、军事指挥L4
  - 职业：军事指挥官（由技能形成）
  - 身份：苍龙帝国北境将军（由效忠关系+地位决定）
  - 头衔："铁壁将军"（荣誉称号）

玩家可能的状态：
  - 技能：商业贸易L3、战斗技巧L2
  - 职业：武装商人（自创职业，非预设）
  - 身份：边境联盟普通居民 → 后来效忠金雀花成为"商会代表"
  - 头衔："丝绸之路守护者"（完成护送商队任务获得）
```

#### NPC也是技能驱动

所有NPC都从普通人开始，他们的"职业"也是技能的结果：

| NPC | 起点 | 技能组合 | 当前身份 |
|-----|------|----------|----------|
| 秋实 | 科举出身的普通书生 | 政治操控L6、情报分析L3 | 苍龙首辅 |
| 破军 | 部落普通战士 | 战斗技巧L5、军事指挥L6 | 苍龙二皇子兼统帅 |
| 金潮 | 普通银行学徒 | 产业管理L6、商业贸易L4 | 金雀花议长 |
| 苍狼 | 普通牧民儿子 | 军事指挥L5、产业管理L3、政治操控L2 | 霜狼首领 |
| 老根 | 普通村民 | 情报分析L2、商业贸易L2、生存L3 | 暮光村村长 |
| 铁匠·老锤 | 普通铁匠学徒 | 锻造L3、商业L1 | 暮光村铁匠 |

**关键洞察**：NPC的"职位"不是天生的，是他们的技能+机遇+选择的结果。玩家走同样的路，也能达到同样的位置。

#### 创新认证体系

**核心理念**：预设的技能树不是世界知识的全部——它只是"已被公认的部分"。角色（玩家或NPC）可以通过组合已有技能、在特定条件下尝试、被关键人物见证并认可，创造出全新的技术和职业。创新是世界进化的引擎。

##### 创新的三层结构

```
┌─────────────────────────────────────────────────────────┐
│  第三层：新职业                                          │
│  一种被公认的新社会角色，有明确的职责和地位                │
│  例："魔纹铁匠"——专门制造魔法武器的职业                  │
│  认证条件：需要权威机构（学院/行会/朝廷）正式承认         │
├─────────────────────────────────────────────────────────┤
│  第二层：新技术                                          │
│  一种前所未有的技能组合方式，可被他人学习                  │
│  例："五行附体"——将五行元素附着在自身而非武器上的技巧     │
│  认证条件：需要在关键场合展示并被专家认可                 │
├─────────────────────────────────────────────────────────┤
│  第一层：新尝试                                          │
│  一种个人的实验性组合，尚未被验证                         │
│  例：你尝试将情报分析的技巧应用到商业谈判中               │
│  结果：可能成功（解锁新技术），可能失败（浪费时间/信誉）  │
└─────────────────────────────────────────────────────────┘
```

##### 创新条件：天时 · 地利 · 人和

创新不是随便组合就能成功的，需要三个条件同时满足：

**天时——时机成熟**
```
- 世界状态满足特定条件（如：某项技术发展到瓶颈、战争催生需求）
- 角色自身状态满足（技能等级、属性达标、身体/精神状态良好）
- 概率因素（即使条件满足，也不是100%成功）

示例：
  你在和平年代尝试研究新的军事战术 → 成功率低（没有需求驱动）
  你在战争年代尝试研究新的军事战术 → 成功率高（天时已到）
```

**地利——环境契合**
```
- 需要在合适的环境中尝试（魔法实验需要魔力充沛的地方）
- 需要合适的材料和工具
- 某些创新只在特定阵营/地区可行

示例：
  在天都尝试政治操控创新 → 有官僚体系支持，成功率+20%
  在边境尝试军事指挥创新 → 实战环境丰富，成功率+15%
  在金雀花尝试商业贸易创新 → 有金融系统支持，成功率+15%
```

**人和——见证与认可**
```
这是最关键的条件：你的创新必须被"对的人"看到并认可。

见证者等级：
  - 普通人见证 → 只能传播小道消息，无法认证
  - 同行专家见证 → 可以验证技术真实性，但影响力有限
  - 权威人物见证 → 可以正式认证，让新技术被世界承认
  - 多位权威同时见证 → 高级认证，新技术立刻获得广泛认可

认可方式：
  - 公开演示：在众人面前成功施展，赢得喝彩
  - 对决验证：在战斗/比赛中使用，击败对手
  - 机构审查：提交给学院/行会/朝廷审查
  - 门派传承：被大师收为弟子，师徒共同完善

没有被见证的创新：
  - 你可能成功创造了新技术
  - 但世界不会知道——它只是你个人的秘密
  - 除非你找到见证者，或者你的弟子在公开场合使用
```

##### 认证流程

```
第一阶段：尝试（个人行为）
  角色选择2-3个已有技能进行组合尝试
  系统判定：天时 + 地利 + 角色属性 → 成功/失败/部分成功
  结果：
    完全失败 → 浪费材料/时间，可能受伤
    部分成功 → 获得不完整的新技术，需要继续完善
    完全成功 → 获得新技术（但尚未认证）

第二阶段：验证（需要见证）
  角色需要在见证者面前成功施展新技术
  见证者类型决定认证级别：
    ├─ 普通场景（如在酒馆里展示）→ 仅限口耳相传
    ├─ 公开场合（如在广场/擂台展示）→ 区域性认可
    ├─ 权威场合（如在学院/朝廷展示）→ 国家级认可
    └─ 跨阵营场合（如在三国共同的场合展示）→ 大陆级认可

第三阶段：传播（新技术扩散）
  认证后的技术可以被他人学习
  学习方式：
    ├─ 拜师学艺（向创造者直接学习，最快）
    ├─ 文献研究（如果有被记录下来，较慢）
    └─ 独立重现（其他人在知道原理后自行尝试，最慢）
```

##### 创新示例

**示例1：玩家创造新技术**
```
场景：你是一名边境指挥官（武力：军事指挥L4 + 经营：产业管理L3）
你尝试用商业物资调配的逻辑来优化军队后勤补给

尝试条件：
  天时：战争期间，补给紧张 ✓
  地利：在边境地区，后勤压力大 ✓
  角色：军事指挥L4 + 产业管理L3，属性达标 ✓

系统判定：成功率65% → 成功！

结果：你创造了"军商统筹术"——用商业贸易的方式管理军队后勤，
大幅提高补给效率。但这只是"未认证"状态——只有你自己能用。

后续：你需要找一个见证者来认证。
  - 在军队内部展示 → 口耳相传，边境驻军知道了
  - 在军演中展示给铁壁将军看 → 将军认可，成为边境驻军可学习的技能
  - 提交给兵部审查 → 如果通过，成为帝国军队标准后勤技能
```

**示例2（MVP后扩展）：NPC创造新职业**
```
场景：当MVP后引入魔法和锻造系统时
炼金堡的星屑（魔法L5）+ 契语的弟子灵约（契约术L3）

星屑和灵约合作研究，尝试将炼金术和契约术融合
经过多次失败后，他们创造了"契约炼成"

认证过程：
  1. 在炼金堡公开展示 → 炼金术士们震惊
  2. 契语（首席契约法师）亲自验证 → 认可技术真实性
  3. 金潮（议长）听闻后投资推广 → "契约炼成师"成为金雀花的新职业

影响：
  - 新职业"契约炼成师"出现在技能树中
  - 这个职业以前不存在——是被创造出来的
```

**示例3：未被见证的创新**
```
场景：荒原狼在荒野中独自作战时，无意中将战斗技巧和商业直觉融合
他创造出了一种"掠夺式贸易"——先武力威慑，再低价收购

但荒原狼没有找见证者，这件事只有他自己知道
"掠夺式贸易"成为他的个人秘密技能

后来：他的副手·独眼在一次行动中看到他使用了这个技巧
如果独眼学会并公开使用 → 这个技术才开始传播
如果独眼也保持沉默 → 这个技术仍然是秘密

极端情况：如果荒原狼突然死亡且没有传承 → 这个技术随他一起消失
```

##### 创新对世界的影响

```
技术创新 → 改变战争形态
  "军商统筹术"的出现 → 后勤效率成为战场胜负关键 → 推动军事与经济的深度融合

职业创新 → 改变社会结构
  "商业政治家"的出现 → 能用经济手段影响政治 → 金雀花的政商融合加深

失传技术 → 世界退步
  某位大师去世且没有传人 → 他的独门技术消失 → 世界失去一种可能性

技术扩散 → 阵营平衡变化
  苍龙的技术被霜狼学会 → 霜狼军事实力提升 → 三国平衡被打破
```

### 4.6 效忠与背叛系统（通用——NPC也会效忠和背叛）

这套系统对所有角色生效。NPC也会做出效忠和背叛的决定，并产生深远后果。

```
效忠流程：

  [角色当前状态] ──→ 可以向任意一国提出效忠
        │
        ▼
  [效忠条件审查] ──→ 该国是否需要你？你的价值是什么？
        │
        ▼
  [效忠代价] ──→ 需要完成效忠任务（可能是道德困境）
        │
        ▼
  [效忠获益] ──→ 获得该国资源、军事、科技支持
        │
        ▼
  [效忠约束] ──→ 必须执行该国的命令（即使你不想）
        │
        ▼
  [背叛条件] ──→ 随时可以背叛，但代价巨大
        │
        ▼
  [背叛后果] ──→ 被原阵营追杀，新阵营可能不信任你
                  所有角色记住这次背叛行为
                  可能获得"叛徒"声望标签
```

**效忠层次**：
```
疏离 → 中立 → 友好 → 效忠 → 核心 → 传奇

疏离：被视为外人
中立：正常贸易
友好：可以获得任务和优惠
效忠：获得正式身份和资源支持
核心：参与国家高层决策
传奇：成为国家历史的一部分
```

**背叛的类型和后果**（通用）：

| 背叛类型 | 后果 | 可恢复性 |
|----------|------|----------|
| 暗中转投 | 原阵营关系-50，新阵营+30但怀疑你 | 可通过长期忠诚修复 |
| 公开反叛 | 原阵营关系-100，被视为死敌 | 几乎不可逆 |
| 自立为王 | 所有阵营关系-30，被视为威胁 | 取决于实力 |
| 回归中立 | 脱离所有阵营，损失所有特权 | 可重新开始 |

**NPC的效忠/背叛**：
- 血狼·屠灭（霜狼强硬派）可能在某个事件中公开反叛苍狼
- 天枢如果被逼到绝境，可能暗中联络金雀花
- 织影（苍龙情报头子）可能同时为多方效力（双面间谍）
- 这些NPC的效忠/背叛由国家Agent和NPC Agent根据世界状态动态决定

### 4.7 成长的影响力（通用）

任何角色变强了，世界都知道。

| 成长阶段 | 世界如何响应 |
|----------|-------------|
| 无名之辈 | 无人关注 |
| 小有名气 | 同行开始注意，有机会接触更高层 |
| 一方豪强 | 国家层面开始关注，拉拢或警惕 |
| 举足轻重 | 重大决策需要考虑你的立场 |
| 影响格局 | 你的选择能改变一场战争的走向 |
| 传奇人物 | 你是历史的书写者 |

**反制机制**（由Chronos主脑Agent控制——对所有强大角色生效）：
- 某角色过于强大 → 其他势力会联合压制
- 某角色过于弱小 → 会有机遇出现
- 某阵营过于偏激 → 会遭遇系统性阻力

## 五、玩家体验层

> 以下是玩家视角独有的设计，不属于世界通用规则，而是"人类操控一个角色"时的体验设计。

### 5.1 日常体验循环

**短期循环**（每次登录5-10分钟）：
```
登录 → 读世界晨报（新鲜感/信息获取）
     → 收获定时奖励（领地产出/任务完成）
     → 做2-3个关键决策（掌控感）
     → 查看之前的决策后果（因果满足）
     → 离开 → 世界继续运转
```

**中期循环**（每周1-2次深度游玩30分钟）：
```
进入 → 查看一周世界大事
     → 规划领地建设（战略决策）
     → 推进外交行动（关系经营）
     → 完成一个事件链（叙事体验）
     → 感受到自己的影响在扩大
```

**长期循环**（每现实月一个"赛季"）：
```
赛季目标 → 完成赛季任务获得赛季奖励
         → 赛季排行榜
         → 赛季大事件（影响世界格局）
         → 新赛季开始
```

### 5.2 心流触发器

| 触发器 | 机制 | 效果 |
|--------|------|------|
| 因果满足 | 3天前的决策今天产生结果 | "我的选择真的有影响！" |
| 稀缺信息 | 情报只有特定时间/条件能获取 | "错过了就没了" |
| 社交压力 | NPC关系在变化，不维护会恶化 | "我得去看看老朋友" |
| 渐进解锁 | 每个阶段都有新内容 | "下一步会有什么？" |
| 身份认同 | 身份和头衔在变化 | "我是这个世界的一部分" |
| 悬念驱动 | 事件链中断，等待下一章 | "接下来会怎样？" |
| 荣誉竞争 | 赛季排行/成就/称号 | "我要比别人更强" |

---

### 5.3 玩家视角的特殊性

**玩家能看到而NPC看不到的**：
- 世界晨报（全大陆视角的新闻汇总）
- 赛季目标和排行榜
- 技能树全貌（NPC只知道自己的技能，不知道完整的技能树结构）
- 声望数值（NPC只能感知大致态度，不能看到精确数字）

**玩家做得到而NPC做不到的**：
- 存档/读档（NPC的世界是不可逆的）
- 选择阵营起点（NPC的出身是世界决定的）
- 随时切换操控方向（NPC被自己的性格和目标绑定）

**玩家做不到而NPC做得到的**：
- 玩家不能同时操控多个角色
- 玩家不能7x24小时在线（NPC在Agent调度下持续运转）
- 玩家需要通过界面交互（NPC之间可以直接通信）

这种不对称正是游戏乐趣的来源：你和NPC在同一个世界、同一套规则下竞争，但你有独特的优势（全知视角、灵活选择）和独特的劣势（时间有限、只能操控一人）。
