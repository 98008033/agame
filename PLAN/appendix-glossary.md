# Agame 名词表

## 附录：Agame 名词表

| 名词 | 含义 |
|------|------|
| **Agame** | 游戏名称（暂缺中文名） |
| **Chronos** | 世界级Agent，命运织网者 |
| **天命司** | 苍龙帝国国家级Agent |
| **先祖议会** | 霜狼联邦国家级Agent |
| **黄金议会** | 金雀花王国国家级Agent |
| **谷主意志** | 边境联盟集体Agent |
| **埃拉西亚大陆** | 游戏世界大陆名称 |
| **灰烬谷** | 三国交界争议地带，边境联盟所在地 |
| **Agent** | AI驱动的自主决策实体 |
| **Template NPC** | 模板化NPC，使用技能+对话模板 |
| **状态差分** | 两次Agent唤醒之间的状态变化量 |
| **平衡指数** | 四阵营势力的相对平衡度 |
| **效忠系统** | 玩家与阵营的关系绑定机制 |
| **十维属性** | 体魄、敏捷、智慧、意志、感知、魅力、名望、恶名、幸运、阵营声望 |
| **技能线** | 谋略线、武力线、经营线（MVP） |
| **技能等级** | 1-6级（MVP），入门→精通→大师→传奇 |
| **身份头衔** | 由效忠任命获得的头衔 |
| **技能头衔** | 由技能等级获得的头衔（如"剑术大师"） |
| **成就头衔** | 由特定成就获得的头衔（如"万人敌"） |
| **声望头衔** | 由名望/恶名获得的头衔 |
| **金币** | 通用货币资源 |
| **影响力** | 社交资本资源 |
| **阵营声望** | 对各阵营的态度值（-100~+100） |
| **名望** | 正面声誉（0-100） |
| **恶名** | 负面威慑（0-100） |
| **晨报** | 每日生成的世界新闻摘要 |
| **事件链** | 多章节连续叙事，决策影响走向 |
| **技能挂载** | Template NPC的技能装配机制 |
| **职业形成** | 技能组合自然形成职业，非预设选择 |
