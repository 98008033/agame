# Agame 技能系统设计（MVP版）

> **核心理念**：技能不是数值，是"能参与的故事类型"。你的技能决定了你能看到什么样的世界。

---

## 一、技能系统概述

### 1.1 设计原则

```
传统游戏：技能 = 数值加成（攻击力+10%）
Agame：技能 = 叙事钥匙（解锁新的故事分支）

武力10的玩家和武力2的玩家面对同一个冲突：
- 武力10：看到"以武服人"的选项 → 触发战斗叙事
- 武力2：看到"寻求和解"的选项 → 触发外交叙事
```

### 1.2 三条主线

| 技能线 | 核心能力 | 解锁故事类型 | 代表人物 |
|-------|---------|-------------|---------|
| **谋略** | 政治、外交、情报 | 权谋斗争、秘密交易、阵营博弈 | 军师、谋士、外交官、间谍 |
| **武力** | 战斗、军事、训练 | 战场厮杀、比武决斗、镇压叛乱 | 将军、武士、雇佣兵、保镖 |
| **经营** | 商业、农业、工艺 | 贸易往来、建设发展、技术创新 | 商人、工匠、农场主、矿主 |

### 1.3 技能等级定义

| 等级 | 称号 | 能力范围 | 叙事影响 |
|-----|------|---------|---------|
| **1-3级** | 入门 | 可以参与相关事件 | 作为旁观者或助手参与 |
| **4-6级** | 精通 | 可以主导相关事件 | 作为主角推动事件发展 |
| **7-9级** | 大师 | 可以改变事件走向 | 创造新的解决路径 |
| **10级** | 传奇 | 定义该领域规则 | 成为历史记载的人物 |

## 二、谋略线设计

### 2.1 技能层级

```
谋略线
├── L1 察言观色
│   └── 能感知他人情绪和立场
├── L2 情报搜集
│   └── 能获得隐藏的线索和信息
├── L3 谈判技巧
│   └── 能在对话中获得优势
├── L4 政治操控
│   └── 能影响他人决策和立场
├── L5 战略规划
│   └── 能设计复杂的长期计划
├── L6 阴谋布局
│   └── 能暗中操控多方势力
├── L7 舆论控制
│   └── 能塑造公众认知和叙事
├── L8 制度设计
│   └── 能建立持久的权力结构
└── L9+ 历史书写
    └── 你的名字出现在史书谋略篇
```

### 2.2 技能解锁的事件类型

| 等级 | 事件类型 | 示例 |
|-----|---------|------|
| L1-L3 | **信息收集类** | 打探消息、旁听会议、阅读密信 |
| L4-L6 | **谈判协商类** | 外交谈判、利益交换、调解冲突 |
| L7-L9 | **权力斗争类** | 策划政变、扶持傀儡、建立派系 |
| L10 | **历史转折类** | 改变国家命运、书写新的政治规则 |

### 2.3 叙事分支示例

**场景**：两国边境冲突

```
谋略等级低的玩家看到：
"边境传来消息，苍龙和霜狼的军队在荒野对峙。"
[前往观望] [询问详情] [保持关注]

谋略等级高的玩家看到：
"边境传来消息...（阴谋嗅觉触发）
你发现这次冲突的时机过于巧合——
刚好在苍龙首辅病逝、权力真空之际。
深入调查可能揭露第三方势力的挑拨。"
[调查幕后黑手] [利用混乱谋取利益] [警告双方] [保持观望]
```

### 5.1 跨线技能组合

| 组合 | 称号 | 特殊能力 | 示例场景 |
|-----|------|---------|---------|
| 谋略L5 + 武力L3 | **统帅** | 能用战术弥补兵力劣势 | 以少胜多的战役 |
| 谋略L3 + 经营L5 | **商业政治家** | 能用经济手段达到政治目的 | 贸易战、经济制裁 |

| 三线均L4+ | **全才** | 能从任意角度解决问题 | 看到所有叙事分支选项 |

## 六、技能获取途径

### 6.1 主要获取方式

| 方式 | 说明 | 示例 | 效率 |
|-----|------|------|------|
| **事件实践** | 参与相关事件 | 参与战斗提升武力，参与谈判提升谋略 | ★★★★★ |
| **NPC传授** | 向有技能的NPC学习 | 拜师学艺，需要好感度达标 | ★★★★☆ |
| **自我修炼** | 消耗时间和资源 | 闭关修炼、阅读典籍、实战训练 | ★★★☆☆ |
| **特殊际遇** | 奇遇或重大转折 | 生死关头的顿悟、神秘人物的指点 | ★★★★★ |

### 6.2 技能获取的具体规则

**事件实践**
```
- 每次参与相关事件，根据表现获得经验
- 成功完成高难度事件：+大量经验
- 失败但有所收获：+少量经验
- 旁观他人操作：+微量经验

示例：
"你参与了一场边境谈判（谋略事件）
表现评级：出色
获得谋略经验：+50点"
```

**NPC传授**
```
- 需要与目标NPC建立足够的关系
- NPC的技能等级决定了能教给你的上限
- 每次学习消耗时间和金钱
- 某些技能需要特定的NPC才能传授（如阵营独有技能）

示例：
"秋实同意传授你政治操控技巧
前提条件：苍龙声望≥友好，已学习情报搜集
预计学习时间：3天
学习费用：100金币"
```

**自我修炼**
```
- 消耗时间和资源进行修炼
- 需要合适的地点（如图书馆、训练场）
- 效率较低，但不受他人限制
- 可能走火入魔（失败惩罚）

示例：
"你选择在铁壁关进行闭关训练
预计时间：7天
每日消耗：食物×3，住宿费用×7
成功率：85%（武力L3→L4）"
```

### 6.3 技能提升的叙事包装

技能提升不只是数字变化，而是有具体的故事：

```
【武力提升示例】
你的武力从3级提升到4级！

【系统提示】
经过这段时间的历练，你的剑术终于突破了瓶颈。
昨天与铁壁关守将的切磋中，你领悟了"以静制动"的奥妙。
现在，你可以挑战更高难度的战斗事件了。

【解锁内容】
✓ 新事件类型：单挑决斗
✓ 新叙事选项：在冲突中"以武服人"
✓ 新NPC互动：某些武者愿意与你切磋
```

## 七、MVP简化版技能系统

### 7.1 核心设计

MVP阶段只保留**最核心的7个技能**，确保每条线都有代表性：

```
【谋略线】
├── 情报分析（1-3级）- 发现隐藏的叙事线索
└── 政治操控（4-6级）- 解锁高级权谋事件

【武力线】
├── 战斗技巧（1-3级）- 参与个人和小规模战斗
└── 军事指挥（4-6级）- 解锁战役级军事事件

【经营线】
├── 商业贸易（1-3级）- 进行基础买卖和贸易
└── 产业管理（4-6级）- 解锁大规模经济活动

【通用技能】
└── 生存能力（1-3级）- 基础生存，所有玩家初始拥有
```

### 7.2 MVP技能与事件对应表

| 技能 | 等级 | 解锁事件示例 |
|-----|------|-------------|
| **生存能力** | 1级 | 野外生存、基础交流、简单工作 |
| **情报分析** | 1级 | 旁听对话、发现异常、收集流言 |
| **情报分析** | 2级 | 解读密信、识别谎言、追踪线索 |
| **情报分析** | 3级 | 建立情报网、预测事件、发现阴谋 |
| **政治操控** | 4级 | 参与高层会议、影响决策、建立派系 |
| **政治操控** | 5级 | 策划政变、扶持代理人、操控舆论 |
| **政治操控** | 6级 | 制定国家政策、改变政治格局 |
| **战斗技巧** | 1级 | 街头格斗、护送任务、击退野兽 |
| **战斗技巧** | 2级 | 武器对决、小型战斗、保镖任务 |
| **战斗技巧** | 3级 | 精英对决、带队剿匪、竞技场挑战 |

### 7.3 MVP技能获取简化

| 技能 | 初始获取 | 提升方式 |
|-----|---------|---------|
| 生存能力 | 所有玩家初始1级 | 完成生存相关事件 |

| 产业管理 | 需商业贸易3级后解锁 | 参与产业事件，NPC传授 |

### 8.1 数据结构

```typescript
// 技能定义
interface Skill {
  id: string;
  name: string;
  category: 'strategy' | 'combat' | 'business';
  maxLevel: number;
  prerequisites?: string[];
  unlocksNarrative: string[];  // 解锁的叙事分支ID
}

// 玩家技能数据
interface PlayerSkill {
  skillId: string;
  level: number;
  experience: number;
  unlockedAt: Date;
}

// 叙事分支条件
interface NarrativeBranch {
  id: string;
  description: string;
  requiredSkills: {
    skillId: string;
    minLevel: number;
  }[];
  exclusiveWith?: string[];  // 互斥选项
}
```

### 8.2 技能检查逻辑

```typescript
// 检查玩家能否看到某个叙事选项
function canAccessNarrativeBranch(
  playerSkills: PlayerSkill[],
  branch: NarrativeBranch
): boolean {
  return branch.requiredSkills.every(req => {
    const playerSkill = playerSkills.find(s => s.skillId === req.skillId);
    return playerSkill && playerSkill.level >= req.minLevel;
  });
}

// 获取事件的所有可用选项
function getAvailableOptions(
  event: GameEvent,
  playerSkills: PlayerSkill[]
): NarrativeBranch[] {
  return event.branches.filter(branch =>
    canAccessNarrativeBranch(playerSkills, branch)
  );
}
```

### 8.3 技能提升触发

```typescript
// 事件完成后计算技能经验
function calculateSkillExp(
  event: GameEvent,
  playerChoices: Choice[],
  outcome: EventOutcome
): SkillReward[] {
  const rewards: SkillReward[] = [];

  // 根据参与的事件类型给予经验
  if (event.type === 'combat') {
    rewards.push({
      skillId: 'combat_technique',
      experience: outcome.success ? 50 : 20
    });
  }

  if (event.type === 'negotiation') {
    rewards.push({
      skillId: 'intelligence_analysis',
      experience: outcome.success ? 40 : 15
    });
  }

  // 根据表现评级加成
  if (outcome.rating === 'excellent') {
    rewards.forEach(r => r.experience *= 1.5);
  }

  return rewards;
}
```

## 九、总结

### 9.1 MVP核心要点

1. **3条线 × 2个技能 = 6个专业技能 + 1个通用技能**
2. **技能等级最高6级**（入门1-3级，精通4-6级），简化成长曲线
3. **技能直接决定可见的叙事分支**，而非提供数值加成
4. **技能获取主要通过参与事件**，符合游戏"叙事优先"的核心理念

### 9.2 与其他系统的关联

```
技能系统 ←→ 事件系统：技能决定能看到的事件选项
技能系统 ←→ NPC系统：高技能等级吸引相关NPC关注
技能系统 ←→ 阵营系统：某些高级技能需要阵营声望
技能系统 ←→ 叙事系统：技能是解锁叙事分支的钥匙
```

### 9.3 后续扩展方向

- **大师级技能**（7-9级）：MVP后逐步开放
- **传奇级**（10级）：赛季制，每个赛季只有极少数玩家能达到
- **技能组合**：开发更多跨线技能组合的独特叙事
- **阵营独有技能**：每个阵营的专属高级技能
