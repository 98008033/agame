# Agame 风险与奖励机制设计

> **核心理念**：选择有意义吗？答案是——每一个选择都在改变你的故事。失败的代价不只是数值下降，而是故事的转折；成功的奖励不只是声望+10，而是新世界的开启。

---

## 一、 stakes 的本质

### 1.1 什么是"有意义的选择"？

```
传统游戏：
  选择 → 数值变化 → 继续游戏
  失败？重新加载就好。

Agame的设计：
  选择 → 故事转折 → 新的故事分支
  失败？这是你故事的一部分，接受它或培养继承人。

有意义的选择 = 有后果 + 不可撤销 + 叙事呈现
```

### 1.2 Stakes的三个维度

| 维度 | 说明 | 示例 |
|------|------|------|
| **即时代价** | 选择当场的损失 | 金币消耗、关系下降、身体受伤 |
| **连锁后果** | 选择引发的未来事件 | 触发追杀、失去机会、引发战争 |
| **叙事转变** | 选择改变的故事走向 | 成为叛徒、错过爱情、改变历史 |

### 1.3 风险的三种层级

```
┌─────────────────────────────────────────────────────────────────┐
│                    风险层级金字塔                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│    ┌────────────────────────────────────────────────────────┐  │
│    │            不可挽回的风险 (5%)                           │  │
│    │  永久死亡、阵营覆灭、重要NPC死亡                          │  │
│    │  → 改变世界格局，无法逆转                                │  │
│    └────────────────────────────────────────────────────────┘  │
│                                                                  │
│    ┌────────────────────────────────────────────────────────┐  │
│    │            深远影响的风险 (15%)                          │  │
│    │  关系断裂、声望崩塌、领地丧失                             │  │
│    │  → 需数月/数年恢复，或培养继承人重新开始                   │  │
│    └────────────────────────────────────────────────────────┘  │
│                                                                  │
│    ┌────────────────────────────────────────────────────────┐  │
│    │            可恢复的风险 (80%)                            │  │
│    │  金币损失、关系下降、短期状态                             │  │
│    │  → 通过后续行动可以恢复                                  │  │
│    └────────────────────────────────────────────────────────┘  │
│                                                                  │
│    日常选择：80%可恢复风险 + 15%深远风险 + 5%不可挽回            │
│    重大决策：风险层级上移，不可挽回比例增加                       │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## 二、失败风险设计

### 2.1 叙事层面的惩罚

**超越数值的设计**：

```
传统失败：
  "战斗失败，金币-100"
  → 数值变化，故事继续

叙事失败：
  "战斗失败，你被俘虏了"
  → 不是数值变化，而是状态改变
  → 触发"俘虏"事件链
  → 你的故事从"领主"变成"囚犯"
  → 等待救援或策划逃脱
```

### 2.2 四类叙事惩罚

| 惩罚类型 | 说明 | 示例 |
|----------|------|------|
| **失去关系** | 与重要NPC关系破裂 | 老根不再信任你、烈焰成为仇敌 |
| **错过机会** | 永久失去某个选择分支 | 错过与天枢结盟的机会窗口 |
| **引发灾难** | 你的选择导致更大事件 | 选择触发边境战争、瘟疫蔓延 |
| **身份转变** | 你的社会身份改变 | 从领主变成流亡者、从忠臣变成叛徒 |

### 2.3 关系断裂的叙事后果

```
关系破裂不只是数值-40：

浅层破裂（关系从60降至20）：
├── NPC拒绝与你合作
├── NPC的情报渠道关闭
├── NPC的朋友圈对你态度冷淡
└── 恢复需要：道歉+补偿+时间（至少1个月）

深层破裂（关系从80降至0）：
├── NPC视你为仇敌
├── NPC可能主动报复
├── NPC的阵营对你敌视
├── 晨报中传播你的"背叛"
└── 恢复几乎不可能（除非重大牺牲）

极端破裂（关系从80降至-50）：
├── NPC发布追杀令
├── NPC的继承人继承仇恨
├── 你的声誉在相关区域崩塌
├── 触发"仇敌追杀"事件链
└── 这是永久性的故事转变
```

### 2.4 机会窗口的设计

```
有些选择只在特定时间出现：

机会窗口 = 条件满足 + 时间限制

示例：废太子天枢的庇护机会
├── 触发条件：苍龙夺嫡指数>70
├── 时间窗口：天枢到达灰烬谷后48小时内
├── 玩家选择：
│   ├── 庇护 → 开启"流亡皇子"故事线
│   ├── 不庇护 → 天枢离开，机会窗口关闭
│   └── 勒索 → 天枢被杀，你获得金币但永远错过
└── 错过后：
    ├── 该故事线不会再次触发
    ├── 天枢可能被杀/隐居/建国（你无法影响）
    └── 你的故事中缺少了这个篇章
```

### 2.5 连锁灾难设计

```
选择可能引发蝴蝶效应：

事件链示例：边境纠纷的处理

选择A（强硬驱逐）：
├── 即时：边境村民被赶走
├── 后续（7天后）：村民投奔霜狼
├── 连锁（14天后）：霜狼以此为借口增兵
├── 爆发（30天后）：边境战争爆发
└── 你的领地成为前线

选择B（温和调解）：
├── 即时：纠纷解决，村民满意
├── 后续：村民成为你的支持者
├── 连锁：边境稳定度+20
└── 爆发：无战争

关键：连锁后果不是立刻显示
玩家只看到即时结果，后续在晨报中逐渐显现
"原来当初那个选择导致了..."
```

### 2.6 可挽回 vs 不可挽回

```
可挽回的失败：
├── 金币损失 → 通过贸易/任务恢复
├── 关系下降（-20以内） → 通过后续互动恢复
├── 短期状态（受伤、被捕） → 时间/事件后恢复
├── 阵营声望下降 → 通过贡献恢复
└── 恢复路径清晰，代价明确

不可挽回的失败：
├── 重要NPC死亡 → 永久失去该NPC
├── 关系破裂至仇敌 → 关系归零且锁定
├── 选择机会错过 → 该故事线永远关闭
├── 获得永久标签 → "叛徒"标签影响所有后续NPC态度
├── 阵营覆灭 → 世界格局改变
└── 恢复不可能，只能接受或培养继承人重新开始
```

## 三、成功奖励设计

### 3.1 叙事层面的奖励

**超越数值的设计**：

```
传统奖励：
  "任务完成，声望+10，金币+100"
  → 数值增长，故事没有变化

叙事奖励：
  "任务完成，老根把你当家人了"
  → 不只是数值变化
  → 你获得了"家人"身份
  → 老根会在危机中主动保护你
  → 你可以进入老根的私人空间
  → 老根的秘密现在也是你的秘密
```

### 3.2 四类叙事奖励

| 奖励类型 | 说明 | 示例 |
|----------|------|------|
| **解锁故事线** | 获得新的叙事入口 | 可参与苍龙宫廷政治、可加入霜狼战团 |
| **获得NPC深度信任** | 关系不只是数值，是羁绊 | NPC主动帮助、分享秘密、在危机中站出来 |
| **开启新区域** | 获得地理/社会准入 | 可进入禁地、可参与高层会议 |
| **独特叙事体验** | 见证/参与独特事件 | 见证王朝更替、改变某人的命运 |

### 3.3 关系升华的叙事设计

```
关系不只是数值区间，而是叙事阶段：

陌生人（关系0-20）：
├── 交易关系
├── 公事公办
└── 不愿意透露私人信息

熟人（关系20-40）：
├── 可以合作
├── 基本信任
├── 愿意分享一般情报
└── 可能接受你的请求

朋友（关系40-60）：
├── 主动帮助（小事情）
├── 分享重要情报
├── 在你困难时伸出援手
└── 允许你进入私人空间

挚友（关系60-80）：
├── 主动提供情报
├── 在危机中站出来
├── 分享秘密
├── 愿意为你冒险（有限度）
└── 你成为他人生故事的一部分

生死之交（关系80+）：
├── 无条件信任
├── 愿意为你牺牲
├── 他的家人也是你的家人
├── 他的仇敌也是你的仇敌
└── 这是真正的羁绊，但也带来责任
```

### 3.4 解锁故事线的奖励

```
某些选择解锁特定的故事线：

示例：效忠苍龙帝国后的解锁

解锁的故事线：
├── "宫廷政治"故事线
│   └── 可参与皇子争斗、派系博弈
│   └── 谋略L3+可深度参与
├── "帝国军旅"故事线
│   └── 可加入正规军、参与战役
│   └── 武力L3+可获得指挥权
├── "五行学院"故事线
│   └── 可学习五行魔法
│   └── 需找到导师并通过考验
└── "天枢之友"故事线（特殊情况）
    └── 如果你在天枢流亡时帮助他
    └── 天枢复位后你成为开国功臣

不效忠的选择：
├── 以上故事线都不解锁
├── 你的故事走向完全不同
└── 但你可能解锁其他故事线（独立建国）
```

### 3.5 开启新区域的奖励

```
区域不只是地理概念，还有社会区域：

地理区域解锁：
├── 苍龙天都（效忠后可进入内城）
├── 霜狼圣地（战团成员可进入）
├── 金雀花金库（特许商人可进入）
└── 灰烬谷深处（边境声望≥60解锁）

社会区域解锁：
├── 阵营决策层（阶层≥贵族+阵营效忠）
├── 团体核心圈（团体等级≥核心成员）
├── NPC私人空间（关系≥挚友）
└── 地下情报网（获得"情报人"身份）

解锁后的叙事变化：
├── 可参与该区域的专属事件
├── 可获得该区域的专属资源
├── 可建立该区域的关系网络
└── 晨报中出现该区域的详细报道
```

### 3.6 独特叙事体验的奖励

```
某些选择带来独特体验：

见证历史：
├── 参与王朝更替的时刻
├── 亲眼见证战争爆发
├── 目睹重要NPC的死亡
└── 这些时刻被写入晨报和个人传记

改变命运：
├── 你的选择改变了某NPC的一生
├── 如：帮助铁心加入战团
│   → 她从"被歧视的女战士"变成"战团传奇"
│   → 你的传记中记录："我帮助了铁心..."
├── 如：阻止烈焰的独立计划
│   → 她的命运走向完全不同
│   → 你的选择被她的传记记录
└── 这是你留下的痕迹

创造历史：
├── 你主导了某个重大事件
├── 如：促成边境联盟独立
│   → 世界格局改变
│   → 晨报记录你的名字
│   → 新势力以你为元勋
└── 你成为历史的一部分
```

## 四、风险感知系统

### 4.1 玩家如何知道风险？

```
风险感知的三个来源：

1. 明示风险（系统直接告知）
├── 事件描述中明确说明后果
├── 如："拒绝缴纳，阵营声望-30，可能触发叛国调查"
└── 简单、直接，但不够深入

2. 暗示风险（叙事线索）
├── NPC的警告、环境的暗示
├── 如：老根说"这可能是陷阱，你要小心"
├── 如：天气异常暗示天灾将至
└── 需要玩家自行判断

3. 需探索的风险（隐藏信息）
├── 需要技能/关系才能发现
├── 谋略L3+：你察觉背后有第三方挑拨
├── 与老根关系≥50：老根私下告诉你真实风险
└── 信息获取本身就是一种奖励
```

### 4.2 风险提示的UI设计

```
事件界面的风险呈现：

┌─────────────────────────────────────────────────────────────┐
│ [边境联盟] 边境匪患                                          │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│ 警报！荒原狼匪帮袭击了商队...                                │
│                                                              │
│ [你的选择]                                                   │
│ ◉ 组织防御                                                  │
│   ├─ 风险提示：需要武力≥60，失败可能受伤                     │
│   ├─ 成功奖励：击退匪帮，金币+100，影响力+20                  │
│   └─ 失败后果：受伤状态（3天），金币-50                       │
│                                                              │
│ ◉ 向苍龙求援 [需苍龙效忠或声望≥30]                           │
│   ├─ 风险提示：援军需3天到达，商队可能已损失                  │
│   ├─ 成功：匪帮撤退，但失去独立行动声誉                        │
│   └─ 失败：援军未到或迟到，情况恶化                           │
│                                                              │
│ ◉ 花钱消灾                                                   │
│   ├─ 风险提示：金币-150，被视为软弱                           │
│   ├─ 即时结果：匪帮撤退                                       │
│   ├─ 长期后果：匪帮可能再次勒索                               │
│                                                              │
│ ◉ 暗中勾结 [需谋略L4+]                                       │
│   ├─ 风险提示：获得"通匪"秘密标签                             │
│   ├─ 即时收益：金币+200                                       │
│   ├─ 隐藏风险：被发现则声望-100，触发清算事件                  │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 4.3 技能解锁的风险感知

```
技能等级 = 信息获取能力：

L1-2（基础）：
├── 只能看到明示风险
├── 事件描述中明确写出的后果
└── 适合新手玩家

L3-4（进阶）：
├── 开始看到暗示风险
├── "你察觉..."类型的额外信息
├── 如："你察觉匪帮背后可能有势力支持"
└── 需要玩家自行判断是否重要

L5-6（精通）：
├── 可以分析风险的真实程度
├── "你分析..."类型的深度信息
├── 如："你分析后发现，这笔投资的真实风险是..."
└── 大幅降低错误决策概率

L7-8（大师）：
├── 可以预见连锁后果
├── "你预感..."类型的未来预测
├── 如："你预感这个选择可能导致..."
└── 可以规避蝴蝶效应

L9-10（传奇）：
├── 几乎可以看清所有风险
├── "你洞察..."类型的全息信息
├── 如："你洞察到这是某阵营的精心布局..."
└── 选择变得更有底气
```

### 4.4 关系解锁的风险感知

```
NPC关系 = 信息来源：

关系≥40的NPC：
├── 会给出一般性警告
├── "我听说...你要小心"
├── 可能不准确，但提供方向
└── 需要玩家自行验证

关系≥60的NPC：
├── 会给出具体警告
├── "我知道...真相是这样的"
├── 信息相对可靠
└── 可能要求回报

关系≥80的NPC：
├── 会主动预警
├── "你必须知道...有危险"
├── 信息高度可靠
├── 不要求回报（或帮助本身就是回报）
└── 可能提前提供解决方案

关系-30以下的NPC：
├── 可能给出假信息
├── "我听说...（其实是陷阱）"
├── 需要玩家辨别
└── 这是仇敌的报复方式之一
```

### 4.5 信息获取的成本

``<arg_value>获取风险信息需要付出代价：

时间成本：
├── 调查需要等待（游戏日）
├── 等待期间机会窗口可能关闭
└── 风险 vs 机会的权衡

金钱成本：
├── 购买情报需要金币
├── 高价值情报价格更高
└── 资源有限时的选择

关系成本：
├── 请求NPC预警消耗关系值
├── 过度依赖某NPC会使其疲惫
└── 关系不是无限资源

技能成本：
├── 提升技能需要经验和时间
├── 投资情报技能会牺牲其他发展
└── 发展路线的选择

设计意图：
├── 信息不是免费获取
├── 获取信息本身也是一种选择
├── 有时冒险比等待更明智
└── 让"冒险"成为一种策略，而非惩罚
```

## 五、Stakes平衡机制

### 5.1 风险太大的问题

```
问题：风险太大，玩家不敢选择

症状：
├── 玩家反复犹豫，不敢做决定
├── 玩家放弃高风险选项，只选安全路
├── 玩家抱怨"这游戏太苛刻"
└── 游戏体验变得焦虑而非刺激

解决：分阶段风险暴露

不一次性展示所有风险：
├── 只展示即时风险
├── 连锁风险在后续晨报中暗示
├── 不可挽回风险用"不可逆选择"标签提示
└── 让玩家感受到风险，但不被吓倒

提供退路（可选）：
├── 某些高风险选择提供"退出"机制
├── 如：效忠后发现不合适，可叛逃（代价大）
├── 如：加入后发现危险，可退出团体
└── 不是所有选择都有退路，但关键选择应该有
```

### 5.2 风险太小的问题

```
问题：风险太小，选择无意义

症状：
├── 玩家随手选择，不考虑后果
├── 所有选项看起来差不多
├── 玩家觉得"随便选都行"
└── 游戏体验变得无聊而非策略

解决：让每个选择有独特后果

后果差异化：
├── 每个选项的后果完全不同
├── 不是一个只是数值多一点少一点
├── 而是：选项A→故事线X，选项B→故事线Y
└── 选择改变的是故事走向，不只是数值

叙事反馈：
├── 后果不只是系统消息
├── 而是NPC的反应、晨报的报道
├── "因为你的选择，..."在叙事中呈现
└── 让玩家感受到"我的选择确实重要"
```

### 5.3 Stakes平衡公式

```
风险-奖励匹配原则：

高风险 = 高奖励
├── 叛逃阵营：声望崩塌风险 → 独立建国机会
├── 庇护废太子：被追杀风险 → 开国功臣可能
├── 深入敌营：被发现风险 → 独特情报获取
└── 风险越大，成功后的故事越精彩

中风险 = 中奖励
├── 投资商业：亏本风险 → 稳定收益
├── 帮助NPC：得罪其仇敌 → 获得其信任
├── 参与调解：调解失败 → 调解者声望
└── 日常选择，日常回报

低风险 = 低奖励
├── 小额交易：几乎无风险 → 小额收益
├── 基础任务：任务失败 → 少量经验
├── 保持中立：得罪双方风险低 → 错过双方机会
└── 安全路是平庸路
```

### 5.4 动态Stakes调节

```
Stakes根据玩家状态动态调整：

玩家等级调节：
├── 低等级：风险比例降低，学习阶段
├── 中等级：正常风险比例
├── 高等级：高风险选项增加，传奇体验
└── 让每个阶段的玩家都有合适的挑战

玩家历史调节：
├── 如果玩家多次选择高风险路线 → 高风险选项增加
├── 如果玩家倾向安全路线 → 安全选项更多
├── 系统学习玩家的偏好并调节
└── 让每个玩家的体验独特

叙事节奏调节：
├── 紧张期（战争、政变）：高风险选项增加
├── 平静期：日常选项为主
├── 按世界状态调节stakes强度
└── 让stakes与世界节奏同步
```

## 六、失败与成功的叙事呈现

### 6.1 失败的叙事呈现

```
失败不是"游戏结束"，而是"故事转折"：

失败叙事的三个层次：

即时反馈（当场呈现）：
├── "战斗失败，你倒在地上..."
├── "谈判破裂，对方转身离开..."
├── 不是"任务失败"，而是场景描述
└── 玩家感受到当下的挫折

后续影响（晨报呈现）：
├── "暮光村领主在某事件中失败..."
├── "传闻...因为某人的失败..."
├── 失败被世界记录和传播
└── 玩家感受到"世界在看我"

连锁后果（后续事件呈现）：
├── "因为当初的失败，现在..."
├── "当初那个选择，导致了现在的..."
├── 失败不是孤立事件，是因果链的一部分
└── 玩家感受到"历史在发展"
```

### 6.2 成功的叙事呈现

```
成功不只是"任务完成"，而是"故事推进"：

成功叙事的三个层次：

即时反馈（当场呈现）：
├── "你成功了！老根紧紧握住你的手..."
├── "谈判达成！对方露出笑容..."
├── 不是"任务成功"，而是情感场景
└── 玩家感受到当下的喜悦

后续影响（晨报呈现）：
├── "暮光村领主成功解决了..."
├── "因为某人的成功，边境局势缓和..."
├── 成功被世界记录和传播
└── 玩家感受到"世界认可我"

连锁后果（后续事件呈现）：
├── "因为当初的成功，现在..."
├── "当初那个选择，开启了今天的..."
├── 成功不是孤立事件，是新故事的入口
└── 玩家感受到"选择有回响"
```

### 6.3 个人传记中的记录

```
玩家的选择被记录在传记中：

传记格式：
┌─────────────────────────────────────────────────────────────┐
│ 传记：暮光村领主（玩家）                                      │
│                                                              │
│ 第15日 秋                                                   │
│ 面对边境匪患，选择组织防御，成功击退敌人。                    │
│ 村民们开始信任这位新领主。                                   │
│                                                              │
│ 第23日 冬                                                   │
│ 在流亡皇子事件中，选择庇护天枢。                              │
│ 这个选择开启了"开国功臣"的故事线。                           │
│                                                              │
│ 第67日 春                                                   │
│ 天枢复位成功，玩家被册封为边境伯爵。                          │
│ 传记中记录："当初那个选择，改变了我的命运。"                  │
│                                                              │
│ [继续阅读]  [关键选择回顾]                                    │
└─────────────────────────────────────────────────────────────┘

关键选择回顾：
├── 突出显示最重要的3-5个选择
├── 每个选择显示后果链
├── "如果当初选择不同..."
└── 让玩家反思自己的道路
```

## 七、特殊Stakes设计

### 7.1 时间压力Stakes

```
某些选择有时间限制：

限时选择的设计：
├── 显示倒计时（游戏时）
├── 超时自动选择"观望/不干预"
├── 观望不是安全路，可能错过机会
└── 时间本身就是一种stakes

示例：紧急求援事件
├── 收到铁壁将军求援信
├── 限时：48小时内必须回应
├── 选项：
│   ├── 立即支援（高代价，高风险）
│   ├── 延迟支援（可能铁壁已战败）
│   ├── 拒绝支援（关系崩塌）
│   └── 超时自动拒绝
└── 时间压力增加决策紧张感
```

### 7.2 信息不完整Stakes

```
某些选择信息不完整：

信息缺失的设计：
├── 只展示部分后果
├── 需要玩家判断风险
├── 技能/关系可补充信息
└── 信息不完整本身就是风险

示例：神秘访客事件
├── 访客自称来自某势力
├── 无法验证其真实性
├── 选项：
│   ├── 相信并接受（可能是真的，也可能是陷阱）
│   ├── 怀疑并拒绝（错过机会）
│   ├── 调查后再决定（需时间和资源）
└── 玩家需要权衡信息不完整的风险
```

### 7.3 道德Stakes

```
某些选择涉及道德困境：

道德困境的设计：
├── 选项不只是利益计算
├── 还涉及价值观判断
├── 无"正确答案"，只有"你的选择"
└── 道德选择塑造玩家角色形象

示例：瘟疫处理事件
├── 选项：
│   ├── 全力救治（资源消耗大，道德正确）
│   ├── 隔离疫区（资源消耗中，实用主义）
│   ├── 驱逐病人（资源消耗小，道德争议）
│   ├── 逃离区域（几乎无消耗，道德质疑）
└── 每个选择都有叙事反馈：
    ├── "仁慈领主" → NPC态度+10，后续事件倾向帮助
    ├── "冷血领主" → NPC态度-10，后续事件可能孤立
    └── 道德选择影响你在世界中的形象
```

### 7.4 情感Stakes

```
某些选择涉及情感：

情感困境的设计：
├── 涉及与NPC的羁绊
├── 不是利益问题，是情感问题
├── 情感选择可能改变故事走向
└── 情感后果比数值后果更深刻

示例：老友背叛事件
├── 老友站在敌对阵营
├── 他私下警告你撤离
├── 选项：
│   ├── 相信老友撤离（放弃领地，保住友情）
│   ├── 不相信准备迎敌（保住领地，可能错杀老友）
│   ├── 利用情报反制（保住领地，老友可能被杀）
│   ├── 说服老友倒戈（高风险，可能成功或失败）
└── 情感选择：
    ├── 选择相信 → 老友关系保持，但失去领地利益
    ├── 选择反制 → 老友可能死亡，你的故事失去这个朋友
    └── 情感后果在后续叙事中发酵
```

## 十、总结：Stakes的核心原则

### 10.1 七大核心原则

```
1. 选择有后果原则
   每个选择都有后果，不是数值变化，而是故事转折

2. 后果叙事化原则
   后果不只是系统消息，而是晨报报道、NPC反应、传记记录

3. 风险分层原则
   可恢复风险80%，深远风险15%，不可挽回5%

4. 风险-奖励匹配原则
   高风险=高奖励，低风险=低奖励，安全路是平庸路

5. 信息获取成本原则
   风险信息需要代价获取，获取信息本身也是一种选择

6. 失败是故事原则
   失败不是游戏结束，而是故事转折，接受或培养继承人

7. 成功是新世界原则
   成功不只是数值增长，而是解锁新故事、新区域、新羁绊
```
