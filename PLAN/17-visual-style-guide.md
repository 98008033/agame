## 整体视觉方向

### 设计理念

- **核心风格**：16-bit像素艺术 + 奇幻魔法元素
- **情绪基调**：神秘、冒险、史诗感
- **氛围目标**：复古游戏的亲切感 + 现代游戏的精致度
- **设计哲学**："像素之中，自有天地" —— 用有限的像素传递无限的想象空间

### 视觉关键词

```
像素颗粒 · 魔法光效 · 复古界面 · 奇幻图腾 · 动态精灵
```

- **《最终幻想6》**（像素美术标杆）

## 二、像素美术规范

### 2.1 像素尺寸

**基础单位**：1像素 = 1px

| 元素类型 | 推荐尺寸 | 说明 |
|---------|----------|------|
| **角色精灵** | 32×32 ~ 48×48 | 主角和重要NPC |
| **NPC精灵** | 24×24 ~ 32×32 | 普通NPC |
| **物品图标** | 16×16 ~ 24×24 | 背包物品、资源 |
| **技能图标** | 32×32 | 技能树、快捷栏 |
| **阵营徽章** | 48×48 ~ 64×64 | 标识和Logo |
| **背景图块** | 16×16 ~ 32×32 | 可拼接的地砖 |
| **UI元素** | 8×8的倍数 | 按钮、边框等 |

### 2.2 像素绘制规则

#### 抗锯齿

- **禁用抗锯齿**：保持硬边缘，体现像素感
- **手动过渡**：用中间色调像素手动创建过渡
- **抖动（Dithering）**：允许使用2×2或4×4的抖动图案增加色彩层次

#### 描边规则

```
内描边优先：1像素深色描边，增强轮廓
外发光效果：用于魔法物品和选中状态
避免纯黑描边：使用深色调（如#2A2A3E）代替纯黑
```

#### 阴影处理

- **投影阴影**：角色和物体下方2-3像素的椭圆阴影
- **接触阴影**：物体与地面接触处的深色像素
- **渐变阴影**：用2-3层不同深度的颜色模拟光照

### 2.3 动画规范

| 动画类型 | 帧率 | 帧数 | 说明 |
|---------|------|------|------|
| **待机动画** | 4fps | 2-4帧 | 轻微呼吸感浮动 |
| **行走动画** | 8fps | 4-6帧 | 经典2帧或4帧行走 |
| **攻击动画** | 12fps | 4-8帧 | 快速有力的动作 |
| **魔法施放** | 8fps | 6-10帧 | 配合粒子光效 |
| **UI动效** | 12fps | 3-6帧 | 按钮hover、切换 |
| **环境动画** | 4fps | 2-4帧 | 旗帜、火焰、水流 |

## 三、字体系统

### 3.1 字体选择

#### 标题字体

- **中文**：**像素字体**（如：Zpix、方正像素12）
- **英文**：**Press Start 2P**（经典游戏字体）
- **用途**：章节标题、阵营名称、重要按钮

#### 正文字体

- **中文**：**霞鹜文楷**（可读性好的像素风字体）
- **英文**：**VT323**（等宽像素字体）
- **用途**：正文描述、事件文本、系统提示

### 3.2 字号规范

| 用途 | 字号 | 行高 | 说明 |
|------|------|------|------|
| **页面大标题** | 24px | 32px | 如：国家名称 |
| **章节标题** | 18px | 24px | 如：事件标题 |
| **正文** | 14px | 20px | 事件描述、新闻内容 |
| **辅助文本** | 12px | 16px | 提示、标签 |
| **小字** | 10px | 14px | 版权、版本号 |

### 3.3 字体渲染

```css
/* 像素字体渲染优化 */
.pixel-text {

  font-smooth: never;
  -webkit-font-smoothing: none;
  -moz-osx-font-smoothing: grayscale;
  image-rendering: pixelated;
}
```

## 四、UI组件规范

### 4.1 按钮样式

#### 主按钮（魔法按钮）

```
┌────────────────────────┐
│ ░░░ 确认行动 ░░░ │  ← 发光边框
└────────────────────────┘
```

- **尺寸**：160×40px
- **背景**：阵营主色（深色版本）
- **边框**：2px亮边 + 1px暗边（像素浮雕效果）
- **文字**：居中，14px，白色
- **Hover**：边框发光动画 + 轻微上移2px
- **Active**：按下效果，边框变暗

#### 次按钮（石质按钮）

- **尺寸**：120×32px
- **背景**：石岩灰 `#4A4A5A`
- **边框**：2px内嵌式边框
- **文字**：羊皮纸色 `#F4E4C1`

#### 危险按钮（血色按钮）

- **背景**：深红色 `#B71C1C`
- **边框**：发光红边
- **用于**：背叛、攻击、危险选项

### 4.2 面板样式

#### 主面板（羊皮卷轴）

```
╔══════════════════════════════════╗
║  ▓▓ 卷轴顶部装饰 ▓▓  ║
╠══════════════════════════════════╣
║                                  ║
║     内容区域                     ║
║     羊皮纸质感                   ║
║                                  ║
╠══════════════════════════════════╣
║  ▓▓ 卷轴底部装饰 ▓▓  ║
╚══════════════════════════════════╝
```

- **背景**：羊皮纸色 `#F4E4C1`
- **边框**：8px像素边框，深色外框 + 浅色内框
- **装饰**：四角像素图腾装饰
- **阴影**：下方4px投影

#### 次级面板（石板）

- **背景**：石岩灰 `#4A4A5A`
- **边框**：浮雕式像素边框
- **用于**：属性面板、信息卡片

#### 阵营面板

- **背景**：阵营主色（20%透明度叠加在羊皮纸上）
- **边框**：阵营主色亮边
- **装饰**：阵营徽章像素图案

### 4.3 分隔线与装饰

#### 像素分隔线

```
═══╦═══╦═══  或  ──▪──▪──
```

- **主分隔线**：3px高，像素图案
- **次分隔线**：1px，虚线像素

#### 装饰元素

- **角落装饰**：8×8像素的图腾角标
- **列表标记**：4×4像素的菱形或圆点
- **滚动条**：9px宽，像素风格滑块
