# 《三界风云录》视觉风格指南

## 整体视觉方向

### 设计理念
- **核心风格**：中国水墨风 + 轻度奇幻元素
- **情绪基调**：淡雅、古朴、有历史感
- **氛围目标**：史诗感、神秘感、叙事感
- **设计哲学**："墨分五色，意在笔先" —— 用极简的色彩层次传递丰富的叙事信息

### 视觉关键词
```
宣纸质感 · 水墨晕染 · 卷轴竹简 · 朱砂印章 · 山水意境
```

## 二、字体系统

### 2.1 字体选择

| 用途 | 推荐字体 | 备选字体 | 字重 | 说明 |
|-----|---------|---------|------|------|
| **大标题** | 方正清刻本悦宋 | 思源宋体 Heavy | 700-900 | 书法感，用于游戏标题、章节名 |
| **小标题** | 思源宋体 Bold | 方正书宋 | 600-700 | 清晰有力度，用于页面标题 |
| **正文** | 思源宋体 Regular | Noto Serif CJK | 400-500 | 易读性优先，用于描述文字 |

| **数据/数值** | DIN / Roboto | 思源黑体 | 500 | 西文数字，清晰易读 |

## 三、UI组件规范

### 3.1 按钮组件

#### 主按钮（朱砂按钮）
```
背景色：#C73E3A（朱砂红）
文字色：#F5F0E6（宣纸白）
边框：无 / 1px solid #8B0000（深红）
圆角：4px（微圆，不显现代感）
内边距：12px 24px
字体：思源宋体 600

悬停态：
- 背景色加深：#A03028
- 轻微上浮：translateY(-2px)
- 阴影：0 4px 8px rgba(199, 62, 58, 0.3)

按下态：
- 背景色更深：#7A2620
- 回弹：translateY(0)
```

背景色：transparent
文字色：#1A1A1A（浓墨）

内边距：10px 20px

- 背景色：rgba(26, 26, 26, 0.05)

#### 水墨分割线

## 五、插图风格规范

### 5.1 人物立绘

#### 风格定义
- **基础风格**：半写实水墨风
- **人物比例**：7-8头身，略微修长
- **线条处理**：
  - 轮廓线：浓淡变化，近实远虚
  - 衣纹：流畅的游丝描或钉头鼠尾描
  - 面部：淡墨渲染，少线多染

#### 色彩处理
- **肤色**：淡赭石打底，朱砂点唇、染颊
- **衣饰**：
  - 苍龙帝国：墨绿为主，金线点缀
  - 霜狼联邦：灰蓝为主，银白装饰
  - 金雀花：暖黄赭石，色彩较明快
  - 边境联盟：土褐灰白，朴素实用

#### 背景处理
- 简单水墨渲染，不抢主体
- 可使用淡墨山水、云雾烘托氛围

### 5.2 场景背景

#### 风格定义
- **基础风格**：写意山水
- **构图原则**：
  - 近景：较为细致的树石
  - 中景：主体场景
  - 远景：淡墨远山，云雾缭绕

#### 常用元素
| 元素 | 表现手法 |
|-----|---------|
| 山 | 斧劈皴或披麻皴，近浓远淡 |
| 水 | 留白或淡墨波纹，平静深远 |
| 云 | 勾云法或染云法，流动感 |
| 树 | 夹叶或点叶，墨色层次丰富 |
| 建筑 | 简笔勾勒，不喧宾夺主 |

### 5.3 事件插图

#### 风格定义
- **基础风格**：连环画风格
- **画面尺寸**：16:9 或 4:3
- **叙事性**：单幅画面讲述事件核心冲突

#### 构图建议
- 戏剧性时刻捕捉
- 人物动态鲜明
- 留白适当，给文字空间

| 印章盖下 | 缩放1.2→1 + 轻微旋转 | 300ms | cubic-bezier(0.68, -0.55, 0.265, 1.55) |

### 6.3 加载状态

```
设计概念：毛笔书写加载动画
表现形式：
- 一个"墨"字或" Loading"文字被逐笔写出
- 或：墨滴落入水中，涟漪扩散
- 进度条：卷轴展开形式
```
