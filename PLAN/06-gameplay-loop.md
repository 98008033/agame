# Agame 核心玩法循环

## 六、核心玩法循环

### 6.1 时间系统

**游戏内时间**：
- 1个游戏日 = 现实中约2小时（可配置）
- 1个游戏月 = 现实中约2.5天
- 1个游戏年 = 现实中约30天
- 1个赛季 = 1个现实月

**Agent唤醒周期**：
```
定时唤醒（Cron）：
  - Chronos: 每天凌晨3:00
  - 国家Agent: 每6小时
  - NPC Agent: 每天1次（随国家Agent后）

事件触发：
  - 玩家重大决策时
  - 战争爆发时
  - 特殊事件链推进时
```

### 6.2 世界运转周期

世界不是回合制——它**持续运转**。Agent按固定周期唤醒，处理状态变化，生成新事件。玩家登录时看到的是世界运转的结果，而不是等待玩家操作才推进。

```
一个游戏日的运转流程（自动，不需要玩家参与）：
1. Agent唤醒 → 检查世界状态变化
2. 事件生成 → 根据状态变化产生新事件
3. NPC行动 → NPC执行各自的任务/目标
4. 资源结算 → 领地产出、消耗、贸易
5. 结果落地 → 事件影响世界状态
6. 等待下一个周期

玩家登录时：
1. 阅读世界晨报（过去N个游戏日发生了什么）
2. 查看个人事件（与你直接相关的变化）
3. 做出决策（回应事件、分配资源、选择行动）
4. 离线后世界继续运转
```

### 6.3 玩家日常循环

**登录阶段**（5-10分钟）：
```
1. 读"世界晨报"——三国的新闻摘要
   - 苍龙帝国：大皇子在朝会上弹劾了首辅...
   - 霜狼联邦：血狼部族拒绝缴纳矿税...
   - 金雀花王国：新航线开辟，海盗活动增加...
   - 灰烬谷：赤焰村发生火灾，烈焰向各国求援...

2. 查看"个人事件"——与你直接相关
   - 你领地的产出和消耗
   - 你的人际关系变化
   - 你收到的任务/邀请/威胁

3. 做出2-3个关键决策
   - 接受/拒绝任务
   - 分配资源
   - 选择行动方向

4. 收取定时奖励
   - 领地产出
   - 任务完成奖励
   - 日常登录奖励
```

**深度游玩阶段**（30分钟，每周1-2次）：
```
1. 经营领地——建设、升级、规划
2. 策划外交——谈判、联盟、挑拨
3. 推进事件链——深入体验叙事
4. 探索冒险——发现新区域、新NPC、新资源
5. 科技/魔法研究——解锁新能力
```
