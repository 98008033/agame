# 边境联盟开局剧本

> **设计定位**：新手推荐起点，展示游戏核心循环，引导玩家理解三界世界观
> **预计游戏时长**：30-45分钟（开局引导）
> **对应文档**：PLAN/03-factions.md 3.4节边境联盟详细设定

---

## 一、阵营背景介绍

### 1.1 边境联盟概况

你站在三界交汇之地——**灰烬谷**。

这里曾是辉煌大帝国的郊区，如今却是地图上的一块空白。三个强大的国家环绕着你：东面的**苍龙帝国**以礼法秩序统治，北面的**霜狼联邦**崇尚武力与荣耀，南面的**金雀花王国**用金币衡量一切。

而你所在的地方，叫做**边境联盟**——或者它对外宣称的正式名称"灰烬谷自由都市"。

这里没有国王，没有议会，没有军队。只有数十个村落组成的松散联盟，以及一个每月召开一次的"谷主大会"。在这个大会上，各村长们争论着如何分配水源、如何防御匪帮、以及——最重要的——如何在三国的夹缝中生存。

### 1.2 当前局势

**暮光村**，联盟最古老的村落之一，位于灰烬谷中央的一片缓坡上。站在这里的小山丘顶，你可以同时看到三个方向的地平线：

- **东面**：苍龙的山脉在朝阳下泛着金光
- **北面**：霜狼的冰原上空偶尔闪过极光
- **南面**：金雀花的丘陵在暮色中呈现暗红色

老人们说，暮光村之所以叫"暮光"，是因为每天黄昏时分，夕阳会同时照亮三个方向——金色、紫色、红色交织在一起，仿佛三界在此握手。

但这份宁静正在破碎。

- 苍龙的密探"龙使"已经在村中住了三个月
- 霜狼的代表"北风"三天前带着一队战士抵达
- 金雀花的商人"金掌柜"正在收购村里的粮食，价格出奇地高

三国都在暗中渗透灰烬谷，而联盟内部也出现了分歧：有人主张独立建国，有人主张选择一方效忠，还有人——比如你的导师老根——坚持维持现状，在夹缝中求生存。

### 1.3 你的故事从这里开始

你来到灰烬谷的原因，只有你自己知道。

也许是因为逃避过去的追兵，也许是因为听闻这里有发财的机会，又或者……你失去了记忆，只记得要"找到真相"。

无论如何，当你踏入暮光村的那一刻，命运已经开始转动。

## 二、初始角色设定

### 2.1 玩家身份选择（三选一）

在剧本开始前，玩家需要选择一个身份背景。这个选择会影响：
- 初始技能点分配
- 部分NPC的初始态度
- 专属的剧情分支

#### 身份A：逃难者（推荐新手）

> 你从三国之一的领土逃来，身后或许有追兵，或许有不愿回首的往事。灰烬谷的"不问过去"传统让你感到安全。

- **初始金币**：100
- **初始影响力**：30
- **初始技能**：
  - 基础格斗 L1（自保本能）
  - 察言观色 L1（在危险中学会读懂人心）
- **特殊优势**：对所选国家的政治体制有内部了解
- **专属剧情**：可能遇到来自祖国的追兵或旧识

#### 身份B：寻机者

> 你听说灰烬谷是"机遇之地"——没有王法约束，没有身份限制，只要够聪明、够勇敢，就能从一无所有变成腰缠万贯。

- **初始金币**：150
- **初始影响力**：40
- **初始技能**：
  - 基础格斗 L1
  - 情报搜集 L1（商人本能）
- **特殊优势**：与商贸村有初始好感度加成
- **专属剧情**：更容易触发商业相关事件

#### 身份C：失忆者

> 你醒来时倒在暮光村外的林间小道上，身上没有任何能证明身份的物品，只有一块刻着奇怪符号的石头。老根收留了你，但你总觉得——你来这里是有原因的。

- **初始金币**：80
- **初始影响力**：20
- **初始技能**：
  - 基础格斗 L2（身体记得战斗）
- **特殊优势**：随着剧情推进，逐步解锁隐藏记忆
- **专属剧情**：专属的主线任务线"寻回自我"

### 2.2 初始关系网

| NPC | 关系值 | 说明 |
|-----|--------|------|
| 老根 | +30（友善） | 收留你的村长，导师角色 |
| 小苏 | +10（认识） | 酒馆老板娘，情报来源 |
| 铁匠·老锤 | 0（陌生） | 村中铁匠，可修理装备 |
| 龙使 | 0（陌生） | 苍龙密探，态度友善但深不可测 |
| 北风 | 0（陌生） | 霜狼代表，直率但强硬 |
| 金掌柜 | 0（陌生） | 金雀花商人，热情洋溢 |

### 2.3 起始装备

- **武器**：旧匕首（攻击力+3）或 木杖（攻击力+2，可施法）
- **防具**：粗布衣服（防御力+1）
- **消耗品**：干粮×3、水袋×1
- **特殊物品**：
  - 逃难者：祖国的某种信物（可出售或留作纪念）
  - 寻机者：一本残缺的商业笔记
  - 失忆者：神秘石头（无法丢弃，用途未知）

## 三、起始场景描述

### 3.1 场景：暮色酒馆

**时间**：游戏第1天，黄昏时分（暮光时刻）

**地点**：暮光村中心，"暮色"酒馆

**环境描写**：

```
暮色酒馆是暮光村最热闹的地方，也是整个边境联盟的消息集散地。

酒馆建在一棵千年古树旁，树干粗得要三人合抱。老根常说，这棵树比任何国家都古老，它见证了帝国的兴衰，也必将见证灰烬谷的未来。

酒馆的屋顶是用茅草和木板拼凑的，墙上挂满了来自三国的奇怪物品——苍龙的丝绸旗、霜狼的兽骨风铃、金雀花的航海图。这些东西大多是客人用来抵酒钱的。

空气中混杂着麦酒、烤肉和烟草的味道。角落里总有几个不愿意被人看清面容的客人，他们的斗篷上沾着旅途的尘埃。

小苏站在吧台后面，她今年三十出头，面容不算出众，但有一双仿佛能看穿人心的眼睛。据说她年轻时在苍龙的天都待过，后来不知为何来到了这里。

"不问来处，不问去处"——这是暮色酒馆的规矩，也是整个边境联盟的生存哲学。
```

**氛围**：
- 温暖的灯光与外面渐暗的天色形成对比
- 背景中有低沉的交谈声，偶尔爆出笑声或争执声
- 小苏擦拭着永远擦不完的酒杯，眼神却不时扫视每个客人
- 老根坐在他惯常的角落位置——能看到大门，背靠墙壁

### 3.2 场景互动点

玩家可以在起始场景中：

| 互动对象 | 可获得信息 |
|----------|------------|
| 老根 | 边境联盟的历史、当前局势、新手建议 |
| 小苏 | 三国使者的动态、最近的传闻、任务委托 |
| 其他客人 | 随机传闻（可能真可能假） |
| 告示板 | 可接取的任务（护送、寻人、收集物资） |

## 四、第一个事件：暮光抉择

### 4.1 事件触发

**触发条件**：开局自动触发，与老根对话后进入剧情

**事件背景**：

老根看着你，眼中带着一种你读不懂的深意。

"年轻人，你既然选择留在暮光村，就要明白这里的规矩。"他顿了顿，"但规矩之外，每个人都要做出自己的选择。"

他指了指酒馆角落里各自坐着的三个人：

"看见那三位了吗？他们代表了三股势力，都在争取联盟的支持。今晚，谷主大会要在村里召开，讨论如何应对三国的'好意'。"

老根站起身，拍了拍你的肩膀。

"我想请你帮我一个忙——去和那三位聊聊，看看他们到底想要什么。然后……告诉我，你觉得我们应该怎么办。"

**【事件正式开始】**

### 4.2 事件流程

#### 第一阶段：情报收集（可选顺序）

玩家需要与三位使者分别对话，了解他们的来意。对话顺序自由。

**龙使（苍龙帝国）**：

> 龙使是个三十岁出头的男人，穿着考究但不张扬的灰色长袍，腰间挂着苍龙官员的令牌。他见到你时露出了礼貌的微笑。
>
> "啊，新来的朋友。请坐，我正好独自一人。"
>
> 他的声音温和，但眼神始终在评估你。
>
> "苍龙帝国向来重视秩序与稳定。灰烬谷……说句实话，这里太混乱了。匪帮肆虐，没有法律保护商人，各村各自为政。"
>
> 他压低声音："陛下愿意提供保护——真正的保护。军队驻防、律法执行、商业规范。作为交换，联盟只需承认苍龙的宗主权，每年缴纳合理的税赋。"
>
> 他递给你一杯茶："考虑一下。在苍龙的保护下，像你这样的年轻人可以安心发展，不必担心哪天醒来发现村子被匪帮烧了。"

**北风（霜狼联邦）**：

> 北风是个身材魁梧的中年男人，穿着霜狼战士常穿的皮甲，脸上有一道从眉骨延伸到下巴的伤疤。他没有龙使的礼貌，但有一种令人信服的直率。
>
> "你就是老根说的那个新人？"他的声音像是从胸腔深处发出的，"坐。喝酒。"
>
> 他推过来一杯浑浊的烈酒。
>
> "苍龙人是不是跟你说'保护'？"他嗤笑一声，"他们的保护就是把你们变成奴隶。交税，服从，听话。一代人之后，你们的孩子会忘记什么是自由。"
>
> 他重重地拍了一下桌子："霜狼不一样。我们尊重强者，尊重自由人。联盟不需要臣服，只需要结盟。我们一起对抗匪帮，一起面对外敌。没有宗主权，没有税赋——只有战士之间的盟约。"
>
> 他盯着你的眼睛："但结盟意味着责任。当霜狼需要你们的时候，你们必须站出来。这是荣誉的问题。"

**金掌柜（金雀花王国）**：

> 金掌柜是个五十岁左右的男人，穿着金雀花商人标志性的金色镶边外套，十个手指上戴着五个戒指。他见到你时热情地招手。
>
> "新朋友！来来来，尝尝我从南方带来的蜜酒！"
>
> 他的热情让人难以拒绝。
>
> "苍龙和霜狼都在谈政治，谈权力——那些太复杂了，对吧？"他眨眨眼，"我们金雀花人只谈一件事：钱。"
>
> "看看这周围，"他张开双臂，"灰烬谷地理位置多好？三国的商路在此交汇，只要稍加经营，这里可以成为大陆最繁荣的贸易中心！"
>
> 他压低声音，做出神秘的样子："我已经和赤焰村、商贸村的几位村长谈过了。如果我们建立自由贸易港，免除关税，允许各国商人自由往来……你猜会怎样？"
>
> "钱。大把的钱。不只是我，每个人都发财。包括你，年轻人。"
>
> 他塞给你一个小袋子——里面有10枚金币。
> "见面礼。考虑一下，谁能让你的生活变得更好？是收税的苍龙，是要求你打仗的霜狼，还是让你发财的我？"

#### 第二阶段：做出抉择

在与三位使者对话后，玩家需要向老根汇报。老根会问："你觉得我们应该怎么办？"

**可选决策**：

| 选项 | 需要的技能/条件 | 后果 |
|------|----------------|------|
| **支持苍龙** | 无 | 与龙使关系+20，与北风关系-10，苍龙声望+15 |
| **支持霜狼** | 无 | 与北风关系+20，与龙使关系-10，霜狼声望+15 |
| **支持金雀花** | 无 | 与金掌柜关系+20，获得50金币，金雀花声望+15 |
| **保持中立** | 谋略L1+ | 与三方关系+5，老根关系+20，获得"平衡者"称号 |
| **提出自己的方案** | 谋略L2+ 或 经营L2+ | 触发隐藏剧情线，老根关系+30 |

**选择后的即时反馈**：

老根听完你的建议，缓缓点头。

"每个选择都有代价，"他说，"但重要的是——这是你自己的选择。在灰烬谷，没有人能替你做决定。"

"明天的谷主大会，我会把你的意见转达给其他村长。现在……"他露出一个疲惫但温暖的微笑，"去休息吧。你的旅程才刚刚开始。"

### 4.3 事件后果

**即时奖励**：
- 经验值：+50
- 与相关NPC关系提升
- 开启对应阵营的任务线

**长期影响**：
- 影响边境联盟的整体走向（多个玩家的选择会累积影响）
- 决定哪些阵营任务对你开放
- 影响最终结局的部分分支

**次日解锁内容**：
1. **晨报系统**：每天可阅读三国动态小说
2. **任务板**：接受各类委托任务
3. **技能学习**：可向老根或其他NPC学习技能
4. **首次许愿**：向Agent许愿（游戏核心循环开始）

## 七、剧本特色设计

### 7.1 与三国开局的区别

| 特色 | 边境联盟 | 三国开局 |
|------|----------|----------|
| 自由度 | 高（无强制立场） | 低（有阵营身份约束） |
| 引导强度 | 强（新手教程） | 中等 |
| 资源起步 | 较低 | 较高（国家背景支持） |
| 故事线 | 探索型（我是谁/我要去哪） | 参与型（改变世界） |
| 推荐玩家 | 新手/喜欢自由探索 | 有经验/喜欢明确目标 |

### 7.2 可重玩性设计

本剧本支持多次重玩，每次可能有不同体验：

1. **身份差异**：三种身份有专属剧情分支
2. **选择差异**：不同选择开启不同任务线
3. **随机元素**：部分NPC对话和事件有随机变化
4. **技能影响**：不同技能等级会看到不同选项

### 7.3 与核心系统的衔接

本剧本引导玩家逐步接触游戏核心系统：

| 系统 | 引入方式 | 完全解锁 |
|------|----------|----------|
| 叙事系统 | Day 1 体验第一个小说章节 | Day 2 晨报系统 |
| 许愿系统 | Day 4 任务引导 | Day 4 正式使用 |
| 技能系统 | Day 1 初始技能 | Day 2 学习新技能 |
| 经济系统 | Day 1 金币使用 | Day 2 自由交易 |
| 关系系统 | Day 1 与NPC互动 | 持续累积 |
| 阵营系统 | Day 1 暮光抉择 | Day 7 确定立场 |

### 8.2 常见问题FAQ

**Q：我选择了支持苍龙，还能和霜狼做生意吗？**
A：可以。在边境联盟，立场选择不会完全封锁其他选项，但会影响难度和收益。

**Q：如果我想换阵营怎么办？**
A：Day 7 时你会有一次正式选择阵营的机会。之后更换阵营会非常困难。

**Q：失忆者的石头有什么用？**
A：随着剧情推进会逐步揭示。建议不要出售。

**Q：新手保护期结束后死亡会怎样？**
A：角色会死亡，可能无法复活（符合游戏核心设定）。请在保护期内熟悉游戏机制。
