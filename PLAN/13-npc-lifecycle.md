# NPC生命周期系统

> **核心理念**：NPC不是永恒的道具，而是会生老病死的活人。一个将军战死，他的位置需要填补；一个村长病逝，村子里的权力格局会改变。这种"人的流动"是世界动态感的核心来源。

---

## 十三、NPC生命周期

### 13.1 时间尺度

```
现实时间              游戏时间
─────────────────────────────────
1小时        ≈       半个游戏日
1天          ≈       1个游戏月
1个月        ≈       1个游戏年
1年          ≈       12个游戏年
```

**NPC寿命范围**：
| 类型 | 平均寿命 | 高龄 | 备注 |
|------|----------|------|------|
| 普通人 | 55-70岁 | 75+ | 受医疗水平限制 |
| 武者 | 50-65岁 | 70+ | 旧伤累积 |
| 魔法使用者 | 60-75岁 | 80+ | 魔力延缓衰老，但契约魔法有代价 |
| 高级治愈师 | 70-85岁 | 90+ | 自愈能力延长寿命 |

### 13.2 年龄阶段与属性变化

```
年龄        阶段        属性变化
──────────────────────────────────────
0-12        幼年        无行动能力，NPC处于"培养中"
12-18       少年        属性开始形成，L1-L2技能
18-30       青年        属性快速增长，建立职业基础
30-45       壮年        属性达到峰值，最适合领导和战斗
45-60       中年        属性缓慢下降，但经验和智慧提升
60-70       老年        属性明显下降，战斗能力衰退
70+         衰老        属性大幅下降，死亡概率急剧上升
```

**属性随年龄变化曲线**：

```
属性值
  │
  │        ╭────╮
  │       ╱      ╲
  │      ╱        ╲
  │     ╱          ╲
  │    ╱            ╲
  │   ╱              ╲
  │  ╱                ╲
  │ ╱                  ╲
  └──────────────────────── 年龄
   0   15   30   45   60   75

军事力/魔法力:    峰值30-40岁，50岁后快速下降
统治力/外交力:    峰值40-55岁，下降缓慢（经验补偿）
商业力:           峰值35-50岁，下降最慢
```

### 13.3 死亡系统

#### 死亡类型与概率

**自然死亡**（老病）：
```
60-65岁：每游戏年 3% 概率自然死亡
65-70岁：每游戏年 8%
70-75岁：每游戏年 15%
75-80岁：每游戏年 25%
80+岁：   每游戏年 40%
```

**战斗死亡**（取决于角色参与的冲突）：
```
普通士兵：  每场战斗 5-15% 战死概率
精锐战士：  每场战斗 2-8%
将军/将领： 每场战斗 1-3%（有护卫）
单挑/决斗： 10-30%（取决于实力差距）
```

**暗杀/谋杀**：
```
需要玩家或NPC主动策划
成功率取决于：目标护卫力量 vs 暗杀者能力
成功后有被追查的风险
```

**瘟疫/疾病**：
```
随机爆发（由Chronos注入）
每个NPC有感染概率（与人口密度、卫生条件相关）
有治愈师/医者可降低死亡率
无医疗：50%致死率
有普通医者：20%致死率
有高级治愈师：5%致死率
```

**处决**（政治死亡）：
```
政变失败者
被俘的敌方将领
叛国者
需要对应政治系统支持（苍龙有刑部，霜狼有荣誉法则）
```

**意外/事故**：
```
探索遗迹
魔法反噬
海上风暴
极低概率但不可预测
```

#### 死亡通知机制

NPC死亡后，通过以下方式通知玩家：
```
1. 世界晨报：重要NPC死亡上头版
   "苍龙帝国首辅秋实于昨夜病逝，享年67岁"

2. NPC传话：与死者有关系的NPC会主动告知
   "你听说了吗？老根村长走了……"

3. 事件触发：重要NPC死亡可能触发事件链
   "秋实之死"→ 首辅之位空缺 → 三位皇子争夺推荐权

4. 被动发现：玩家到达某地时发现变化
   你来到铁壁关，发现守将已不是铁壁，而是一个陌生的年轻人
```

### 13.4 继承系统

不同阵营有不同的继承规则：

#### 苍龙帝国——嫡长子继承 + 科举选拔

```
皇位继承：
  嫡长子优先，但皇帝可以"立贤"
  当前局面：皇帝未立太子，三子争夺

官位继承：
  六部尚书去世 → 由同部侍郎接任，或皇帝从科举人才中选拔
  将领去世 → 由副将接任，或兵部推荐

关键机制：玩家可以影响继承
  - 支持某位皇子 → 提高其夺嫡成功率
  - 推荐某个人才 → 可能成为新任官员
  - 暗杀某位继承人 → 改变继承格局
```

#### 霜狼联邦——实力推选

```
首领继承：
  首领去世或退位 → 首领议会重新推选
  推选标准：战功、威望、部族实力
  可能引发部族冲突甚至内战

部族首领继承：
  部族内部决定，通常是首领之子或最强战士
  可以通过决斗挑战获得首领之位

关键机制：
  - 帮助某人建立战功 → 提高其推选竞争力
  - 挑拨部族关系 → 可能引发继承纷争
  - 决斗系统：玩家或NPC可以挑战首领
```

#### 金雀花王国——金钱与选举

```
议长继承：
  议长去世 → 贵族议会投票选举
  实际上由最大的商业集团操控

银行家继承：
  通常由家族内部继承
  但如果有更强的金融势力，可以"收购"其位置

关键机制：
  - 资助某位候选人 → 提高其当选概率
  - 经济打击某家族 → 削弱其继承实力
  - 契约魔法：可以用契约"绑定"某人的效忠
```

#### 边境联盟——自然更替

```
村长继承：
  村长去世 → 村民推选新村长
  通常是村里最有能力或最受尊敬的人
  玩家作为暮光村的村长，可以指定继承人

谷主大会：
  某村村长去世 → 该村派新代表参加
  不涉及复杂的继承规则
```

### 13.5 NPC关系网

**核心原则**：新人不是凭空生成的——他们必须通过游戏世界内部的关系（血缘、师徒、门生）自然产生。NPC首先是一个"人"，有父母、有子女、有老师、有学生。

#### 13.5.1 NPC关系类型

```
血缘关系：
  └─ 父母/子女    直系传承，最强纽带
  └─ 兄弟姐妹    潜在合作者或竞争者
  └─ 配偶        政治联姻或感情纽带
  └─ 远亲        弱纽带，但在继承中有用

师徒关系：
  └─ 师父/弟子   技能和职位传承的第二通道
  └─ 学院同窗    弱纽带，但可能形成派系

政治关系：
  └─ 上下级      职位继承的第一顺位（副手接任）
  └─ 政治盟友    共同利益驱动
  └─ 政敌        对立关系

社会关系：
  └─ 挚友        情感纽带
  └─ 仇敌        对立关系
  └─ 恩人/欠债   道德债务
```

#### 13.5.3 新人的产生途径

新NPC只能通过以下途径在世界中出现：

**途径1：子女成年**（最常见）
```
NPC有子女 → 子女随时间成长 → 18岁时成为可用NPC
子女的属性 = 父母属性的加权平均 ± 随机变异
子女的技能 = 受父母技能影响 + 教育环境 + 个人天赋
子女的性格 = LLM生成，受家庭环境影响

例子：秋实有两子
  长子·秋铭：继承了父亲的政治天赋，性格刚烈（与父亲不同）
  次子·秋墨：痴迷五行魔法，对政治不感兴趣
```

**途径2：师徒传承**（技能和职位）
```
NPC收弟子 → 弟子跟随学习 → 师父去世后弟子继承衣钵
弟子继承的不只是职位，还有知识、人脉、和义务
弟子的性格和能力可能与师父不同（LLM决定差异程度）

例子：五行学院长·墨池是灵元的弟子
  灵元死后，墨池接任天师府掌教
  但墨池比师父更开放，愿意接受外系魔法
```

**途径3：副手接任**（军事/行政）
```
上级死亡 → 副手/下属接任
副手的能力通常不如上级（但也有青出于蓝的可能）
副手的忠诚度和性格会影响接任后的行为

例子：铁壁战死 → 副将·望山接任北境将军
  望山能力不如铁壁，但更加谨慎
```

**途径4：移民/流亡者**（罕见，需要事件触发）
```
这不是系统自动生成的，而是：
- 由事件链产生（如苍龙内战导致流亡者北上霜狼）
- 由Chronos注入（为了平衡世界，注入关键人才）
- 由玩家行为产生（玩家招募了来自其他阵营的人）

移民/流亡者是重大事件，不是日常现象
```

**途径5：底层崛起**（罕见，需要条件）
```
科举考试中涌现的人才（苍龙特有）
佣兵战争中脱颖而出的战士（霜狼/边境特有）
商业竞争中胜出的新人（金雀花特有）

这些不是凭空出现的，而是：
- 系统中有大量"无名NPC"（不在玩家视野内）
- 他们默默成长，当有空缺时被发现
- LLM给他们赋予背景和性格
```

#### 13.5.4 继承优先级（通用规则）

当一个关键NPC死亡时，系统按以下顺序寻找继承人：

```
第一顺位：指定继承人
  └─ NPC生前指定的继承人（通常是子女或心腹）
  └─ 如果指定继承人还活着且有能力 → 自动接任

第二顺位：直系血亲
  └─ 长子/长女（苍龙嫡长子制）
  └─ 最有能力的子女（霜狼实力制）
  └─ 持股最多的亲属（金雀花金钱制）

第三顺位：师徒/门生
  └─ 师父的首席弟子
  └─ 该职位培养的接班人

第四顺位：副手/下属
  └─ 同部门的副职
  └─ 军队中的副将

第五顺位：外部竞争
  └─ 如果以上都没有 → 触发"继承危机"事件
  └─ 多方势力争夺空缺职位
  └─ 这本身就是重大事件，不是简单的"系统生成"
```

#### 13.5.5 继承危机（当没有明确继承人时）

如果一个关键NPC死亡且没有合适的继承人，不是系统偷偷生成一个，而是触发一场"继承危机"：

```
触发条件：关键NPC死亡 + 无明确继承人

事件表现：
  - 世界晨报报道空缺
  - 多方势力开始角逐
  - 玩家可以介入支持某一方
  - 最终胜出者获得职位

结果：
  - 可能是一场和平的选拔（苍龙科举）
  - 可能是一场残酷的斗争（霜狼决斗/金雀花金钱战）
  - 可能导致职位长期空缺（影响城市运转）
  - 可能引来外部干预（其他国家趁虚而入）
```

#### 13.5.6 对玩家的影响

```
玩家也需要遵守这套规则：
  - 玩家有配偶和子女
  - 子女随时间成长
  - 玩家死亡后由继承人接替
  - 如果玩家没有培养继承人 → 游戏结束

玩家可以主动利用这套系统：
  - 安排政治联姻（让自己的子女与NPC子女结婚）
  - 培养弟子（收NPC为徒，培养接班人）
  - 暗杀继承人（消灭某NPC的所有继承人，导致继承危机）
  - 扶植傀儡（让能力弱的人接任，方便操控）
```

### 13.6 玩家的生命周期

玩家角色也遵循同样的衰老规则，但有特殊机制：

```
玩家寿命：与NPC相同（55-70岁平均）
玩家衰老：
  - 属性随年龄变化
  - 战斗能力在50岁后下降
  - 但统治力/外交力可以持续提升

玩家死亡后果：
  - 如果有继承人 → 继承人接替，玩家切换操控角色
  - 如果没有继承人 → 游戏结束（该存档）
  
延长寿命的方式：
  - 高级治愈师的治疗
  - 某些稀有魔法/药剂
  - 契约魔法（有代价）
  - 裂隙能量（有风险）
```

**继承人系统**：
```
玩家可以培养继承人（子女或弟子）
继承人跟随玩家学习技能
玩家死亡后自动切换为继承人
如果培养得当，继承人可以继承部分属性和技能
```

### 13.7 生命周期对世界的影响

NPC的生死会引发连锁反应：

| 事件 | 短期影响 | 长期影响 |
|------|----------|----------|
| 皇帝病逝 | 权力真空，夺嫡白热化 | 新皇帝改变国家方向 |
| 大将军战死 | 军心动摇，防线不稳 | 新将领带来新战术 |
| 首席治愈师病逝 | 瘟疫风险增大 | 医疗体系需要重建 |
| 大商人破产/死亡 | 贸易线路中断 | 新商人填补真空或形成垄断 |
| 村长死亡 | 村民推选新村长 | 新村长可能改变效忠对象 |
| 情报头子被杀 | 情报网络瘫痪 | 新人重建或情报网被敌方接管 |

### 13.8 与其他系统的联动

```
生命周期 ←→ Agent系统
  Agent需要感知关键NPC的死亡并做出响应
  国家Agent在核心NPC死亡时触发继承决策
  Chronos在某方关键人物集体死亡时注入平衡事件

生命周期 ←→ 事件系统
  重要NPC死亡触发事件链
  "秋实之死" → 首辅之争 → 新政策 → 影响灰烬谷

生命周期 ←→ 技能系统
  NPC死亡后，他的技能/知识不会完全消失
  可能被弟子继承、写成书籍、或失传

生命周期 ←→ 外交系统
  某国外交官死亡 → 新外交官带来新的外交风格
  两国关系可能因人事变动而骤变

生命周期 ←→ 玩家体验
  "你认识的人正在老去和死去" → 情感投入
  "你培养的继承人接替了你的位置" → 传承感
  "你暗杀的敌方将军改变了战争走向" → 掌控感
```
