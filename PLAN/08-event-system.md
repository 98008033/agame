# 事件系统

## 八、事件系统

### 8.1 事件来源

```
世界事件（Chronos）    → 影响多阵营，每现实日0-2个
国家事件（国家Agent）  → 影响单阵营，每现实日1-3个
个人事件（NPC Agent）  → 与玩家直接相关，每现实日1-5个
环境事件（系统自动）   → 天气、季节、灾害
```

### 8.2 事件链系统

**事件链结构**：
```
触发条件 → 前奏事件 → 核心事件 → 后续事件 → 结局事件
  世界状态   营造氛围   关键抉择   选择后果    长期影响
```

**示例事件链："流亡皇子"**
```
触发：苍龙帝国夺嫡进入白热化（夺嫡指数>70）

前奏：
  - 世界晨报：大皇子被废，二皇子夺权
  - NPC情报：有神秘车队进入灰烬谷

核心：
  - 废太子天枢逃至你的领地
  - 选择：庇护 / 通知苍龙 / 驱逐 / 勒索

后续（根据选择分支）：
  A-庇护：苍龙密探追查，需要应对
  B-通知：获得苍龙重赏，但失去民心
  C-驱逐：安全，但错过机会
  D-勒索：获得资源，但得罪两边

结局（多种可能）：
  - 元昊复位成功 → 你成为开国功臣
  - 元昊被杀 → 你被苍龙追杀
  - 元昊隐居 → 获得一个强力NPC
  - 元昊建国 → 第五势力出现
```
