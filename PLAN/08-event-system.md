# 事件系统

## 八、事件系统

### 8.1 事件来源

```
世界事件（Chronos）    → 影响多阵营，每现实日0-2个
国家事件（国家Agent）  → 影响单阵营，每现实日1-3个
个人事件（NPC Agent）  → 与玩家直接相关，每现实日1-5个
环境事件（系统自动）   → 天气、季节、灾害
```

**事件来源优先级**：

| 来源 | 触发频率 | 影响范围 | 示例 |
|------|----------|----------|------|
| **Chronos** | 每日0-2个 | 世界格局 | 战争爆发、和平条约、重大政治变动 |
| **国家Agent** | 每日1-3个 | 单一阵营 | 政策调整、军事行动、外交行动 |
| **NPC Agent** | 每日1-5个 | 个人相关 | 任务请求、冲突调解、商机分享 |
| **环境系统** | 随机触发 | 特定区域 | 旱灾、瘟疫、匪患 |

### 8.3 事件链系统

**事件链结构**：
```
触发条件 → 前奏事件 → 核心事件 → 后续事件 → 结局事件
  世界状态   营造氛围   关键抉择   选择后果    长期影响
```

**示例事件链："流亡皇子"**
```
触发：苍龙帝国夺嫡进入白热化（夺嫡指数>70）

前奏：
  - 世界晨报：大皇子被废，二皇子夺权
  - NPC情报：有神秘车队进入灰烬谷

核心：
  - 废太子天枢逃至你的领地
  - 选择：庇护 / 通知苍龙 / 驱逐 / 勒索

后续（根据选择分支）：
  A-庇护：苍龙密探追查，需要应对
  B-通知：获得苍龙重赏，但失去民心
  C-驱逐：安全，但错过机会
  D-勒索：获得资源，但得罪两边

结局（多种可能）：
  - 元昊复位成功 → 你成为开国功臣
  - 元昊被杀 → 你被苍龙追杀
  - 元昊隐居 → 获得一个强力NPC
  - 元昊建国 → 第五势力出现
```

### 8.4 触发机制

**触发条件类型**：

| 类型 | 说明 | 示例 |
|------|------|------|
| **等级条件** | 玩家等级要求 | 等级≥5 |
| **资源条件** | 金币/影响力要求 | 金币≥500 |
| **关系条件** | NPC关系值要求 | 关系≥40 |
| **阵营条件** | 效忠/声望要求 | 苍龙声望≥30 |
| **技能条件** | 技能等级要求 | 谋略L3+ |
| **时间条件** | 游戏时间要求 | 效忠≥7天 |
| **状态条件** | 世界状态要求 | 饥荒状态 |
| **随机触发** | 基础概率+权重 | 边境区域匪患概率+20% |

**触发权重配置**：
```
效忠选择类：20%（游戏前期更高）
资源抉择类：30%（日常事件）
人际冲突类：25%（与NPC关系密切时更高）
危机应对类：25%（边境区域更高）
```

**冷却机制**：
- 同一事件类型：7天内不重复触发
- 同一NPC相关事件：3天内不重复触发
- 失败的事件：30天后可再次触发

### 8.5 后果设计

**后果类型**：

| 类型 | 说明 | 示例 |
|------|------|------|
| **资源变化** | 金币/影响力增减 | 金币+500 |
| **关系变化** | NPC关系值变化 | [NPC]关系+30 |
| **声望变化** | 阵营声望变化 | 苍龙声望+20 |
| **状态获得** | 获得标签/身份 | 获"叛徒"标签 |
| **事件触发** | 触发后续事件 | 触发"追杀"事件 |
| **技能成长** | 技能经验增加 | 谋略+10经验 |

**标签系统**：

标签是玩家行为的历史记录，影响NPC对玩家的态度和可选事件：

| 标签类型 | 示例 | 效果 |
|----------|------|------|
| **正面标签** | 仁慈、忠诚、勇敢 | 解锁善意事件，NPC态度+10 |
| **负面标签** | 叛徒、懦夫、奸商 | 解锁敌意事件，NPC态度-10 |
| **中性标签** | 独行者、谨慎、务实 | 解锁特殊事件分支 |

**叙事结构**：

每个事件包含五个叙事部分：

```typescript
interface EventNarrative {
  eventId: string;
  preface: string;      // 事件前置叙述（触发时的场景）
  choices: string[];    // 每个选项的叙事描述
  outcomes: string[];   // 每个结果的叙事反馈
  aftermath: string;    // 事件后续影响（在晨报中提及）
}
```

### 8.7 技能与事件的关联

**技能等级决定可见选项**：

| 技能要求 | 效果 |
|----------|------|
| 谋略L3+ | 解锁隐藏选项、降低风险 |
| 武力L4+ | 解锁战斗选项、提升胜率 |
| 外交L3+ | 解锁谈判选项、改善关系 |
| 商业L3+ | 解锁交易选项、提高收益 |

**示例**：
```
事件：资源争夺战

基础选项：
- 武力夺取
- 高价收购

谋略L4+ 解锁额外选项：
- 暗中破坏（削弱对方后再夺取）
- 调虎离山（转移对方注意力）
```

## 附录：事件设计检查清单

### 设计新事件时确认：

- [ ] 触发条件是否合理且可检测？
- [ ] 是否有明确的决策点？
- [ ] 每个选项都有合理后果吗？
- [ ] 后果与选择逻辑一致吗？
- [ ] 是否有技能关联的额外选项？
- [ ] 事件文本符合世界观风格吗？
- [ ] 是否会在晨报中产生后续影响？

### 平衡性检查：

- [ ] 奖励是否与风险匹配？
- [ ] 是否有明显的"最优解"？
- [ ] 不同阵营玩家体验是否公平？
- [ ] 是否会导致数值膨胀？
