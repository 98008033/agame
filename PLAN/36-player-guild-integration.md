# Agame 玩家融入职业团体系统

> **核心理念**：玩家加入团体不只是"获得好处"，而是真正"融入社会"。团体是玩家建立人脉、学习技能、获取身份的核心渠道，是玩家从"无名小卒"变成"社会参与者"的关键桥梁。

---

## 一、玩家与团体交互的核心机制

### 1.1 交互层次模型

玩家与团体的交互分为五个层次，从浅到深：

| 层次 | 名称 | 玩家行为 | 团体响应 | 深度影响 |
|------|------|----------|----------|----------|
| **第1层** | 观察 | 了解团体信息、旁观活动 | 无特别响应 | 无关系建立 |
| **第2层** | 互动 | 与团体NPC交流、完成相关任务 | NPC个体响应 | 建立NPC关系 |
| **第3层** | 加入 | 成为正式成员、参与团体活动 | 团体认可、提供资源 | 获得"成员"身份 |
| **第4层** | 参与 | 积极参与团体事务、贡献资源 | 团体提升待遇、给予权力 | 成为"核心参与者" |
| **第5层** | 领导 | 成为团体管理者或领袖 | 团体服从决策、提供最高权限 | 成为"社会领导者" |

### 1.2 交互触发点

玩家可以通过多种方式发现和接触团体：

```
发现途径：

主动发现：
├── 通过晨报（团体相关新闻）
├── 通过NPC对话（NPC提及自己所属团体）
├── 通过探索（发现团体总部/聚会地点）
├── 通过技能学习（师傅提及相关团体）
└── 通过事件（事件中接触团体成员）

被动触发：
├── 团体NPC主动邀请（如果玩家技能达标）
├── 团体招募事件（团体公开招募成员）
├── 团体活动事件（路过看到团体聚会）
├── 团体危机事件（团体求助）
└── 团体推荐事件（关系好的NPC推荐）
```

**触发示例**
```
【晨报触发】
晨报内容：
"铁匠工会将在下周举办锻造大会，欢迎所有铁匠参加。"

玩家响应：
[了解更多] → 显示铁匠工会详情
[参加锻造大会] → 触发团体活动事件（需要技能）
[忽略]

---

【NPC对话触发】
与铁匠·老锤对话：
"你是铁匠吗？有没有考虑加入铁匠工会？
工会里有很多同行，可以互相学习。"

玩家响应：
[询问如何加入] → 进入加入流程
[婉拒] → 老锤可能下次再问
[询问工会的好处] → 获得详细信息

---

【被动邀请触发】
事件：
"你在街上遇到了铁匠工会会长·铜须。
他注意到你身上带着一把精心锻造的刀。
'年轻人，这把刀的手法不错。有没有兴趣加入我们工会？'"

玩家响应：
[接受邀请] → 获得邀请信，加入门槛降低
[婉拒] → 铜须好感略降
[询问更多信息]
```

## 二、玩家融入团体的完整路径

### 2.1 融入路径总览

玩家从"无名小卒"到"团体核心"需要经历完整的融入过程：

```
融入路径：

阶段1：陌生人 → 认识者（了解阶段）
  ├─ 了解团体信息
  ├─ 与团体成员建立初步关系
  ├─ 完成与团体相关的小任务
  └─ 获得"认识团体"的标签

阶段2：认识者 → 申请者（准备阶段）
  ├─ 确认加入意向
  ├─ 准备加入条件（技能、金钱、关系）
  ├─ 寻找推荐人或完成考验
  └─ 提交申请

阶段3：申请者 → 新成员（入会阶段）
  ├─ 完成入会考验/仪式
  ├─ 正式加入团体
  ├─ 获得"团体成员"身份和初级头衔
  └─ 开始享受基础资源

阶段4：新成员 → 活跃成员（成长阶段）
  ├─ 积极参与团体活动
  ├─ 完成团体任务、贡献资源
  ├─ 与团体成员建立深层关系
  └─ 提升团体内部等级

阶段5：活跃成员 → 核心成员（深入阶段）
  ├─ 担任团体内部职责
  ├─ 参与团体决策
  ├─ 代表团体对外交涉
  └─ 获得团体高级头衔

阶段6：核心成员 → 团体领袖（领导阶段）
  ├─ 竞选或被推举为管理者
  ├─ 管理团体日常事务
  ├─ 代表团体参与阵营决策
  └─ 获得"团体领袖"头衔和最高权力
```

### 2.2 各阶段详细设计

**阶段1：陌生人 → 认识者**

```
目标：让玩家了解团体，建立初步印象

触发条件：
├─ 与团体相关NPC对话
├─ 晨报看到团体新闻
├─ 路过团体聚会地点
└─ 完成与团体相关的任务

认识者的特征：
├─ 知道团体的名称、功能、特点
├─ 认识至少1名团体成员
├─ 可能获得团体成员的推荐
└─ 可以查看团体的公开信息

认识者事件示例：
"你帮铁匠·老锤运送了一批矿石。
老锤很高兴：'干得不错。要不要来铁匠工会看看？'
他邀请你参观工会。"

玩家选择：
[接受邀请] → 参观工会，获得"认识铁匠工会"标签
[婉拒] → 老锤下次再问
[询问工会更多] → 获得详细信息
```

**阶段2：认识者 → 申请者**

```
目标：玩家准备加入条件，正式申请

准备内容：
├─ 技能条件：技能等级达到门槛
├─ 资源条件：准备入会费/材料
├─ 关系条件：找到推荐人
└─ 特殊条件：完成前置考验

申请流程：
├─ 向团体管理者表达加入意愿
├─ 提交申请材料（推荐信、证明等）
├─ 团体审核申请
└─ 确定入会考验内容

申请者事件示例：
"你向铁匠工会表达了加入意愿。
工匠长·炉火审查了你的申请：
'技能达标，但需要老锤的推荐。
还有，你需要完成一件作品作为入会考验。'"

玩家行动：
[找老锤写推荐信] → 需要老锤关系≥50
[开始制作作品] → 选择物品进行锻造
[暂时搁置] → 申请保留，下次可继续
```

**阶段3：申请者 → 新成员**

```
目标：完成入会考验/仪式，正式加入

入会形式（因团体类型不同）：
├─ 技能考验型（工艺团体）：完成一件作品
├─ 战功考验型（军事团体）：完成一次任务
├─ 金钱门槛型（商业团体）：支付入会费
├─ 契约签订型（金雀花团体）：签订入会契约
├─ 仪式认证型（学术团体）：通过仪式认证
└─ 推荐快速型（精英团体）：推荐人担保

新成员获得：
├─ "××团体学徒/新兵"头衔
├─ 团体成员身份记录
├─ 基础资源使用权（学习、材料）
├─ 团体活动邀请
└─ 与其他成员的基础关系

入会事件示例：
"你的作品——一把精工锻造的刀——被工匠长认可。
炉火宣布：'欢迎加入铁匠工会！'
你获得'铁匠工会学徒'头衔。
工会成员向你表示欢迎。"

玩家获得：
✓ 头衔：铁匠工会学徒
✓ 可以在工会学习锻造技能
✓ 可以借用工会共享材料
✓ 收到工会活动通知
```

**阶段4：新成员 → 活跃成员**

```
目标：积极参与，积累贡献，提升地位

活跃方式：
├─ 参加团体定期活动（聚会、训练）
├─ 完成团体任务（团体发布的专属任务）
├─ 学习团体技能（向团体导师学习）
├─ 帮助团体成员（协助其他成员）
└─ 贡献团体资源（捐赠材料、金钱）

活跃成员特征：
├─ 在团体有一定知名度
├─ 与多名团体成员建立关系
├─ 获得团体中级头衔
├─ 享受更多团体资源
└─ 可能担任小组职务

活跃事件示例：
"你在过去半年里积极参与铁匠工会活动：
- 完成了15次工会任务
- 帮助3名学徒学习基础锻造
- 捐赠了10单位稀有矿石

工匠长炉火找到你：
'你的贡献得到认可。工会决定提升你为'正式工匠'。'"

玩家获得：
✓ 头衔升级：学徒 → 正式工匠
✓ 可以收徒（教导他人）
✓ 优先获得稀有材料
✓ 参与工会决策讨论
```

**阶段5：活跃成员 → 核心成员**

```
目标：担任职责，参与决策，成为核心

核心成员职责：
├─ 管理团体某方面事务（如新人培训）
├─ 参与团体决策会议
├─ 代表团体对外交涉
├─ 仲裁团体内部纠纷
└─ 传承团体核心技能

核心成员特征：
├─ 在团体有较高影响力
├─ 与团体领袖建立深层关系
├─ 获得"团体骨干"头衔
├─ 享受最高级别资源
└─ 可能被推荐为继承人

核心成员事件示例：
"铁匠工会年度大会上，会长铜须宣布：
'鉴于你的长期贡献和出色表现，
工会决定让你担任'技艺监督'职责，
负责审核新人作品和监督产品质量。'"

玩家获得：
✓ 头衔：技艺监督
✓ 权力：审核新人入会
✓ 权力：监督产品质量
✓ 参与：工会决策会议
✓ 收入：职责津贴
```

**阶段6：核心成员 → 团体领袖**

```
目标：成为团体最高管理者，代表团体

成为领袖的方式：
├─ 选举型（松散团体）：成员投票选举
├─ 推举型（紧密团体）：现任领袖推荐
├─ 挑战型（实力团体）：通过挑战上位
├─ 继承型（血缘团体）：继承领袖职位
└─ 契约型（金雀花团体）：签订领袖契约

领袖权力：
├─ 管理团体所有事务
├─ 决定团体政策方向
├─ 代表团体参与阵营决策
├─ 任命团体内部职位
└─ 使用团体所有资源

领袖事件示例：
"铁匠工会会长铜须年事已高，决定退休。
他召开大会宣布继承人：
'经过慎重考虑，我推荐你接任会长。
工会的未来交给你了。'"

玩家获得：
✓ 头衔：铁匠工会会长
✓ 权力：管理工会全部事务
✓ 地位：阵营内重要人物
✓ 影响力：参与阵营经济决策
✓ 传承：可以指定继承人
```

## 四、玩家在团体中的角色定位

### 4.1 三种角色类型

玩家在团体中可以扮演不同的角色：

| 角色 | 定义 | 特点 | 获得方式 | 适合玩家 |
|------|------|------|----------|----------|
| **普通成员** | 基础参与者 | 享受资源，完成义务 | 满足加入门槛即可 | 希望低调发展、专注技能的玩家 |
| **贡献者** | 积极参与者 | 贡献资源，获得认可 | 积累贡献值 | 希望快速成长、积累人脉的玩家 |
| **领导者** | 决策参与者 | 管理事务，影响团体 | 达到核心地位+推举/竞选 | 希望影响社会、参与政治的玩家 |

### 4.2 普通成员角色

```
普通成员定位：

核心特征：
├─ 享受团体基础资源
├─ 完成团体基本义务
├─ 不参与团体决策
├─ 不担任团体职务
└─ 低调、专注个人发展

适合场景：
├─ 玩家刚加入团体
├─ 玩家只想学习技能
├─ 玩家没有时间/精力参与团体事务
└─ 玩家同时属于多个团体（精力分散）

普通成员体验：
"作为铁匠工会的普通成员，
你定期参加工会聚会，
偶尔完成工会任务，
主要是利用工会资源学习锻造。
你不参与工会决策，
但工会对你的态度友好。"

## 五、团体作为"社会融入"的核心机制

### 5.1 社会融入的意义

团体不只是资源渠道，而是玩家融入社会的核心桥梁：

```
融入社会的含义：

从"无名小卒"到"社会参与者"：
├─ 无名小卒：没有人认识，没有社会关系
├─ 认识者：知道某些人，有初步关系
├─ 参与者：有身份，有关系网，参与社会活动
├─ 影响者：有地位，有影响力，能影响他人
└─ 领导者：有权力，有决策权，塑造社会

团体在融入过程中的作用：
├─ 提供身份：让玩家从"无名"变成"有身份的人"
├─ 提供关系：让玩家建立社会关系网络
├─ 提供渠道：让玩家参与社会活动
├─ 提供认可：让玩家获得社会认可
└─ 提供权力：让玩家获得社会影响力
```

### 5.2 团体融入事件链

玩家通过团体事件链逐步融入社会：

```
完整融入事件链示例（铁匠工会）：

事件1：初次接触（陌生人阶段）
"你帮铁匠·老锤运送矿石，他邀请你参观工会。"
→ 获得"认识铁匠工会"标签

事件2：深入了解（认识者阶段）
"你参加铁匠工会聚会，认识了几名工匠。"
→ 获得"认识工会成员"关系

事件3：决定加入（申请者阶段）
"老锤推荐你加入工会，你需要完成入会考验。"
→ 开始加入流程

事件4：入会考验（申请者阶段）
"你精心锻造了一把刀，工匠长炉火认可了你的作品。"
→ 获得"入会通过"状态

事件5：正式加入（新成员阶段）
"工会宣布你成为正式成员，获得'学徒'头衔。"
→ 获得"铁匠工会学徒"身份

事件6：团体活动（活跃成员阶段）
"工会举办锻造培训会，你积极参与并帮助其他学徒。"
→ 获得"活跃成员"认可

事件7：团体任务（活跃成员阶段）
"你完成了一系列工会任务，积累了贡献值。"
→ 获得"贡献工匠"称号

事件8：晋升核心（核心成员阶段）
"工匠长推荐你担任'技艺监督'，负责审核新人作品。"
→ 获得"技艺监督"职责和身份

事件9：团体决策（核心成员阶段）
"工会召开决策会议，你的意见被采纳。"
→ 获得"决策参与者"地位

事件10：成为领袖（领导者阶段）
"会长退休，推荐你接任会长职位。"
→ 获得"铁匠工会会长"身份和最高权力

融入完成：
├─ 玩家从"无名小卒"变成"工会会长"
├─ 玩家在铁匠行业有影响力
├─ 玩家参与阵营经济决策
└─ 玩家成为"社会参与者"
```

├─ 团体内贡献高 → 团体推荐 → 阵营认可
├─ 团体内地位高 → 代表团体 → 社会身份
├─ 团体内技能高 → 团体认证 → 专业认可

├─ 团体内背叛 → 团体追责 → 社会孤立
└─ 团体内丑闻 → 团体曝光 → 名誉下降

事件示例：

击退了霜狼入侵者。

佣兵工会铁面注意到你：
'年轻人，战斗能力不错。
有没有兴趣加入佣兵工会？
我们可以给你更多战斗机会。'"

玩家选择：
[接受邀请] → 加入佣兵工会
[婉拒] → 铁面下次可能再问
[询问更多信息]
```

### 8.3 非正式融入

玩家可以不正式加入，但仍参与团体活动：

```
非正式参与方式：

观察者：
├─ 参加团体公开活动（如聚会、竞赛）
├─ 不是正式成员
├─ 不享受成员资源
└─ 可能建立成员关系

合作者：
├─ 与团体合作完成任务
├─ 不是正式成员
├─ 获得任务报酬
└─ 可能建立合作关系

客户：
├─ 使用团体提供的商业服务
├─ 不是正式成员
├─ 支付服务费用
└─ 可能建立商业关系

朋友：
├─ 与团体成员建立个人关系
├─ 不是正式成员
├─ 通过成员了解团体
└─ 可能获得非正式帮助

非正式参与事件示例：
"你不是铁匠工会的正式成员，
但你经常参加工会聚会（开放活动），
认识了很多工匠，
偶尔帮他们完成任务获得报酬。

工会对你印象：
'这个年轻人经常帮忙，虽然不是成员，
但我们的关系不错。'

你获得'工会友好者'标签，
工匠们愿意帮助你（非正式）。"
```

## 九、离开与退出机制

### 9.1 正常退出

玩家可以正常退出团体：

```
正常退出条件：

契约期满（金雀花团体）：
├─ 契约到期
├─ 选择不续签
├─ 正常退出，无惩罚
└─ 保留关系，无负面影响

申请退出（松散团体）：
├─ 向团体提交退出申请
├─ 团体批准（通常容易）
├─ 正常退出
└─ 可能需要完成未尽义务

完成任期（职务型团体）：
├─ 职务任期结束
├─ 选择不再任职
├─ 可能降为普通成员或退出
└─ 保留部分关系

正常退出事件示例：
"你向铁匠工会提交退出申请。
工匠长炉火批准：
'理解你的选择。虽然退出了，
但我们还是朋友。'

你失去工会成员身份，
但保留与工会成员的关系。"
```

### 9.2 违约退出

违约退出有惩罚：

```
违约退出类型：

契约违约（金雀花团体）：
├─ 契约期内强制退出
├─ 需支付违约金
├─ 信用受损
└─ 可能被黑名单

义务违约（紧密团体）：
├─ 未完成团体义务就退出
├─ 可能被降级或惩罚
├─ 声誉受损
└─ 可能被追责

效忠违约（军事团体）：
├─ 未履行服役义务就退出
├─ 可能被视为"逃跑"
├─ 声誉严重受损
└─ 可能被追捕

违约退出事件示例：
"你在金雀花银行公会的契约期还有半年，
但你决定提前退出。

公会回应：
'契约违约。需要支付违约金1500金币。
违约记录将影响你的信用评级。'

玩家选择：
[支付违约金] → 正常退出，但信用下降
[拒绝支付] → 可能被公会追责
[协商减免] → 需要关系≥60"
```

### 9.3 被驱逐

团体可能驱逐玩家：

```
被驱逐原因：

行为失当：
├─ 损害团体利益
├─ 背叛团体成员
├─ 泄露团体秘密
└─ 违反团体规则

能力不足：
├─ 长期技能不达标
├─ 长期贡献不足
├─ 未履行职责
└─ 长期不参与活动

阵营冲突：
├─ 加入敌对阵营
├─ 与敌对势力合作
├─ 被质疑忠诚
└─ 可能被驱逐

被驱逐后果：
├─ 失去所有团体资源
├─ 失去团体身份和头衔
├─ 声誉受损（可能严重）
├─ 关系可能恶化
└─ 可能被追责或报复

被驱逐事件示例：
"铁匠工会召开紧急会议，
讨论你的行为：
'你向竞争对手泄露了工会的技术信息，
损害了工会利益。'

工会决定：
'驱逐你，并通知其他工会注意此人。'

你失去工会身份，
声誉严重受损，
可能影响在其他工会的加入机会。"
```

## 十一、MVP简化建议

### 11.1 MVP必须实现

**P0优先级**：
- 基础加入流程（申请 → 考验 → 加入）
- 3种角色定位（普通成员、贡献者、领导者）
- 团体资源提供（技能学习、人脉建立）
- 团体定期活动（聚会、任务）
- 表现对地位的影响

**核心团体优先**：
- 铁匠工会（工艺类代表）
- 佣兵工会（军事类代表）
- 商会（商业类代表）
- 五行学院（学术类代表）
- 谷主大会（边境自治代表）

### 11.2 MVP简化内容

```
简化加入流程：
├─ 只保留"申请 → 考验 → 加入"三步
├─ 不实现复杂的推荐和特殊路径
└─ 入会考验简化为单一任务

简化角色系统：
├─ 只保留"普通成员"和"活跃成员"两种
├─ 不实现"领导者"阶段
└─ 不实现多团体组合

简化活动系统：
├─ 只实现月度聚会
├─ 不实现竞赛和大会
└─ 不实现团体间合作

简化地位影响：
├─ 只实现团体内地位
├─ 不实现团体外影响
└─ 不实现复合声誉
```

## 十二、总结

### 12.1 核心设计要点

1. **五个交互层次**：观察 → 互动 → 加入 → 参与 → 领导
2. **六个融入阶段**：陌生人 → 认识者 → 申请者 → 新成员 → 活跃成员 → 核心成员 → 团体领袖
3. **四种帮助方式**：技能学习、人脉建立、资源获取、身份获得
4. **三种角色定位**：普通成员（低调）、贡献者（积极）、领导者（决策）
5. **社会融入核心**：团体是玩家从"无名小卒"变成"社会参与者"的桥梁
6. **动态社交内容**：定期活动、内部竞争、合作任务
7. **表现影响地位**：团体内表现影响团体外声誉和社会地位

### 12.2 与其他系统关联

```
玩家融入系统 ←→ 团体系统：基于团体结构设计融入路径
玩家融入系统 ←→ 技能系统：团体是技能学习核心渠道
玩家融入系统 ←→ 关系系统：团体是人脉建立核心渠道
玩家融入系统 ←→ 身份系统：团体是身份获取核心渠道
玩家融入系统 ←→ 事件系统：团体融入触发事件链
玩家融入系统 ←→ 阵营系统：团体是阵营关系桥梁
```
